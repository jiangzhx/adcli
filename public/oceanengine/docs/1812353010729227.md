# 获取创编可用的抖音图文素材

该接口用于获取「巨量营销平台」创建营销时可用的抖音图文素材，您可以获取抖音图文素材的item_id，在营销创编接口使用。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/file/carousel/aweme/get/](https://api.oceanengine.com/open_api/v3.0/file/carousel/aweme/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1812353010729227)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

# 请求参数

字段

类型

描述

number

巨量营销投放账户ID

aweme_id

string

抖音号id

filtering

object

过滤器

item_ids

number[]

抖音图文唯一ID

cursor

string

页码游标值，首次传空字符串

后续请求传上一次请求返回的cursor_info.cursor

count

number

页面数据量，默认10，最大值100

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

message

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

aweme_carousel_list

object[]

抖音图文素材列表

item_id

number

抖音图文id，在抖音号内上传图文素材时生成的id，是抖音端图文素材的唯一id

material_id

number

抖音图文素材id，在查询素材维度的报表（会用该id作为维度，对应「素材管理」-「[获取图文素材](https://open.oceanengine.com/labels/7/docs/1773554026740736)」等相关接口中的carousel_ids.id或carousel_id）

carousel_title

string

抖音图文标题

carousel_play_url

string

抖音图文播放地址，格式为：https://douyin.com/video/XXXXXX

carousel_cover_url

string

抖音图文素材内封面图片预览url，有效期10小时

images

object[]

抖音图文素材内的图片信息，最多35张图片

按照图文本身的顺序返回，最前面的图片在最上面

image_id

string

抖音图文素材内图片的uri

width

number

抖音图文素材内图片宽度

height

number

抖音图文素材内图片高度

cursor_info

object

分页信息

cursor

string

下一次分页拉取的游标值

has_more

bool

是否有下一页

request_id

string

请求日志id
