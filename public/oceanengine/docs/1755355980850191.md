# 批量获取营销成本保障状态

批量获取营销成本保障状态

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/cost_protect_status/get/](https://api.oceanengine.com/open_api/v3.0/promotion/cost_protect_status/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1755355980850191)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| promotion_ids 必填 | number[] | 营销ID集合，每次最多传入50个 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| compensate_status_info_list | object[] |  |
| query_id | number | 请求计划id |
| status | string | 当前请求是否成功, 枚举值: SUCCESS: 查询成功 FAILED: 查询失败，请重试 |
| compensate_status | string | 成本保障状态 枚举值: DEFAULT_STATUS: 当前计划未产生首次send或者不在成本保障范围内 IN_EFFECT: 成本保障生效中 INVALID: 成本保障已失效 CONFIRMING: 成本保障确认中 PAID: 成本保障已到账 ENDED: 成本保障已结束 |
| invalid_reasons | dict[string] | 成本保障失效原因 枚举值: AUD_CHANGES单日修改定向超过2次, BID_CHANGES单日修改出价超过2次 ROI_CHANGES单日修改roi_goal超过2次 ANTI_SPAM命中反作弊 BID_TYPE_EXPIRED选择的出价产品暂不支持成本保障 MANUAL_JUDGE_SPAM有异常的作弊行为 AUD_BID_CHANGES单日修改定向/出价超过2次 AUD_ROI_CHANGES单日修改定向/ROI目标超过2次 |
| end_reasons | dict[string] | 成本保障结束原因 UN_OBERCOST 超成本比例没有达到1.2倍 ROI_REAL_EXPECTED实际roi大于目标roi的80% CONVERT_UNDER_THRESHOLD转化数没有达到门槛 CURRENCY_PRECISION赔付金额小于0.01元 |
| compensate_amount | number | 赔付金额 |
| url | string | 赔付规则链接 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
