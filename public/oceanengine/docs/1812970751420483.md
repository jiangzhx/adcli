# 创建【核算明细数据】下载任务

返回文件详细内容请查看： [https://bytedance.larkoffice.com/docx/EBETdc8IIonmWoxWYJ7c1dNtndg](https://bytedance.larkoffice.com/docx/EBETdc8IIonmWoxWYJ7c1dNtndg)

# 请求地址

[https://api.oceanengine.com/open_api/2/file/rebate/rebate_download/create_task/](https://api.oceanengine.com/open_api/2/file/rebate/rebate_download/create_task/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1812970751420483)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

agent_id

number

代理商账户id

query_type

string

查询类型,目前包括 month_punish_details:下载违规明细月度 quarter_punish_details:下载违规明细季度 month_rebate_details:下载核算明细月度 quarter_rebate_details:下载核算明细季度 month_performance_details:下载业绩明细月度 quarter_performance_details:下载业绩明细季度 可选值:

year

number

年

month_quarter

string

月/季度，可以传多个，以逗号分隔，如传入 1,2,3
