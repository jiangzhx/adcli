# 获取不活跃账户列表

您可通过此接口查询开发者应用（APPID）下最近7天请求过的不活跃账户，平台建议您降低此类账户的请求频次或无需请求

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/inactive_advertiser/list/](https://api.oceanengine.com/open_api/v3.0/tools/inactive_advertiser/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1834548530596872)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

App-Access-Token

string

】接口获取

# 请求参数

字段

类型

描述

app_id

number

应用ID，即发起请求的应用程序ID（APPID），可在开发者后台-[应用管理](https://open.oceanengine.com/developer/admin/ad_list)页面查看

1次请求会返回最近7天该APPID调用过的不活跃客户列表

cursor

number

游标，翻页使用 最小值：0 翻页使用，第一页传0，第二页开始，需要传前一页resp返回的cursor

count

number

单页拉取的数据量

允许值1-100000，默认值1000

advertiser_type

string
