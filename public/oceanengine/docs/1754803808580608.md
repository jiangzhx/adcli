# 获取评论列表

获取营销账号下的所有评论列表

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/comment/get/](https://api.oceanengine.com/open_api/v3.0/tools/comment/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1754803808580608)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户id

start_time

string

查询起始时间，格式：yyyy-MM-dd，时间跨度最大三个月，仅支持获取2021-01-01之后的评论

end_time

string

查询截止时间，格式：yyyy-MM-dd，时间跨度最大三个月，仅支持获取2021-01-01之后的评论

order_field

string

排序字段，允许值：

order_type

string

排序类型，允许值：

filtering

object

过滤条件，若此字段不传，或传空则视为无限制条件

ad_ids

number[]

计划id列表，一次最多100个

item_ids

number[]

营销视频id列表，一次最多100个，可通过【[获取抖音授权关系](https://open.oceanengine.com/docs/1729983667746823)】获取item_id和视频相关信息

level_type

string

查询的评论等级，允许值：

is_replied

number

是否已回复

hide_status

string

隐藏状态， 允许值：

emotion_type

string

评论情感，允许值：

content

string

评论关键词，最长支持10个字符

author_ids

number[]

视频作者抖音id，最多支持100个

comment_type

string

评论内容类型，允许值：

page

number

页码，默认值: 1

page_size

number

页面大小 默认值: 10，最大值：100。单次查询仅支持查询10000条以内评论，即page*page_size需要小于10000，否则报错
