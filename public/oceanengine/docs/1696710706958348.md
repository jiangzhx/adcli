# 兴趣关键词查询

兴趣关键词查询

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/interest_action/interest/keyword/](https://ad.oceanengine.com/open_api/2/tools/interest_action/interest/keyword/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710706958348)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token | string | 授权Access-Token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| query_words必填 | string | 关键词 |

# 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| list | object[] | 词包列表 |
| id | string | 关键词id |
| name | string | 关键词名称 |
| num | string | 关键词数目 |
| request_id | string | 请求日志id |
