# 查询动态创意词包

创建动态创意词包：创建的词包你可以使用在创意标题中做固定词位的替换，例如创建地理位置的词包，针对用户的位置属性显示你词包中相应的地理位置词。如果相应查看可使用的词包请参考查询创意词包接口。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/creative_word/select/](https://ad.oceanengine.com/open_api/2/tools/creative_word/select/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710718137356)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| creative_word_ids | number[] | 创意词包id列表，如不填默认返回所有创意词包 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| creative_word | list | 创意词包列表 |
| creative_word_id | number | 创意词包ID |
| name | string | 创意词包名称 |
| default_word | string | 默认词 |
| words | string[] | 替换词 |
| content_type | string | 创意词包类型，详见[【附件-创意词包类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535) |

| min_word_len | number | 替换词最小长度 |
| status | string | 创意词包状态，详见[【附件-创意词包状态】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| user_rate | float | 创意词包人群覆盖率 取值范围: 0-1 |
| request_id | string | 请求日志id |
