# 上传/更新投放资质（新版）

# 使用必看

该接口用于客户投放资质上传，请注意以下使用规则：

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_qualification/submit/](https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_qualification/submit/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1761501324513352)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户账号ID

qualifications

object[]

投放资质信息

qualification_id

number

资质id，选填

① 提交新资质时请传0，系统会在应答参数中返回生成的qualification_id

② 该接口允许您修改投放资质，因此如需修改，可传入qualification_id进行编辑

qualification_type

string

可选值:

attachment_ids

number[]

图片附件ids，通过【[上传资质图片](https://open.oceanengine.com/labels/7/docs/1696710599659532)】接口获取
