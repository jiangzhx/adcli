# 获取图片素材

通过此接口，用户可以获取经过一定条件过滤后的客户下创意素材库的图片及图片信息。

# 请求地址

[https://api.oceanengine.com/open_api/2/file/image/get/](https://api.oceanengine.com/open_api/2/file/image/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710601254912)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

filtering

object

图片过滤条件

image_ids

string[]

图片ids，可以根据图片ids（创意中使用的图片key，存在一张图片对应多个image_ids的情况）进行过滤 数量限制：<=100 注意：image_ids、material_ids、signatures只能选择一个进行过滤

material_ids

number[]

素材id列表，可以根据material_ids（素材报表使用的id，一个素材唯一对应一个素材id）进行过滤 数量限制：<=100 注意：image_ids、material_ids、signatures只能选择一个进行过滤

signatures

string[]

md5值列表，可以根据素材的md5进行过滤 数量限制：<=100 注意：image_ids、material_ids、signatures只能选择一个进行过滤

width

double

图片宽度

height

double

图片高度

ratio

double[]

图片宽高比，eg: [1.7, 2.5]，输入1.7则搜索满足宽高比介于1.65-1.75之间的图片，即精度上下浮动0.05

start_time

string

根据图片上传时间进行过滤的起始时间，与end_time搭配使用，格式：yyyy-mm-dd

end_time

string

根据图片上传时间进行过滤的截止时间，与start_time搭配使用，格式：yyyy-mm-dd

page

number

页码，默认值1

page_size

number

页面大小，默认值20
