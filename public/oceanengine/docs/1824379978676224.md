# 查询字节小游戏归因激活时间窗

使用场景：可通过该接口查询当前字节小游戏最新的归因激活时间窗配置内容，如希望修改归因激活时间窗，需要前往编辑接口。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/micro_game/convert_window/get/](https://api.oceanengine.com/open_api/v3.0/tools/micro_game/convert_window/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1824379978676224)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| account_id必填 | number | 账户ID，account_type对应的账户ID |
| account_type必填 | string | 账户类型，允许值： AD投放账户 本次字节小游戏归因激活时间窗传AD |
| asset_id必填 | string | 资产id 字节小游戏传instance_id的值（[对应创建接口出参](https://open.oceanengine.com/labels/34/docs/1780614284853257?origin=left_nav)） 本次字节小游戏传instance_id的值 |
| asset_type必填 | number | 资产类型，允许值： BYTE_GAME：字节小游戏 目前仅支持字节小游戏，固定传BYTE_GAME |
