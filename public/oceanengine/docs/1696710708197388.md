# 获取行为兴趣推荐关键词

获取行为兴趣推荐关键词

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/interest_action/keyword/suggest/](https://ad.oceanengine.com/open_api/2/tools/interest_action/keyword/suggest/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710708197388)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| id必填 | number | 类目或关键词id |
| tag_type必填 | string | 查询类型：类目还是关键词 允许值：CATEGORY（类目）、KEYWORD（关键词） |
| targeting_type必填 | string | 查询目标：兴趣还是行为 允许值：ACTION（行为）、INTEREST（兴趣） |
| action_days | number | 行为天数，查询目标为行为时必填，兴趣不生效 允许值：7, 15, 30, 60, 90, 180, 365 |

| name | string | 关键词名称 |
