# 异步上传视频文件

将视频文件通过连山云素材服务上传后获取到视频文件链接，再将获取到的连山云视频文件url作为入参的video_url通过素材库提供的视频上传接口进行文件上传

仅支持开发者购置连山云素材服务上传生成的tos链接/连山点播云产品的视频链接上传，不支持其他三方链接地址

# 请求地址

[https://api.oceanengine.com/open_api/2/file/upload_task/create/](https://api.oceanengine.com/open_api/2/file/upload_task/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1801533703648324)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

account_id

number

账户ID

account_type

string

账户类型，允许值：ADVERTISER 客户、AGENT 代理商

filename

string

素材的文件名

labels

string[]

标签，每个标签长度是10，最大支持50个label

video_url

string

视频 url地址，最大支持上传文件大小：1000M

is_aigc

bool

视频素材是否为AIGC生成

is_guide_video

bool

上传视频场景是否是引导视频（此功能仅游戏行业白名单客户可用，如需使用请联系运营）

is_need_auth

bool

标注是否允许生效授权保护（素材需要是自产）
