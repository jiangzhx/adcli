# 获取单元诊断建议

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/promotion_diagnosis/suggestion/get/](https://api.oceanengine.com/open_api/v3.0/tools/promotion_diagnosis/suggestion/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1754715780584459)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

promotion_ids

number[]

单元ID列表，最多100个

scenes

string[]

希望获取建议的场景，允许值：

CLEAN清理场景

POTENTIAL 获取潜力场景

ZOMBIE 活跃度低单元

# 工具参数名称&工具参数值 对应说明

| 工具参数名称 | 工具参数值类型 | 描述 |
| --- | --- | --- |
| title_materials | object_list_param | 标题对象的列表，一个标题对象有两个属性（均为字符串类型）: title：标题文案 word_ids ：标题文案中如果有词包时，传入词包ID，多个词包ID用英文逗号分隔 |
| image_materials | object_list_param | 图片素材对象的列表，一个图片素材对象有两个属性（均为字符串类型）： image_mode ：素材类型枚举值 image_id ：图片素材URI |
| video_materials | object_list_param | 视频素材对象的列表，一个视频素材对象有三个属性（均为字符串类型）： image_mode ：素材类型枚举值 video_id ：视频vid cover_image_id：封面图片URI |
