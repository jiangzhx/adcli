# 创建素材清理任务

创建低效/同质素材清理任务的异步接口，最多同时创建10个运行中的清理任务，配合「获取清理任务列表」、「下载清理任务结果」接口使用

低效素材：[低效素材说明文档](https://bytedance.feishu.cn/docx/doxcncThI89dZfnMjVLfrAr0c9g)

同质化素材：[「同质化素材」说明文档（对外）持续更新](https://bytedance.feishu.cn/docx/Sh16dMqcUorPB4xrHTzcleQenag)

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/material/clear_task/create/](https://api.oceanengine.com/open_api/2/file/video/material/clear_task/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1764209655140366)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: `application/json` |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| clear_task_params 必填 | object | 任务参数，若同时填写多项任务的参数，则最终返回各个参数的交集 |
| material_source 必填 | string | 清理素材来源 允许值：QIANCHUAN 千川、AD 营销平台 |
| material_ids | number[] | 待清理素材id列表，单次最多支持100个 注：清理工具的清理对象为所有在投素材，因此当传入素材id列表中包含不在投素材时，该部分不在投素材将不会被清理。此情况下不推荐使用该字段进行清理，推荐使用clear_material_types字段进行清理。 |
| clear_material_types | string[] | 清理素材类型，允许值：INEFFICIENT_MATERIAL低效素材；SIMILAR_MATERIAL 同质化挤压严重素材；SIMILAR_QUEUE_MATERIAL同质化排队素材 注：该字段从当前在投的素材中扫描出需要清理的素材进行清理（推荐使用该方式），如果不传该字段，则默认清理这三个标签的全部在投素材 |
| create_time_upper | string | 清理创建时间"yyyy-mm-dd"之前的素材，不包括这一天 |
| start_time | string | 账户下累积转化数和累积消耗数的数据统计开始时间，格式为"yyyy-mm-dd"，且包含该日期 |
| end_time | string | 账户下累积转化数和累积消耗数的数据统计结束时间，格式为"yyyy-mm-dd"，且包含该日期 |
| convert | number | 账户下累积转化数，在start_time和end_time期间发生的累积转化数，清理小于该累积转化数的素材 当start_time和end_time传入时有效 |
| cost | float | 账户下累积消耗，在start_time和end_time期间发生的累积消耗，清理小于等于该累积消耗的素材 当start_time和end_time传入时有效 |
