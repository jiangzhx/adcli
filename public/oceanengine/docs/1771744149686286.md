# 创建微信小程序

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/wechat_applet/create/](https://api.oceanengine.com/open_api/v3.0/tools/wechat_applet/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1771744149686286)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

name

string

小程序名称，须与小程序实际名称保持一致，最大长度不超过50

user_name

string

小程序原始ID，获取方式：小程序后台>设置>基本设置>账号信息>原始ID，例如“gh_”，最大长度不超过20

path

string

小程序路径参数，如有跳转指定页面或自定义参数监测营销效果数据诉求，需填写。获取方式：登录关联的公众号后台>素材管理>新建图文素材>小程序>选择小程序>获取更多路径

remark_message

string

资产备注信息，最大长度不超过15

icon_image_url

string

小程序icon图片的url，尺寸60*60px,大小不超2MB

图片格式要求jpeg、png或gif

header_image_url

string

顶部头图的url，大小不超过5MB的图片，推荐尺寸2:1

图片格式要求jpeg、png或gif

labels

string[]

小程序标签，每个标签长度不超过6，最多支持6个标签

guide_text

string

引导文案，最大长度不超过14

images_vertical_url

string[]

小程序竖图URL地址

images_horizontal_url

string[]

小程序横图URL地址

introduction

string

小程序简介，最大长度不超过50

tag_info

string

细分类型

填写细分类型的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

payment_form

string

付费形式

membership_type

string

会员类型

revenue_model

string

变现模式

has_online_earning

bool

是否包含网赚内容

max_recharge_tier

string

最高充值档位（元）

min_recharge_tier

string

recommended_recharge_tier

string

prop_name

string

道具名称

min_payment_tier_range

string

最低付费档位区间（元）

mid_payment_tier_range

string

中间付费档位区间（元）

可选值:

max_payment_tier_range

string

最高付费档位区间（元）

可选值:

has_discount

bool

是否有折扣

discount_rate

number

折扣比例
