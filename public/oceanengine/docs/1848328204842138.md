# 取消资产共享

适用场景：（升级版巨量引擎工作台资产共享能力介绍参考[帮助中心](https://support.oceanengine.com/support/content/145109?spaceId=377&mappingType=2)）

# 请求地址

https://api.oceanengine.com/open_api/v3.0/tools/ebp/asset/auth/cancel/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1848328204842138)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
