# 获取原生锚点详情

根据锚点唯一id，获取到锚点详情，支持查询账户下锚点的详情（包括被共享和自有锚点）

暂不支持获取「高级在线预约」锚点详情

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/native_anchor/get/detail/](https://api.oceanengine.com/open_api/v3.0/native_anchor/get/detail/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1786407342576640)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

# 请求参数

字段

类型

描述

anchor_ids

string[]

锚点id列表，list长度为20，注意只允许传入同一类（anchor_type)查询详情，否则报错

您可从「[获取原生锚点列表](https://open.oceanengine.com/labels/7/docs/1757715831323652)」获取锚点的anchor_type

advertiser_id

number

投放账户ID

anchor_type

string

锚点类型，同时只能查询一种锚点类型，可选值:

暂不支持获取「高级在线预约」锚点详情
