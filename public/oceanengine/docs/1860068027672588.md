# 投放账户对公充值

投放账户对公充值生成固定码

# 请求地址

https://api.oceanengine.com/open_api/v3.0/prepay_charge/generate_fix_remiattance_code/create/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1860068027672588)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
