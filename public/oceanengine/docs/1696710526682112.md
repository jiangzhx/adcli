# 查询账户日流水

获取客户日流水信息，一般每天8点会出来前一天的数据，如果当天存在数据延迟可往后再尝试。

### 请求地址

[https://ad.oceanengine.com/open_api/2/advertiser/fund/daily_stat/](https://ad.oceanengine.com/open_api/2/advertiser/fund/daily_stat/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710526682112)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

字段

类型

描述

advertiser_id

number

客户ID/代理商ID

注意：代理商仅支持AD、星图，暂不支持本地推代理商

start_date

string

开始时间，格式YYYY-MM-DD，默认当前年份的1月1日

end_date

string

结束时间，格式YYYY-MM-DD，默认为今天，起止时间间隔不能超过一年。

page

number

页码. 默认值: 1，注意：page*page_size不可大于10000

page_size

number

页面数据量. 默认值: 10

account_type

string

账户业务类型，可选值:

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| list | list | 客户ID |
| advertiser_id | number | 客户ID |
| date | string | 日期 |
| balance | float | 日终结余(单位元） |
| grant_balance | float | 日终赠款结余(单位元）。包括冻结和实结订单pending部分 |
| non_grant_balance | float | 日终非赠款结余(单位元）。包括冻结和实结订单pending部分 |
| cash_cost | float | 现金支出(单位元) |
| cost | float | 总支出(单位元) |
| frozen | float | 冻结(单位元) |
| income | float | 总存入(单位元) |
| reward_cost | float | 赠款总支出(单位元)，包括赔付赠款等全部赠款支出 |
| shared_wallet_cost | float | 共享返货支出（单位元） |
| company_wallet_cost | float | 账户消耗的子钱包的共享余额（单位元） |
| transfer_in | float | 总转入(单位元) |
| transfer_out | float | 总转出(单位元) |
| realtime_coupon_cost | float | 立减红包消耗(单位元) |
| deduction_cost | float | 消返红包消耗(单位元) |
| compensation_reward_cost | float | 赔付赠款消耗(单位元) |
| return_goods_reward_cost | float | 非共享返货赠款消耗(单位元) |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数量 |
| total_page | number | 总页数 |
| request_id | string | 请求日志id |
