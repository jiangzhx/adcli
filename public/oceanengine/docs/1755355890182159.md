# 批量修改深度出价

批量修改深度出价

不支持搜索周期稳投营销修改深度出价（项目层级delivery_type = DURATION）

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/deepbid/update/](https://api.oceanengine.com/open_api/v3.0/promotion/deepbid/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1755355890182159)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| data 必填 | object[] | 批量修改，包含计划id和出价（批量修改数量不能超过10个) |
| promotion_id 必填 | number | 营销ID，营销ID需要属于客户，否则会报错 |
| deepbid 必填 | float | 深度出价，单位“元”，精度：两位小数。取值范围：0.1-10000元. |
