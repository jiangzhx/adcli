# 代理商消耗报表数据

代理商消耗报表查询，对应方舟平台侧「商务-流水查询-消耗报表」数据

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/adv/cost_report/list/query/](https://api.oceanengine.com/open_api/2/agent/adv/cost_report/list/query/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1784979080790218)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST
