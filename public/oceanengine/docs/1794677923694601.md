# 终止订单接口

可批量终止DOU+在投订单(注：此接口白名单管控，请联系巨量引擎销售开通）

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/douplus/order/close/](https://api.oceanengine.com/open_api/v3.0/douplus/order/close/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1794677923694601)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

aweme_sec_uid

string

抖音号ID，通过已授权账户接口可以获取

order_ids

number[]

需要关停的order_id列表，最多10

task_ids

number[]

需要关停的task_id列表（PC端能看到的订单号），最多10

备注：请优先使用task_id终止订单，task_id和order_不允许同时传
