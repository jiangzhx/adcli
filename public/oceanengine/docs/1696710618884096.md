# 更改橙子建站站点状态

通过此接口，用户可以更改橙子建站站点状态

更改橙子建站站点状态前，可在[获取橙子建站站点详细信息](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710620022799)的返回参数中获取当前站点状态status ; 站点状态的枚举值详见 [【站点状态】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E7%AB%99%E7%82%B9%E7%8A%B6%E6%80%81)，状态不同，可对应的操作会做相应限制。如DELETED 和AUDIT_BANNED状态下不可再上线；EDIT状态下不可直接使用。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site/update_status/](https://ad.oceanengine.com/open_api/2/tools/site/update_status/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710618884096)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id， 传的这个advertiser_id的数字的范围：1 <= advertiser_id <= MAX_INT64 |
| site_ids必填 | string[] | 橙子建站站点id列表 ：1 <= 长度 <= 20 可通过[【获取橙子建站站点列表】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710620579852)获取应答字段list |
| status必填 | string | 站点操作状态 详见：[【站点操作状态】，枚举如下：](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535) published 发布 unpublished 下线 delete 删除 undeleted 恢复删除 注意：传入不在要求范围内的枚举值，不会生效 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| fail | list[string] | 更新失败的site_id的list及失败的原因。如果全部更新成功，len(list) = 0 |
| message | string | 错误信息 |
| site_id | 站点id | 更新失败的site_id |
| success | list[string] | 更新成功的site_id 列表。如果全部更新失败，len(list) = 0 |
| request_id | string | 请求日志id |
