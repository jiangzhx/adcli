# 资金共享-发起转账

发起转账，支持大钱包与小钱包互转，1:N批量转账

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/cg_transfer/wallet/transfer/create/](https://api.oceanengine.com/open_api/v3.0/cg_transfer/wallet/transfer/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1799669807408128)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

account_id

number

鉴权账户

account_type

string

鉴权账户类型 可选值:

biz_request_no

string

请求幂等id，同一biz_request_no发起转账代表同一转账申请，返回的转账单号相同，推荐uuid

main_wallet_id

number

申请转账的大钱包id

target_wallet_detail_list

object[]

申请转账的小钱包列表

sub_wallet_id

number

申请转账的小钱包id

transfer_capital_detail_list

object[]

申请转账资金列表

capital_type

string

资金类型 可选值:

platform

string

业务线 可选值:

transfer_amount

number

申请转账金额（单位：分）

capital_sub_type

string

transfer_direction

string

转账方向，以小钱包视角确定 可选值:

remark

string

转账备注
