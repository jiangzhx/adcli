# 【代理商】营销违规信息推送

即对应方舟后台「公司」-「营销违规信息」列表，当账户下存在：在投放中的图片、视频和落地页被拒审的巨量营销信息，会触发消息通知至订阅任务设置回调地址。

配套接口：[https://open.oceanengine.com/labels/7/docs/1790052406659072](https://open.oceanengine.com/labels/7/docs/1790052406659072)

# 请求地址

订阅任务设置回调地址

# 请求方式

POST

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type | string | 请求消息类型，允许值：application/json |
| X-Open-Signature | string | 签名，用来验证此次请求的合法性，具体算法见[【X-Open-Signature签名校验】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1722481732805646) |

# 请求参数

## 基础信息

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| message_id | string | 消息ID，唯一标识 |
| subscribe_task_id | number | 订阅任务ID |
| advertiser_ids | number[] | 消息对应的客户账号 |
| account_relation | map | 账号对应关系，包含所有advertiser_ids的授权关系 |
| cc_ids | number[] | 旧版巨量引擎工作台账号 |
| agent_ids | number[] | 代理商授权 |
| advertiser_ids | number[] | 客户id直接授权 |
| core_user_ids | number[] | 用户授权账号 |
| service_label | string | 订阅服务类型 agent_reject_promotion 代理商营销违规信息推送 |
| publish_time | number | 消息实际产生时间，毫秒时间戳 |
| timestamp | number | 毫秒时间戳，本条消息实际推送时间，获取推送数据列表时无该参数数据 |
| nonce | number | 随机数，和timestamp组合防重放，获取推送数据列表时无该参数数据 |
| data | json string | 推送数据信息，订阅服务类型不同，返回结构不同 |
| event | string | 事件名称，枚举值 REJECT_MATERIAL：拒审素材 |
| user_id | number | 素材归属的客户id |
| content | object | 事件内容 |
| promotion_id | number | 营销id |
| agent_id | number | 代理商id |
| illegal_material_ids | number[] | 违规素材id list |
