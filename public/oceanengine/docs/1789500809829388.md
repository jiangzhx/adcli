# 获取视频素材评估标签（新版）

本接口是「[获取素材标签列表](https://open.oceanengine.com/labels/34/docs/1761499216182279)」、「[获取素材标签信息](https://open.oceanengine.com/labels/34/docs/1763768835703812)」接口的升级版，支持查询账户下视频库的素材评估标签。新接口在支持原接口能力的基础上，扩展了以下新能力：

# 概念说明

注意本接口支持在不同账户维度下查询素材的评估标签（[创意生态标签介绍](https://bytedance.larkoffice.com/docx/LCSYd7lJ3oKEG8x1p1icGDHGn0d)），能力支持情况如下：

| 素材评估标签名称（点击超链接查看说明） | 巨量营销（AD) | 巨量千川 | 工作台（旧版巨量引擎工作台） | 方舟 | 标签更新频率 |
| --- | --- | --- | --- | --- | --- |
| [首发素材](https://bytedance.larkoffice.com/docx/IYTId7xsEo8p3ax3YkHcTHdxnNb) | ✅ | ✅ | ✅ | ✅ | 天级别更新。素材判定首发后不再变更，若非首发，用户在平台申诉成功，则标签更新为首发 |
| [低质素材](https://bytedance.larkoffice.com/docx/LCSYd7lJ3oKEG8x1p1icGDHGn0d) | ✅ | ✅ | ✅ | ✅ | 天级别更新 |
| [优质素材](https://bytedance.larkoffice.com/docx/KnqaddiXconCtax5LlkciXP7n75) | ✅ | ✅ | ✅ | ✅ | 天级别更新，素材判定优质标签后不再变更，若非优质，用户在平台申诉成功，则标签更新为优质 |
| [低效素材](https://bytedance.larkoffice.com/docx/doxcncThI89dZfnMjVLfrAr0c9g) | ✅ | ✅ | ✅ | ✅ | 天级别更新，当标签为是，该标签状态不发生改变 |
| [同质化素材风险-](https://bytedance.larkoffice.com/docx/Sh16dMqcUorPB4xrHTzcleQenag)挤压严重素材 | ✅ | ✅ | 🚫 | 🚫 | 实时计算，客户操作改变则会重新计算（可按照小时更新） |
| [同质化素材风险-](https://bytedance.larkoffice.com/docx/Sh16dMqcUorPB4xrHTzcleQenag)排队投放素材 | ✅ | ✅ | 🚫 | 🚫 | 实时计算，客户操作改变则会重新计算（可按照小时更新） |
| [同质化素材风险](https://bytedance.larkoffice.com/docx/Sh16dMqcUorPB4xrHTzcleQenag)-未投放预计排队素材 | ✅ | ✅ | 🚫 | 🚫 | 实时计算，客户操作改变则会重新计算（可按照小时更新） |
| [存在搬运打压风险](https://bytedance.larkoffice.com/docx/Fpn4dn4ldoAHrmxrCEfcpWO1n8f) | ✅（建议） | ✅（建议） | ✅ | ✅ | 实时计算，客户操作改变则会重新计算（可按照小时更新) |

注意3个同质化素材标签的区别

| 同质化素材风险-标签类型 | 是否可通过素材清理接口清理 | 素材是否在营销下 | 素材是否在投 |
| --- | --- | --- | --- |
| [同质化素材风险-](https://bytedance.larkoffice.com/docx/Sh16dMqcUorPB4xrHTzcleQenag)挤压严重素材 | 🚫该类素材没有可投/绑定创意，不能清理 | 🚫 | 🚫 |
| [同质化素材风险-](https://bytedance.larkoffice.com/docx/Sh16dMqcUorPB4xrHTzcleQenag)排队投放素材 | ✅可清理同质化排队素材 | ✅ | 🚫 |
| [同质化素材风险](https://bytedance.larkoffice.com/docx/Sh16dMqcUorPB4xrHTzcleQenag)-未投放预计排队素材 | ✅可清理同质化挤压严重素材 | ✅ | ✅ |

# 请求参数

字段

类型

描述

account_id

number

各平台账户id，必须同时选择account_type类型

account_type

string

账户类型， 可选值:

filtering

object

过滤条件

不同条件之间为「且」的关系，只有同时满足传入的多个条件，才会返回信息

material_ids

number[]

按素材ID过滤，范围为1-1000

material_properties

string[]

素材标签过滤项，如果不传，则默认返回客户ID下所有素材。

允许值：

start_time

string

起始日期，表示按照素材上传到账户下的时间过滤，格式为yyyy-mm-dd HH:MM:SS，最早允许传入时间：2022-01-01 00:00:00 （时间是账号和素材绑定的时间，删除重绑时间会更新）

end_time

string

结束日期，表示过滤出一段时间内上传的素材，格式为yyyy-mm-dd HH:MM:SS，如果传入起始日期，未传此参数，则默认为当前时间

attributes_modify_time

string

「存在搬运打压风险」属性最后一次更新时间，格式为yyyy-mm-dd ，筛选传入代表筛选出当天「存在搬运打压风险」发生过变化的素材信息

return_lowquality_suggestions

bool

应答参数是否返回低质原因，不传或传入false时，应答参数不返回低质原因

当素材为低质素材时，可查询素材低质原因，允许值：

返回：true

不返回：false

page_size

number

页面数据量，默认值: 100，page*page_size 最大1000

page

number

页码，默认值: 1，page*page_size 最大1000
