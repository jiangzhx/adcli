# 取消应用共享关系

取消应用共享关系

当前共享类型支持PART（指定账户共享）、ALL（旧版巨量引擎工作台内某类账户所有共享）和COMPANY （公司主体内某类型所有账户共享）。

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/bp_share/cancel/](https://ad.oceanengine.com/open_api/2/tools/app_management/bp_share/cancel/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1710413893446668)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| organization_id | number | 旧版巨量引擎工作台id |
| package_id | string | 应用包ID，可通过「查询应用信息」接口查询获取 |
| share_mode | string | 共享类型 允许值：PART 指定账户共享、 ALL 旧版巨量引擎工作台内某类型所有账户共享、COMPANY 公司主体内某类型所有账户共享 |
| all_accounts条件必填 | object[] | 取消共享账号类别，一次最多操作1个 当 share_mode 为ALL时可用且必填 |
| account_type必填 | string | 业务线 允许值：AD 营销、STAR 星图 |
| account_infos条件必填 | object[] | 取消共享账号信息，一次最多操作1个 当 share_mode 为PART时可用且必填 |
| account_type必填 | string | 业务线 枚举值：AD 营销、STAR 星图、BP 旧版巨量引擎工作台 |
| account_id必填 | number | 账号id，accout_type类型对应账户ID |
| all_account_by_company条件必填 | object[] | 公司主体信息，一次最多操作1个 当share_mode为 COMPANY时可用且必填 |
| account_type必填 | string | 业务线 枚举值：AD 营销、STAR 星图 |
| company_id必填 | number[] | 公司主体id, 一次最多操作10个 |
