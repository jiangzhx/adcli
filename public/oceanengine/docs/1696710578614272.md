# 获取商品详情

根据商品库查询商品库中的商品列表

若仅填写商品库ID（platform_id）不填filtering.product_ids则表示查询该商品库下所有商品 传入商品ID后视作查询商品精准查询，不会回传分页信息

只传商品库ID，查询商品库下所有商品；传了商品ID进行精准查询

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/product/detail/get/](https://ad.oceanengine.com/open_api/2/dpa/product/detail/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710578614272)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| platform_id 必填 | number | 商品库ID，可通过[【获取商品库信息】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710577036288)获取 若仅填写platform_id不填filtering.product_ids则表示查询该商品库下所有商品 |
| filtering | object | 过滤条件 |
| product_ids | number[] | 商品id，精准匹配。商品库ID必须与商品ID匹配，否则会报错。最多允许传入100个 |
| page | number | 页码， 默认值: 1 |
| page_size | number | 页面数据量， 默认值: 10，最大 100 |

### 应答字段

| 字段 | 类型 | 描述 |

| platform_id | number | 商品库ID |
| product_id | number | 商品ID |
