# 创建落地页组

通过此接口，用户可以创建按照流量分配的落地页组，创建成功后，接口会返回"code_0"。

落地页组信息会包括落地页组 ID、落地页组名称、落地页组 URL、落地页组状态、流量分配方式； 站点列表信息会包括成员 ID（即站点在落地页组中的唯一标识）、站点 ID、站点URL、站点启用状态、站点审核状态。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/landing_group/create/](https://ad.oceanengine.com/open_api/2/tools/landing_group/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710628472844)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，范围：1 <= advertiser_id <= MAX_INT64 |
| group_title必填 | string | 落地页组名称，范围：1 <= 长度 <= 20 |
| group_flow_type必填 | string | 流量分配方式。详见：[【附录-枚举值-落地页组流量分配方式】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E8%90%BD%E5%9C%B0%E9%A1%B5%E7%BB%84%E6%B5%81%E9%87%8F%E5%88%86%E9%85%8D%E6%96%B9%E5%BC%8F) |
| site_ids必填 | string[] | 橙子建站站点id列表 ：2 <= 长度 <= 10 可通过[【获取橙子建站站点列表】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710620579852)获取应答字段list |
| experiment_site_type | string | 落地页类型， 可选值: OWN: 橙子建站落地页 |
