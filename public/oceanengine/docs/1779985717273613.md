# 【代理商】批量暂停明点无效素材

通过【[【代理商】明点无效素材查询](https://open.oceanengine.com/labels/7/docs/1779971491239040)】接口查询无效素材，然后进行暂停

本接口不支持暂停自动化素材，自动化素材可通过【[批量更新营销素材启用状态](https://open.oceanengine.com/labels/7/docs/1755355780973568)】暂停

素材清理范围限「同质化素材」

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/pause/](https://api.oceanengine.com/open_api/2/file/video/pause/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1779985717273613)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| material_ids 必填 | number[] | 待暂停素材id列表，长度范围：1～20 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| material_clear_result | dict[object] | 暂停结果明细 |
| material_id | number | 素材id |
| is_cleared | bool | 素材是否被暂停（存在内部错误导致素材没有被暂停） |
| creative_success_list | number[] | 暂停成功的使用该素材的创意ID |
| creative_failure_list | number[] | 暂停失败的使用该素材的创意ID |
| promotion_success_list | number[] | 暂停成功的使用该素材的营销ID |
| promotion_failure_list | number[] | 暂停失败的使用该素材的营销ID |
| msg | string | 素材未被清理原因 |
| status | string | 暂停结果，允许值： SUCCESS 成功 PART_SUCCESS 部分成功 FAIL 失败 |
| fail_material_ids | number[] | 暂停失败的素材id明细(素材id下仍然有至少一个的可投创意) |
| request_id | string | 请求的日志id，唯一标识一个请求 |
