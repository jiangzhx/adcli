# 资金共享-共享钱包绑定/换绑/解绑

传入唯一操作id，关系变更类型，钱包id和账号id，进行共享钱包的绑定/换绑/解绑操作

# 请求地址

https://api.oceanengine.com/open_api/v3.0/shared_wallet/shared_relation/create/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1848121521443095)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

Content-Type

string

# 请求参数

字段

类型

描述

account_id

number

account_type

string

operation_id

number

操作id，调用获取唯一操作id接口获得，路径：/open_api/v3.0/ad_billing/unique_id/get/

relation_change_mode

string

from_wallet_id

number

如果是绑定/换绑，填 0

to_wallet_id

number

adv_range_parameter

object

(adv_ids 和 self_company_Ids不能同时传)

adv_ids

number[]

self_company_ids

number[]

（绑定公司id查找对应可绑定账户，解绑时不支持）

account_platform_list

string[]
