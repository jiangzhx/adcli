# 查看小游戏/小程序共享范围

获取字节、微信小游戏/小程序资产共享范围

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/get/](https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773105971421187)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

sharedAccounts

object[]

共享账号列表

share_mode

string

共享类型 允许值：PART 指定账户共享、BP_ALL_ACCOUNTS旧版巨量引擎工作台内所有账户共享、COMPANY_ALL_ACCOUNTS 公司主体内所有账户共享

共享类型为PART，读取account_infos；

共享类型为BP_ALL_ACCOUNTS，读取all_account_by_bp，即表示该业务线下的全量账号；

共享类型为COMPANY_ALL_ACCOUNTS，读取all_account_by_company

account_info

object

共享账号信息
