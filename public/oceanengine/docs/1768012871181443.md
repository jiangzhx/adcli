# 获取关键词列表

根据过滤条件获取符合条件的所有关键词。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/keyword/list/](https://api.oceanengine.com/open_api/v3.0/keyword/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1768012871181443)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

filtering

object

过滤结构

promotion_id

number

待过滤的营销ID

project_id

number

待过滤的项目ID

e.printStackTrace();

.url(url)
.method("GET", null)
.addHeader("Access-Token", ACCESS_TOKEN)

}

string

number

number
