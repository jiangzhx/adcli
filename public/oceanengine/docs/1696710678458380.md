# 获取地域列表

获取地域列表（当前仅支持获取商圈ID）

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/region/get/](https://ad.oceanengine.com/open_api/2/tools/region/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710678458380)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| region_type必填 | string | 地域类型，目前只支持：BUSINESS_DISTRICT(商圈) 允许值:"BUSINESS_DISTRICT" |
| region_level | string | 只获取某层级数据，详见【[附录-地域层级](https://open.oceanengine.com/labels/7/docs/1696710760171535?origin=left_nav)】 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| list | object[] | 行业列表 |
| id | number | ID |
| name | string | 名称 |
| parent_id | number | 父级id |
| region_level | string | 地域所在层级 |
| request_id | string | 请求日志id |
