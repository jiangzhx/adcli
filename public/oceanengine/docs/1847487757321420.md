## 小程序其他信息data说明

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| auditCompanyId | string | 主体id，范围为升级版巨量引擎工作台认证主体及关联认证主体 |
| tagInfo必填 | string | 玩法分类，可以通过[应用管理标签及类目](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)询枚举 |
| introduction | string | 小程序简介 |
| paymentFrom条件必填 | string | 付费形式，支持类目：工具、社交，仅该两个类目下参数必填，可选枚举如下 1 付费解锁完整功能 2 付费购买内容或服务 3 以上两者兼有 |
| membershipType条件必填 | string | 会员类型 支持类目：工具、社交，仅该两个类目下参数必填 可选值: 1 终身 2 年度 3 月度 4 周/天 5 无 |
| revenueModel条件必填 | string | 变现模式 支持类目：工具、社交、游戏道具售卖，仅该类目下参数必填 可选值: 1 付费 2 营销 3 混合 |
| revenueModelPoint条件必填 | string | 混合变现模式下主要变现模式，当revenueModel枚举为3时，需要传入当前字段 1 IAAP_P，重付费 2 IAAP_A，重营销 |
| hasOnlineEarning条件必填 | string | 是否包含网赚内容 支持类目：工具、社交，仅该类目下参数必填 当变现模式为付费/混合时必填 可选枚举： true false |
| maxRechargeTier条件必填 | string | 最高充值档位 支持类目：工具、社交，仅该类目下参数必填 当变现模式为付费/混合时必填 可选值: 1 1-50 2 50-100 3 100-200 4 200以上 |
| minRechargeTier条件必填 | string | 最低充值档位（元） 支持类目：工具、社交，仅该类目下参数必填 当变现模式为付费/混合时必填 可选值: 1 1-50 2 50-100 3 100-200 |
| recommendedRechargeTier | string | 推荐充值档位（元） 支持类目：工具、社交 当变现模式为付费/混合时选填 可选值: 1 1-50 2 50-100 3 100-200 4 200以上 |
| propName | string | 道具名称 |
| minPaymentTierRange条件必填 | string | 最低付费档位区间（元） 支持类目：游戏道具售卖，必填 当变现模式为付费/混合时必填 可选值: 1 0~5 2 6~10 3 10~20 4 20~50 |
| midPaymentTierRange条件必填 | string | 中间付费档位区间（元） 支持类目：游戏道具售卖 当变现模式为付费/混合时必填 可选值: 1 100以下 2 100~500 3 500以上 |
| maxPaymentTierRange条件必填 | string | 最高付费档位区间（元） 支持类目：游戏道具售卖必填 当变现模式为付费/混合时必填 可选值: 1 500以下 2 500-1000 3 1000以上 |
| scheduleCards | string[] | 是否含周卡/月卡/季卡等，当变现模式为付费/混合时必填 可选值: 1 周卡 2 月卡 3 季卡 4 年卡 5 终身卡 6 无 |
| hasDiscount条件必填 | string | 是否有折扣 支持类目：游戏道具售卖必填 当变现模式为付费/混合时必填 可选枚举： true false |
| discountRate | string | 折扣比例，为0-1之间的小数，最多两位小数 支持类目：游戏道具售卖 当变现模式为付费/混合，且是否有折扣=是 时选填 |
