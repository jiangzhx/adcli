# 更新字节小游戏

请注意：两种方式仅生效一种。当以上两个参数均有值时，将按照方式a.生效逻辑。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/micro_game/update/](https://api.oceanengine.com/open_api/v3.0/tools/micro_game/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1780614388338698)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

POST

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户id

instance_id

number

字节小游戏资产id

remark

string

备注

game_link

object[]

小游戏链接，小游戏调起链接存在两种录入方式（选其一）：

id

number

链接id，通过【[小程序详情接口](https://open.oceanengine.com/labels/8/docs/1778265753535620)】接口获取。当操作类型为 MODIFY和DELETE时，链接id必填

link

string

小游戏链接，以 sslocal://microgame 开头。生效优先级大于 启动参数，详细说明见接口说明

remark

string

链接备注

operate_type

string

操作类型，可选值:

start_param

string

启动参数，形如：name=zhangsan&age=25。不得以问号、连接符等特殊字符开头，请确保各参数间仅使用一个连接符

introduction

string

小游戏简介

tag_info

string

玩法分类

填写玩法分类的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

theme_tag

string

题材标签

填写题材标签的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

art_style

string

画风分类

填写画风分类的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

network_environment

string[]

网络环境，最多选择三个

填写网络环境的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

feature_tags

string[]

特色标签，最多选择三个

填写特色标签的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

age_limit

string

可选值:

string

可选值:

bool

可选值:

mid_payment_tier_range

string

可选值:

string

可选值:
