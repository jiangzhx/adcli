# 获取升级版商品详情

获取行业产品中心「产品管理-升级版」商品详情

# 请求地址

[https://api.oceanengine.com/open_api/2/dpa/clue_product/detail/](https://api.oceanengine.com/open_api/2/dpa/clue_product/detail/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1779436000953415)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

string
