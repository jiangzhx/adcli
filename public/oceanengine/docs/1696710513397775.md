# 获取投放资质（旧版）

获取客户资质信息，资质分为描述，营业执照，开户资质，投放资质。不同类型的资质有不同的字段，具体字段见下表。

【注意】本接口不再维护，建议使用「[获取投放资质（新版）](https://open.oceanengine.com/labels/34/docs/1761500990156815)」接口

##### 请求地址

[https://ad.oceanengine.com/open_api/2/advertiser/qualification/select_v2/](https://ad.oceanengine.com/open_api/2/advertiser/qualification/select_v2/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710513397775)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息，详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| request_id | string | 请求日志id |
| data | object | json返回值 |
| list | list | 资质列表 |
| qualification_id | number | 资质id |
| qualification_type | number | 投放资质类型 1：肖像、商标或其他授权 2: 专利证明 3: 商标证 4: 营销内容中的数据证明 5: 质检报告 |
| picture_preview_url | string | 投放资质预览url |
| status | number | 状态 0: 待提交 1: 待审核 2: 审核中 3: 审核通过 4: 审核拒绝 |
| reject_reason | string | 审核原因 |
| audit_time | string | 审核时间，形如 yyyy-MM-dd HH:mm:ss |
| image_list | object[] | 资质图片列表 |
| attachment_id | number | 附件id |
| image_url | string | 图片预览地址 |
