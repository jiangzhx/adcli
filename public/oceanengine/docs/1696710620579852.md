# 获取橙子建站站点列表

通过此接口，用户可以获取客户建站列表。

列表栏包括建站ID、建站名称、建站状态、建站类型、建站类别、站点缩略图等信息。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site/get/](https://ad.oceanengine.com/open_api/2/tools/site/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710620579852)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，范围：1 <= advertiser_id <= MAX_INT64 |
| page | int | 页码，默认值: 1，范围：page >= 1 |
| page_size | int | 页面数据量，默认值：10，范围：20～300 |
| status | string | 建站粗粒度状态，详见[【附录-枚举值-建站粗粒度状态】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%BB%BA%E7%AB%99%E7%B2%97%E9%A2%97%E7%B2%92%E5%BA%A6%E7%8A%B6%E6%80%81) |
| share_type 新增 | string | 站点来源 SHARE 共享站点 MY_CREATIONS账户创建站点 (默认） |
| filtering | json | 过滤条件 |
| search | string | 搜索字段，按照建站id和建站name进行模糊匹配，范围：1 <= 长度 <= 50 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| list | list | 建站列表 |
| siteId | string | 建站id |
| name | string | 建站名称 |
| status | string | 建站状态，详见附录[【附录-枚举值-建站状态】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%BB%BA%E7%AB%99%E7%8A%B6%E6%80%81) |
| siteType | string | 建站类型，详见附录[【附录-枚举值-建站类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%BB%BA%E7%AB%99%E7%B1%BB%E5%9E%8B) |
| function_type | string | 建站类别， SITE_FUNC_TYPE_NEW_MODULAR（智能建站） SITE_FUNC_TYPE_NORMAL （普通建站） |
| thumbnail | string | 站点缩略图 |
| request_id | string | 请求日志id |
