# 行为类目查询

行为兴趣定向行为类目查询。

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/interest_action/action/category/](https://ad.oceanengine.com/open_api/2/tools/interest_action/action/category/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710705116175)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token | string | 授权Access-Token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| action_days必填 | number | 行为天数 默认值: 7、15、30、60、90、180、365 |
