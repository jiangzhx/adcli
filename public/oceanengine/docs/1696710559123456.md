# 直播分析数据报表

直播分析数据报表，全方位地满足直播中和直播后的数据分析需求，帮助客户和代理商，深入洞察账户内全部直播、单场直播、直播商品的详细数据，查看流量来源分布，进行受众分析，助力直播间精细化运营，科学指导直播营销投放！

如何为投放账户关联抖音号/企业号？ 1）在[营销投放平台](https://ad.oceanengine.com/)-工具-账户工具-抖音号授权中，添加对应的抖音号，并选择授权类型为「抖音号授权」 2）在[巨量引擎工作台](https://business.oceanengine.com/site/index)-设置-账户管理-企业号/抖音号中，选择绑定企业号，并在完成绑定后，在操作栏中点击“关联账户"，选择“旧版巨量引擎工作台所有账户可用”。
