# 查询智能电话拨打记录

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/smartphone/record/](https://ad.oceanengine.com/open_api/2/clue/smartphone/record/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710645723148)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| instance_ids | number[] | 智能电话ID，不超过100个 |
| ad_ids | number[] | 营销计划ID，不超过100个 |
| clue_ids | number[] | 线索ID，不超过100个 |
| site_ids | number[] | 建站落地页ID，不超过100个 |
| start_time | string | 起始时间，格式：%Y-%m-%d，默认7天前，即不指定起止时间获取最近7天数据 |
| end_time | string | 截止时间，格式：%Y-%m-%d，默认当天，即不指定起止时间获取最近7天数据 |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值10，不超过50 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] | 智能电话列表 |
| instance_id | number | 智能电话ID |
| ad_id | number | 营销计划ID |
| site_id | number | 建站落地页ID |
| clue_id | number | 线索ID |
| call_duration | number | 呼叫时长，秒单位 |
| real_duration | number | 通话时长，秒单位 |
| virtual_number | string | 呼叫的虚拟号码 |
| start_time | string | 拨打时间，格式：%Y-%m-%d %H:%M:%S |
| create_time | string | 创建时间，格式：%Y-%m-%d %H:%M:%S |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
