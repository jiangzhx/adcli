# 获取返货共享钱包余额

此接口功能是查询同客户主体下返货共享钱包相关余额信息。

返货相关需要咨询相关的运营和销售同学对接，具备返货相关前置条件下，相关返货资金信息可以通过本接口获得

返货共享钱包需要在同客户主体下共享，返货共享钱包中的返货资金

### 请求地址

[https://ad.oceanengine.com/open_api/2/fund/shared_wallet_balance/get/](https://ad.oceanengine.com/open_api/2/fund/shared_wallet_balance/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1703323812831239)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_ids 必填 | number[] | 客户ID列表，长度1-100 |

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| list | object[] | 账户列表 |
| advertiser_id | number | 账户id |
| balance_detail | object[] | 余额详情列表 |
| balance | number | 余额，单位：元，精度：两位小数 |
| billing_inventory | string | 余额可用投放版位，DEFAULT：通用、SEARCH：搜索、UNION：穿山甲、COMMON：信息流 |
| status | string | 余额查询状态，SUCCESS：成功、FAIL：失败、NO_WALLET：钱包不存在 |
| status_message | string | 状态说明 |
| request_id | string | 请求日志id |
