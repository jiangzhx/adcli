# 新建优选起量任务

新建优选起量任务

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/task_raise/create/](https://api.oceanengine.com/open_api/2/tools/task_raise/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1733956164012035)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id |
| budget_mode必填 | string | 预算设置，允许值: LIMIT 指定预算、NO_LIMIT 不限预算 |
| allocated_budget | number | 日预算金额，当budget_mode为LIMIT时必填，范围：1000-9999999 |
| end_time | string | 起量任务结束时间yyyy-mm-dd，传空为不限时长，起量将在指定日期0点结束 |
| raise_mode | string | 起量模式，允许值：常规起量CUSTOM（默认值）、强劲起量STRONG |
