# 落地页数据报表

服务对象为营销场景作为转化载体的落地页，综合投放后验效果数据、用户行为数据落地页数据报表主要向客户披露落地页的性能以及用户行为关键指标 帮助客户洞察落地页数据背后的含义，深入挖掘数据价值，用智能化的手段制作更好的落地页

#### 橙子建站落地页数据报表

该接口披露橙子建站落地页用户行为关键指标分日数据，包括： 页面访问量:PV 用户访问量:UV 转化率:营销convert数/营销click事件 平均加载时长:页面平均加载渲染时长。单位：秒 平均阅读比例:用户平均浏览落地页的比例：用户离开时所在页面位置/页面总长度 平均访问时长:平均在落地页的停留时长。单位：秒
