# 查询受众预估结果

通过此接口你可以查询计划的受众条件，预估其在今日头条、抖音短视频、火山小视频和西瓜视频中的覆盖用户量，当受众预估的结果低于3万时，建议重新设置受众选项。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/estimate_audience/](https://ad.oceanengine.com/open_api/2/tools/estimate_audience/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710675422208)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET
