# 获取投放资质（新版）

用于查询账户投放资质

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_qualification/list/](https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_qualification/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1761500990156815)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

qualification_type

string

资质类型，允许值：

AGENT_ARRANGEMENT 营销代投协议

AUTHORIZATION_CONTRACT 委托加工/生产合同

DISTRIBUTION_AUTHORIZATION 经销授权书

ICP_RECORD_AUTHORIZATION icp域名备案/授权

OTHER_CERTIFICATION 其他资质

PATENT_CERTIFICATE 专利证书

PORTRAIT_AUTHORIZATION 肖像授权书

QUALITY_REPORT 鉴定/质检报告

SOFTWARE_COPYRIGHT_REGISTRATION_CERTIFICATE 软件著作权登记证书

TRADEMARK_REGISTRATION_CERTIFICATE 商标注册证

VIDEO_MATERIAL_PRODUCTION_IP_AUTHORIZATION 营销视频/素材/作品/IP授权

FOOD_BUSINESS_LICENSE食品经营许可证

status

string

状态，允许值：

STATUS_CONFIRM 审核通过

STATUS_CONFIRM_FAIL 审核不通过

STATUS_NOT_SUBMIT 未提交

STATUS_PENDING_CONFIRM 审核中

STATUS_WAIT_CONFIRM 待审核

page

number

页数默认值：1，page范围为[1,1000]

page_size

number

页面大小默认值：10，page_size范围为[1,100]
