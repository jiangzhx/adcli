# 获取营销产品资质规则配置

根据商业化行业获取不同行业下对应的资质提交规则。需注意：每个行业下的资质提交规则可能会因平台及外部监管的要求而发生变化，当规则发生变更时，规则的版本号+1

# 逻辑说明

业务逻辑总体介绍：根据投放内容所在的不同行业需要一组完整的、相关联的资质才可以形成证明效力。调用API之前，您需要首先查询本接口获取所选行业投放产品资质的配置规则（拿到config_id)

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_pkg_config/](https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_pkg_config/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1776710970860551)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
