# 获取行业列表

获取行业列表，通过接口可以获取到一级行业、二级行业、三级行业列表，其中代理商创建客户时使用的是二级行业，而在创建创意填写创意分类时使用的是三级行业，请注意区分。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/industry/get/](https://ad.oceanengine.com/open_api/2/tools/industry/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710677879808)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| level | number | 只获取某级别数据，1:第一级,2:第二级,3:第三级，默认都返回 |
| type | string | 可选值："ADVERTISER"，"AGENT"，"ADVERTISER"为原有营销3.0行业, "AGENT"为代理商行业获取，代理商行业level都为1 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| list | object[] | 行业列表 |
| industry_id | number | 行业ID |
| industry_name | string | 行业名称 |
| level | number | 所在级别，1：一级行业、2：二级行业、3：三级行业 |
| first_industry_id | number | 该行业的一级行业ID |
| first_industry_name | string | 该行业的一级行业名称 |
| second_industry_id | number | 该行业的二级行业ID |
| second_industry_name | string | 该行业的二级行业名称 |
| third_industry_id | number | 该行业的三级行业ID |
| third_industry_name | string | 该行业的三级行业名称 |
| request_id | string | 请求日志id |
