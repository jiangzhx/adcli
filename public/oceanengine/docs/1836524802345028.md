# 查询代理商充值记录

代理商充值记录查询，相关功能与代理商平台的「商务-充值记录-账户充值记录」模块对齐。

[https://api.oceanengine.com/open_api/2/agent/adv/recharge/recharge_record/](https://api.oceanengine.com/open_api/2/agent/adv/recharge/recharge_record/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1836524802345028)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
