# 查询组件列表

该接口用户获取创意组件信息。

# 请求地址

[https://ad.oceanengine.com/open_api/2/assets/creative_component/get/](https://ad.oceanengine.com/open_api/2/assets/creative_component/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710673645580)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| filtering | json | 过滤条件 |
| component_id | number | 组件ID |
| component_name | string | 组件名称。支持模糊查询 |
| component_types | string[] | 组件类型，不传查全部。 CHOICE_MAGNET 选择磁贴 VOTE_MAGNET 投票磁贴 IMAGE_MAGNET 图片磁贴 COMMERCE_MAGNET 电商磁贴 COUPON_MAGNET 优惠券磁贴 GAME_PACK 游戏礼包码 PROMOTION_CARD 投放卡片 LIGHT_INTER_ACTIVE 轻互动组件 RESERVATION_BUTTON 游戏预约按钮 UNION_LIGHT_INTERACTIVE 穿山甲轻互动组件 ECOMMERCE_CARD卡片-展示价格力 HALF_INTERACTIVE半屏-点击互动类 |
| status | string[] | 组件审核状态，不传查全部。 PASS 通过 UNDER 组件新建 REJECT 未通过 |
| page | number | 页码，默认值1 |
| page_size | number | 页面大小，默认值20，最小值10，最大值40 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| list | json | 组件列表 |
| component_id | number | 组件ID |
| component_name | string | 组件名称 |
| component_type | string | CHOICE_MAGNET 选择磁贴 VOTE_MAGNET 投票磁贴 IMAGE_MAGNET 图片磁贴 COMMERCE_MAGNET 电商磁贴 COUPON_MAGNET 优惠券磁贴 GAME_PACK 游戏礼包码 PROMOTION_CARD 投放卡片 LIGHT_INTER_ACTIVE 轻互动组件 RESERVATION_BUTTON 游戏预约按钮 UNION_LIGHT_INTERACTIVE 穿山甲轻互动组件 ECOMMERCE_CARD卡片-展示价格力 HALF_INTERACTIVE半屏-点击互动类 |
| component_data | object | 组件详细信息。不同的component_type对应的值不同，具体的结构见创建或更新接口定义 |
| create_time | string | 创建时间。格式"2020-12-25 15:12:08" |
| status | string | 组件审核状态。 PASS 通过 UNDER 组件新建 REJECT 未通过 |
| page_info | json | 分页信息 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| page_size | number | 页面大小 |
| page | number | 当前页 |
| request_id | string | 请求日志id |
