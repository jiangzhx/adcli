# 上传图文

利用图片和音频塑造图文。图文是一种创意内容体裁，在包含多张图片和一段背景音乐，支持自动轮播图片。利用MAPI创建图文素材，需首先获取图文素材内的图片/音频id

利用上述两步的图片、音频id塑造图文，获取出参图文mid。不同图片顺序对应不同的图文mid（即接口应答参数中的carousel_id）。

# 请求地址

[https://api.oceanengine.com/open_api/2/carousel/create/](https://api.oceanengine.com/open_api/2/carousel/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773552433235140)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户id

images

object[]

图片信息，包含图片id和图片主题

image_id

string

图片id，根据[上传营销图片](https://open.oceanengine.com/labels/7/docs/1696710600176640)接口获取

audio_id

string

音频id，根据[「上传图文内的音频素材」](https://open.oceanengine.com/labels/34/docs/1780005237921792)接口获取

非必填，不传系统将自动匹配一段纯音乐

file_name

string

图文标题，可自定义图文素材名

不传则默认取第一张图片的名称，最长255个字符

description

string

千川图文素材描述
