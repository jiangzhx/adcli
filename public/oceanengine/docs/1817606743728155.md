# 删除升级版商品

【使用场景】对客户创建的「升级版」商品进行删除操作，注意已经关联计划的商品不允许进行删除，支持批量，一次性调用最大个数为100，服务为部分成功部分失败。

【支持的删除方式】

| 删除模式 | 支持类目 | 类目id（category_id） |
| --- | --- | --- |
| 支持按照店铺ID+外部商品ID删除 | 电商店铺商品 | 140000000 |
| 按照product_id指定删除 | 除电商店铺商品外的商品 截止12.9为活动、房产、招商加盟、教育培训、医疗服务、人资代招、岗位招聘 | 250000000 130000000 240000000 40000000 50900000 110100000 120000000 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| product_ids必填 | number[] | 通过巨量商品id删除，支持批量，一次请求最多允许传入100个 需有商品的权限才允许删除，否则会报错 |
| store_id_and_outer_id | object | 电商店铺商品（category_id为140000000时）支持按照店铺id+外部商品id删除，其他类目商品不需要传 |
| store_id条件必填 | string | 店铺ID，传入时必须与商品外部ID同时传入 |
| outer_id条件必填 | string | 商品外部ID |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |

| data | object | 返回数据 |
| product_ids | number[] | 删除成功的商品ID集合 |
| errors | object | 删除失败的商品ID集合及错误信息 |
| product_id | number | 商品ID |
| error_message | string | 错误信息（主要的可能错误为商品关联了在投计划） |
| request_id | string | 请求的日志ID，唯一标识一个请求 |
