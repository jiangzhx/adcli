# 提交解析应用包任务

当您需要创建一个附加创意类型为游戏礼包码的营销计划时，如果该游戏礼包的download_url为未在字节的营销系统中上传或使用过的新download_url，则需要使用该接口提交包解析任务。 使用提交解析应用包任务时，您只需要传入参数advertiser_id和该游戏礼包的download_url，即可解析出APP相关信息（包名、appname、icon等），这些信息将在礼包码原生转化卡中展示。 提交解析应用包任务后，您可以使用[【查询包解析状态】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口对当前包解析状态进行查询。查询到返回参数package_status为"SUCCESS"后，方可提交创建营销计划。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/download/package/parse/](https://ad.oceanengine.com/open_api/2/tools/download/package/parse/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710744089612)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |
| Content-Type | string | 请求消息类型 允许值: "application/json" |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| download_url必填 | string | 下载链接 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| request_id | string | 请求日志id |
| data | json | 返回数据 |
| event_id | string | 事件ID，用于查询解析包状态接口，有效期为24小时 |
