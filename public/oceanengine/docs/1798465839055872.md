# 资金共享-批量查询钱包信息

批量查询钱包信息（包含共享钱包和子钱包）

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/shared_wallet/wallet_info/get/](https://api.oceanengine.com/open_api/v3.0/shared_wallet/wallet_info/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1798465839055872)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

account_id

number

账户ID，对于巨量营销、千川、本地推是客户ID，对于代理商是代理商ID

wallet_id_list

number[]

资金共享钱包id列表，注意：传入的列表长度不可大于200

account_type

string

账户类型 可选值:

e.printStackTrace();
}
});

Request request = new Request.Builder()
.url(url)
.method("GET", null)
.addHeader("Access-Token", ACCESS_TOKEN)
.build();
Response response = client.newCall(request).execute();
return response.body().string();
}
