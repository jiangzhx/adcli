# 批量修改DPA商品状态

此接口用于批量修改DPA商品状态

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/product_status/batch_update/](https://ad.oceanengine.com/open_api/2/dpa/product_status/batch_update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1726637930114048)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| platform_id 必填 | number | 商品库ID |
| product_ids 必填 | number[] | 商品ID列表 |
| opt_status 必填 | string | 操作 ENABLE开启投放 DISABLE停止投放 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| success_list | number[] | 修改状态成功的商品列表 |
| error_list | object[] | 修改状态失败的商品列表 |
| product_id | number | 修改失败的商品id |
| error_msg | string | 失败原因 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
