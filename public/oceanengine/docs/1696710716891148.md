# 动态创意词包管理

相关链接：

动态创意词包结合用户内在属性与外在场景，基于丰富的用户数据，精准推送营销内容。 动态词包分类公共词包（后台提供）和自建词包（客户自建）两种类型，其使用场景较为多元化，如：地域属性、产品多元化、私人特征、节日/活动投放等。
