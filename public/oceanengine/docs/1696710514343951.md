# 上传投放资质（旧版）

通过此接口，用户可以批量上传投放资质。

【注意】本接口不再维护，请使用【[上传/更新投放资质（新版）](https://open.oceanengine.com/labels/7/docs/1761501324513352)】接口

##### 请求地址

[https://ad.oceanengine.com/open_api/2/advertiser/qualification/create_v2/](https://ad.oceanengine.com/open_api/2/advertiser/qualification/create_v2/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710514343951)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：application/json |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| qualification_type 必填 | string | 资质类型，详见[【附录-资质类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%B9%BF%E5%91%8A%E4%B8%BB%E8%B5%84%E8%B4%A8%E7%B1%BB%E5%9E%8B).允许值: "QUALIFICATION_AD" |
| qualifications 必填 | list | 资质列表 min>=1 && max<=50 |
| qualification_img_id 必填 | string | 资质图片id，通过【[上传资质图片](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710599659532)】获得（需指定 upload_to 为 AD） |
| ad_qualification_type 必填 | string | 资质图片类型 AUTHORIZATION 肖像、商标或其他授权 CERTIFY 专利证明 TRADEMARK_CERT 商标证 AD_DATA_CERT 营销内容中的数据证明 INSPECT_REPORT 质检报告 |
