# 查询抖音类目下的推荐达人

该接口用于解决 创建计划时，设置抖音达人定向需要的相关id值

接口将返回抖音类目下的推荐达人，根据类目id查询抖音推荐达人

抖音类目id可以通过【查询抖音类目列表】接口获取

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/aweme_category_top_author/get/](https://ad.oceanengine.com/open_api/2/tools/aweme_category_top_author/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710689096719)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token | string | 授权Access-Token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| category_id | number | 类目id，一级，二级，三级类目id均可 |
| behaviors | string[] | 抖音用户行为类型，详见[【附录-抖音达人互动用户行为类型】](https://ad.oceanengine.com/openapi/doc/index.html?key=qianchuan&type=api&id=1697577930691592#item-link-20)默认为空,仅影响覆盖人群数 注：该字段为选填，需表达特定行为类型才可返回有效值 |

# 应答参数
