# 获取升级版巨量引擎工作台下账户列表

账户量级不超过1w。仅支持拉取直接挂接的账户列表，可通过「[获取工作台组织关系（升级版）](https://open.oceanengine.com/labels/7/docs/1829551362400384)」获取EBP团队结构

# 请求地址

[https://api.oceanengine.com/open_api/2/ebp/advertiser/list/](https://api.oceanengine.com/open_api/2/ebp/advertiser/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1829550825614739)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
