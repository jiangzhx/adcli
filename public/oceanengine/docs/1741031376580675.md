# 批量删除营销

批量删除 巨量引擎体验版 营销

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/delete/](https://api.oceanengine.com/open_api/v3.0/promotion/delete/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1741031376580675)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

POST Request-Body

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type | string | 请求消息类型，允许值：application/json |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

## 基础参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户ID |
| promotion_ids 必填 | number[] | 营销ID集合，list长度限制1～10 |

# 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见【附录-返回码】 |
| message | string | 返回信息,详见【附录-返回码】 |
| data | json | json返回值 |
| promotion_ids | number[] | 删除成功的营销ID集合 |
| errors | object[] | 删除失败的营销列表 |
| promotion_id | number | 营销ID |
| error_message | string | 错误信息 |
| request_id | string | 请求日志id |
