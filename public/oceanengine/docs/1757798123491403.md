# 创建原生锚点

该接口暂不支持创建「高级在线预约」、「外跳」、「字节小程序」锚点

# 近期更新

2023/12/27，请求参数中product_price类型调整为double，允许2位小数，原按整数型传入不受影响

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/native_anchor/create/](https://api.oceanengine.com/open_api/v3.0/native_anchor/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1757798123491403)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
