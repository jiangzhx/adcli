# 查询视频是否挂载下载类锚点

建议优先选择带游戏/网服/平台电商的行业锚点视频投放，更有可能获得平台优质流量补贴

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/video/check_available_anchor/](https://ad.oceanengine.com/open_api/2/tools/video/check_available_anchor/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1729984030929924)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | string | 客户ID |
| item_ids必填 | number[] | 抖音视频ID ，通过[【获取抖音主页视频】](https://open.oceanengine.com/labels/7/docs/1729982871844879) 或 【[获取抖音授权关系](https://open.oceanengine.com/labels/7/docs/1729983667746823)】获取主页视频或合作授权视频 |
| landing_type必填 | string | 营销目的 可选值: APP: APP |
| external_action必填 | string | 转化目标 可选值: AD_CONVERT_TYPE_ACTIVE: 激活、AD_CONVERT_TYPE_ACTIVE_REGISTER: 注册、AD_CONVERT_TYPE_APP_ORDER: app内下单、AD_CONVERT_TYPE_APP_UV: app内访问、AD_CONVERT_TYPE_APP_PAY: app内付费、AD_CONVERT_TYPE_PAY: 激活且付费 |
