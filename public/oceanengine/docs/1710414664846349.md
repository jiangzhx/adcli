# 查询安卓应用信息（支持所有账户体系）

查询账户下安卓应用信息（支持所有账户体系）及应用详细信息

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/android_app/list/](https://ad.oceanengine.com/open_api/2/tools/app_management/android_app/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1710414664846349)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| account_id必填 | number | 账户id，accout_type类型对应账户ID |
| account_type必填 | string | 账户类型允许值： BP 巨量引擎工作台、AD 客户账号、 STAR 星图 |
| filtering | object | 过滤条件 |
| search_key | string | 搜索关键字 app_id或者应用名称，可以为空，可以传中文，长度不超过50 |
| search_type | string | 搜索类型 允 许值：ALL 查询全部，包括创建和被共享的应用（默认）、 CREATE_ONLY 只查询客户创建的应用、SHARE_ONLY 只查询被共享的应用 |
| status | string | 查询应用状态 允许值：ALL 所有状态 、AUDIT_DOING 审核中、AUDIT_REJECTED 审核失败 、AUDIT_ACCEPTED 审核成功，即待发布 、ENABLE 已发布 默认值：ENABLE |
| publish_time | object | 按发布时间查询的时间范围 |
| start_time | string | 发布起始时间，格式：%Y-%m-%d |
| end_time | string | 发布结束时间，格式：%Y-%m-%d |
| create_time | object | 按创建时间查询的时间范围 |
| start_time | string | 创建起始时间，默认为空，格式：%Y-%m-%d |
| end_time | string | 创建结束时间，默认为当天，格式：%Y-%m-%d |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值为10，最大值100 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] | 应用列表 |
| package_id | string | 应用包ID |
| app_cloud_id | number | app id |
| app_name | string | 应用名 |
| package_name | string | 包名 |
| download_url | string | 下载地址 |
| version | string | 版本号 |
| update_time | string | 更新时间，格式：%Y-%m-%d %H:%M:%S |
| icon_url | string | icon地址 |
| publish_time | string | 发布时间，格式：%Y-%m-%d %H:%M:%S |
| create_time | string | 创建时间，格式：%Y-%m-%d %H:%M:%S |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
