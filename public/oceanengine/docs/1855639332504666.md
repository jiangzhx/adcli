# 投放账户充值校验

校验投放账户是否可以充值

# 请求地址

https://api.oceanengine.com/open_api/v3.0/charge/verify/get/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1855639332504666)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

GET

# Header

字段

类型

描述

string

# 请求参数

字段

类型

描述

number

number
