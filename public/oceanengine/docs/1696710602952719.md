# 获取同主体下客户图片素材

通过此接口，用户可以查询获取同主体下的客户图片素材信息。

接口权限：查看图片的信息，获取图片md5、宽高、预览地址等内容，合理利用还可以搭建自己的素材库进行素材的管理。

成功调用本接口需同时满足以下条件：

### 请求地址

[https://ad.oceanengine.com/open_api/2/file/image/ad/get/](https://ad.oceanengine.com/open_api/2/file/image/ad/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710602952719)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| image_ids 必填 | string[] | 图片ids，长度限制1~100 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| id | string | 图片ID |
| size | number | 图片大小 |
| width | number | 图片宽度 |
| height | number | 图片高度 |
| url | string | 图片预览地址 |
| format | string | 图片格式 |
| signature | string | 图片md5 |
| material_id | number | 素材id，一个素材唯一对应一个素材id |
| request_id | string | 请求日志id |
