# 批量修改DPA商品状态

适用场景：仅支持对归属升级版巨量引擎工作台的通用版商品批量修改商品状态

# 请求地址

https://api.oceanengine.com/open_api/v3.0/dpa/ebp/product_status/batch_update/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1846200705851659)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

POST

# Header

字段

类型

描述

string

Content-Type

string
