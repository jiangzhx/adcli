# 获取视频智能封面

通过此接口，用户可以获取针对素材视频推荐的智能封面。智能封面是通过提取视频关键帧筛选出推荐封面，帮助发现视频内优质封面素材。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/video_cover/suggest/](https://ad.oceanengine.com/open_api/2/tools/video_cover/suggest/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710602404864)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| video_id 必填 | string | 视频id |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| status | string | 封面生成的状态 枚举值：RUNNING（生成中）、SUCCESS（成功）、FAILED（失败） 注意：视频封面不是实时生成，需要根据status字段判断封面生成的状态，大概需要6，7s中生成，所以当status=RUNNING时请等待后重试 |
| list | object[] | 视频封面列表 |
| id | string | 图片ID |
| width | number | 图片宽度 |
| height | number | 图片高度 |
| url | string | 图片预览地址 |
| request_id | string | 请求日志id |
