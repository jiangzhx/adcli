# 获取异常应用资产列表

可通过接口查询确认客户未完成融合归因方案改造的异常资产。

接入逻辑可参考[巨量营销转化融合归因优化方案接入手册V2.0](https://bytedance.larkoffice.com/docx/CgYBdVzoBogND2xv7PhcSfFNnyc)，接入中如有问题可通过【[雅典娜工单](https://ad.oceanengine.com/athena/user-feedback/?identify_key=e8bdace58c96e8b79fe8b8aae5a496e983a8e4b893e58cba3136333935333733)】排查/联系对接销售，如您认可通过巨量平台完成归因也可接入转化SDK简化接入流程，详情参考[「对外」巨量营销转化安卓端SDK使用说明 for 6.16.9手册](https://bytedance.larkoffice.com/docx/doxcnjAFo7iUImEIq9DuA7Tr69g)

接口仅返回异常资产内容，数据更新间隔为T+1

[https://api.oceanengine.com/open_api/v3.0/event_manager/abnormal_assets/get/](https://api.oceanengine.com/open_api/v3.0/event_manager/abnormal_assets/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1823745189417120)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
