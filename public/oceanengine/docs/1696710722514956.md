# 删除穿山甲流量包

删除穿山甲流量包。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/union/flow_package/delete/](https://ad.oceanengine.com/open_api/2/tools/union/flow_package/delete/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710722514956)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| flow_package_id必填 | number | 流量包ID，从[【获取穿山甲流量包】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710720565248)获取 |
