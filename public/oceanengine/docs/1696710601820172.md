# 获取视频素材

通过此接口，用户可以获取经过一定条件过滤后的客户下创意素材库对应的视频及视频信息。

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/get/](https://api.oceanengine.com/open_api/2/file/video/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710601820172)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

filtering

object

视频过滤条件

width

number

视频宽度

height

number

视频高度

ratio

float[]

视频宽高比，示例: [1.7, 2.5]

输入1.7则搜索满足宽高比介于1.65-1.75之间的视频，即精度上下浮动0.05

video_ids

string[]

视频ids，示例: ["86adb23eaa21229fc04ef932b5089bb8"]

数量限制：<=100

注意：video_ids、material_ids、signatures只能选择一个进行过滤

material_ids

number[]

素材id列表，可以根据material_ids（素材报表使用的id，一个素材唯一对应一个素材id）进行过滤

数量限制：<=100

注意：video_ids、material_ids、signatures只能选择一个进行过滤

signatures

string[]

md5值列表，可以根据素材的md5进行过滤

数量限制：<=100

注意：video_ids、material_ids、signatures只能选择一个进行过滤

start_time

string

根据视频上传时间进行过滤的起始时间，与end_time搭配使用，格式：yyyy-mm-dd

end_time

string

根据视频上传时间进行过滤的截止时间，与start_time搭配使用，格式：yyyy-mm-dd

labels

string[]

视频标签

source

string[]

素材来源，详见[【附录-素材来源】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)

枚举值大小写敏感，请严格按照定义的名称传参

star_author_ids

string[]

星图达人 id 检索，仅当source = STAR 时，支持通过星图达人ID进行筛选，单次最多支持传入20 个id进行检索

page

number

页码，默认值1

page_size

number

页面大小，默认值20
