# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/keywords_bid_ratio/create/](https://api.oceanengine.com/open_api/v3.0/tools/keywords_bid_ratio/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1760582291523597)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| word_list 必填 | string[] | 优词列表，最多支持10个，字数最大限制：30，每个优词生成一个优词计划id 每个账户最多支持200个优词 |
| bid_ratio 必填 | float | 优词提量系数，取值范围:[1.1,2]，仅支持一位小数 |
| dimension 必填 | string | 生效维度，可选值: ADVERTISER: 账户维度 PROJECT: 项目维度 |
| project_ids 条件必填 | number[] | 生效的项目id列表，最多100个，当dimension=PROJECT时必填 |
