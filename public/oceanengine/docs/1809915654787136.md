# 投放账户信息查询

投放账户信息查询，包括：基础信息、自运营报备标签、账户绑定时间、账户联系人、账户负责人和协作者等

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/advertiser_info/query/](https://api.oceanengine.com/open_api/2/agent/advertiser_info/query/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1809915654787136)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET
