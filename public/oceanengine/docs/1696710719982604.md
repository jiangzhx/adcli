# 穿山甲流量包

相关链接：

穿山甲精选流量包，通过行业首创的独家双向建模，结合转化行为，挖掘行业优质流量，助力客户后端效果提升： 1、适用行业：应用软件&游戏 2、营销目的：APP应用&落地页-应用下载 3、投放范围：穿山甲直投计划（激励视频/原生信息流/开屏-竞价）

游戏精选流量包和应用软件精选流量包互斥，不能同时选择 2、仅有穿山甲直投计划可用，优选投放版位不支持
