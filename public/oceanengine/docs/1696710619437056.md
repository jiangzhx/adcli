# 获取橙子建站站点预览地址

通过此接口，用户可以获取已创建站点的预览地址 预览地址有效期：20分钟

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site/preview/](https://ad.oceanengine.com/open_api/2/tools/site/preview/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710619437056)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，客户id， 传的这个advertiser_id的数字的范围：1 <= advertiser_id <= MAX_INT64 |
| site_id必填 | string | 橙子建站站点id 可通过[【获取橙子建站站点列表】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710620579852)获取 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| url | string | 建站站点的预览地址，有效期：20分钟 |
| request_id | string | 请求日志id |
