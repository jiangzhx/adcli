# 获取评论回复

获取营销账号下的评论回复（用于获取二级评论）

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/comment_reply/get/](https://api.oceanengine.com/open_api/v3.0/tools/comment_reply/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1754803965480960)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户id

comment_id

number

评论id

page

number

页码，默认值：1

page_size

number

页面大小，默认值：10，最大值：100
