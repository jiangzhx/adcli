# 资金共享-批量查询钱包余额

资金共享-批量查询钱包余额

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/shared_wallet/wallet_balance/get/](https://api.oceanengine.com/open_api/v3.0/shared_wallet/wallet_balance/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1798465689405452)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

account_id

number

账户ID，对于巨量AD、千川、本地推是客户ID，对于代理商是代理商ID

account_type

string

账户类型 可选值:

wallet_id_list

number[]

资金共享钱包id列表，注意：传入的列表长度不可大于200

wallet_balance_filters

object

余额过滤条件

account_platform_type

string

业务线：AD/千川/本地推;不填默认不过滤 可选值:

默认值: NO_LIMIT

capital_type

string

资金类型包括 预付、授信; 不填默认不过滤资金类型 可选值:

默认值: NO_LIMIT

delivery_type

string

投放类型：竞价、品牌、通用; 不填默认不过滤 可选值:

默认值: NO_LIMIT
