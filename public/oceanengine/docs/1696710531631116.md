# 更新账户日预算

此接口可以更新客户账号设置的预算类型与预算

### 请求地址

https://ad.oceanengine.com/open_api/2/advertiser/update/budget/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710531631116)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: "application/json" |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| budget_mode 必填 | string | 预算模式，详见[【附录-枚举值】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)，可选值: BUDGET_MODE_DAY（日预算） BUDGET_MODE_INFINITE（不限） |
| budget 条件必填 | float | 预算值，取值范围：1000～9999999.99，当budget_mode=BUDGET_MODE_DAY时，必填 单位：元； 最小值：0； 精度：小数点后两位 举例：100.01 |
