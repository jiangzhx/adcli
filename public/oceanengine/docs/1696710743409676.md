# 查询游戏预约记录详情

查询游戏预约记录详情数据接口

其中应答参数是按creative_time正序返回的。如果您有数据处理的需求，可以参考此排序逻辑。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/booking_records/get/](https://ad.oceanengine.com/open_api/2/tools/app_management/booking_records/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710743409676)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| package_id必填 | string | 应用id 如果是Anroid应用，package_id从【[查询应用信息](https://ad.oceanengine.com/openapi/doc/index.html?key=ad&type=api&id=1696710742760448)】接口获取； 如果是IOS应用，package_id为App Store中的ItunesID； |
| host_type必填 | string | 应用是否寄存在应用管理中: HOST_IN:寄存 HOST_OUT:未寄存 如果是Anroid应用，必须先上传至应用管理中，请选择寄存； 如果是IOS应用,请选择未寄存; |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值为10，最大不超过200 |
| create_time | object | 按创建时间查询的时间范围 |
| start_time | string | 创建起始时间，默认为7天前，格式：%Y-%m-%d |
| !end_time | string | 创建结束时间，默认为当天，格式：%Y-%m-%d |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] | 预约记录数据 |
| ad_id | number | 营销计划ID |
| creative_id | number | 营销创意ID |
| order_id | string | 预约ID |

| req_id | string | 请求ID |
| create_time | string | 创建时间，格式：%Y-%m-%d %H:%M:%S |
| page_info | object | 分页信息 |

| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
