# 查询项目关联结算单信息

代理商查询项目关联结算单信息

# 请求地址

[https://api.oceanengine.com/open_api/2/query/statement/](https://api.oceanengine.com/open_api/2/query/statement/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1783699726068811)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

agent_id

number

代理商ID

project_id_list

number[]

项目id列表，限制1000个
