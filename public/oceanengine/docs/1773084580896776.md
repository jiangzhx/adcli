# 获取评论视频ID列表

提供由mid查询对应的item_id的能力

该接口仅适用于评论管理，仅返回有评论内容的对应抖音视频。若相关素材id的关联item_id均无评论，则返回为空

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/comment/mid2item_id/](https://api.oceanengine.com/open_api/v3.0/tools/comment/mid2item_id/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773084580896776)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

start_time

string

查询起始时间，格式：yyyy-MM-dd，时间跨度最大三个月，仅支持获取2021-01-01之后的评论

end_time

string

查询截止时间，格式：yyyy-MM-dd，时间跨度最大三个月，仅支持获取2021-01-01之后的评论

material_id

number

素材ID

page

number

页码，默认值1

page_size

number

页面大小，默认值10，最大值100
