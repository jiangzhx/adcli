string[]

素材标签筛选项（如传入多个标签，取交集）

FIRST_PUBLISH_MATERIAL（首发素材）

is_live_rebate_type

string

是否直播返点类型（仅支持有效优质或有效首发查询时筛选，入参有效首投，该筛选需要置空） 可选值:

is_accumulation

string

是否累量 可选值:

is_valid_material

string

【政策粒度】是否有效素材 可选值:

policy_cost_min

number

【政策粒度】累量消耗范围区间下限 - 仅支持录入整数,单位：元

policy_cost_max

number

【政策粒度】累量消耗范围区间上限 - 仅支持录入整数,单位：元

material_is_effective

string

数据更新日期前一日是否在投 可选值:

material_first_effective_start_date

string

素材首投日期范围开始日期 格式：yyyy-MM-dd

material_first_effective_end_date

string

素材首投日期范围结束日期 格式：yyyy-MM-dd

this_period_cum_day_num

number

本期已累量天数
