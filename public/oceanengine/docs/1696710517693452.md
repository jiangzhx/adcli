# 二级代理商列表

获取代理商下的二级代理商ID列表。

### 请求地址

[https://ad.oceanengine.com/open_api/2/agent/child_agent/select/](https://ad.oceanengine.com/open_api/2/agent/child_agent/select/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710517693452)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 代理商ID |
| page | string | 页码.默认值: 1 |
| page_size | string | 页面数据量.默认值: 100 |
