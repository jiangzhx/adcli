# 更新安卓应用分包版本（支持所有账户体系）

通过账户id、账户类型和应用包id，更新全部或部分应用子包版本。支持AD、巨量引擎工作台-旧版及巨量引擎工作台-升级版客户。

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/app_management/extend_package/update_v2/](https://api.oceanengine.com/open_api/2/tools/app_management/extend_package/update_v2/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1785215984044233)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

account_id

number

账户ID，account_type对应的账户ID（近期升级到巨量引擎工作台-升级版的客户请传入应用包所属旧版巨量引擎工作台/子旧版巨量引擎工作台ID，账户类型传入BP）

account_type

string

账户类型，默认值: BP允许值：

package_id

string

应用包ID，可通过「[查询应用信息](https://open.oceanengine.com/labels/7/docs/1696710742760448)」接口查询获取

mode

string

更新分包模式，默认值: List，可选值:

channel_info_list

object[]

渠道号信息，（mode=List时需指定），一次调用list的size取值范围1～100

channel_id

string

渠道号，渠道号ID支持英文、数字、下划线和连字符-，不超过50个字符，超出部分会被截断。示例：oceanengine_ads_sample-12

remark

string

备注，渠道包备注信息，至多20个字符，超出部分会被截断处理

download_url

string

下载链接，（mode=Customize时需指定）渠道包（子包）的下载地址
