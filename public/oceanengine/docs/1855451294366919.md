# 升级版工作台查询素材共享范围

查询EBP组织素材的共享范围

# 请求地址

https://api.oceanengine.com/open_api/v3.0/tools/ebp/material/auth/list/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1855451294366919)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
