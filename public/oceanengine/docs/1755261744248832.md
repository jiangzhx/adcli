# 获取自定义报表可用指标和维度

获取自定义报表可用指标和维度

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/report/custom/config/get/](https://api.oceanengine.com/open_api/v3.0/report/custom/config/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1755261744248832)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

data_topics

string[]

数据主题查询列表，允许值：

BASIC_DATA营销基础数据

QUERY_DATA搜索词数据

BIDWORD_DATA关键词数据

MATERIAL_DATA素材数据

PRODUCT_DATA产品数据

ONE_KEY_BOOST_DATA一键起量（巨量营销升级版）

DMP_DATA人群包数据

VIDEO_DUARATION_DATA视频分秒数据

MATERIAL_BOOST_DATA 素材一键起量

e.printStackTrace();

.url(url)
.method("GET", null)
.addHeader("Access-Token", ACCESS_TOKEN)
.build();
Response response = client.newCall(request).execute();
return response.body().string();
}
