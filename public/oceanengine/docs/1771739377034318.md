# 获取订单数据报表

获取DOU+订单指标数据

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/douplus/order/report/](https://api.oceanengine.com/open_api/v3.0/douplus/order/report/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1771739377034318)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

aweme_sec_uid

string

抖音号ID，通过已授权账户接口可以获取

stat_time

object

时间范围

begin_time

string

开始时间，格式 yyyy-mm-dd

end_time

string

结束时间，格式 yyyy-mm-dd，时间跨度不能超过180天

注：返回的数据不包含end_time当天的数据，例：2024-01-01~2024-01-02，仅会返回2024-01-01号当天的数据。

group_by

string[]

分组方式，允许值：GROUP_BY_AD_ID 按订单分组（默认值）、GROUP_BY_CREATIVE_ID 按素材（视频/直播间）分组

目前支持传入组合方式：

filter

object

数据过滤

item_ids

number[]

视频ID

order_ids

number[]

订单ID，按id查询数据

备注：请优先使用order_id查询

task_ids

number[]

订单ID，（PC端能看到的订单号）

room_ids

number[]

直播间ID

author_unique_ids

string[]

达人ID

order_create_time

object

订单创建时间筛选

begin_time

string

开始时间，格式yyyy-mm-dd，日期最早可传 2022.01.01

end_time

string

结束时间，格式yyyy-mm-dd，结束时间不能早于开始时间

page_size

number

页面大小，允许值：1- 100，默认值：10

page

number

页码，默认值：1
