# 【代理商】查询营销违规信息

通过此接口，用户可以获取代理商账户下：

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/query/risk_promotion_list/](https://api.oceanengine.com/open_api/2/agent/query/risk_promotion_list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1790052406659072)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

agent_id

number

代理商账户ID

business_type

string

业务线

start_date

string

推送开始时间，支持格式：yyyy-MM-dd 或 yyyy-MM-dd HH:mm:ss，如果是 yyyy-MM-dd，默认 yyyy-MM-dd 00:00:00

比如上传：2024-03-01，默认转换查询2024-03-01 00:00:00

end_date

string

推送结束时间，支持格式：yyyy-MM-dd 或 yyyy-MM-dd HH:mm:ss，如果是 yyyy-MM-dd，默认 yyyy-MM-dd 23:59:59，

比如上传：2024-03-01，默认转换查询2024-03-01 23:59:59（最长跨度31天）

cursor

number

页码游标值，初始从Long.MAX开始，后续传入返回的cursor值，不传值相当于page=1，查询count条数据

count

number

页码游标值，最大支持500

filtering

object

过滤器

promotion_ids

number[]

营销ID，最多支持100个

promotion_name

string

营销名称，模糊搜索，长度不能超过30

promotion_status

string

营销状态 可选值:

illegal_material_ids

number[]

违规素材ids，最多支持100个

advertiser_ids

number[]

投放账户ID，最多支持100个

advertiser_name

string

投放账户名称，模糊搜索，长度不能超过30

final_operator_tag

string

自走收综合标签（T+1后数据稳定） 可选值:
