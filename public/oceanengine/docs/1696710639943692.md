# 编辑卡券

编辑卡券接口。 只有deliver_start、deliver_end、global_limit、user_limit允许编辑。 卡券开始后，所有的数据都不允许修改。 status字段控制卡券的上下线，只支持：NORMAL：上线，OFFLINE：下线

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/coupon/update/](https://ad.oceanengine.com/open_api/2/clue/coupon/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710639943692)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: "application/json" |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| activity_id 必填 | number | 青鸟营销活动ID |
| deliver_start | string | 活动开始时间，格式：%Y-%m-%d 小于结束时间，最多往前调一年 若小于当天则立即生效 活动开始后不允许修改 |
| deliver_end | string | 活动结束时间，格式：%Y-%m-%d，大于当前时间，最多往后调一年 deliver_end <= valid_end valid_end为优惠券的有效期结束时间 活动开始后不允许修改 |
| global_limit | object | 总库存量限制活动开始后不允许修改 |
| day_limit | number | 每日投放的总库存，不填表示不改，0表示不限制 |
| total_limit | number | 总投放库存限制，不填表示不改，0表示不限制 |
| user_limit | object | 单个用户的库存量限制活动开始后不允许修改 |
| day_limit | number | 用户每日可参与库存限制，不填表示不改，0表示不限制 |
| total_limit | number | 用户可参与总库存限制，不填表示不改，0表示不限制 |
| status | string | NORMAL：上线 OFFLINE： 下线 |
