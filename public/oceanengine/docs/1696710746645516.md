# 更新安卓应用分包版本

通过客户id和应用包id，更新全部或部分应用子包版本。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/extend_package/update/](https://ad.oceanengine.com/open_api/2/tools/app_management/extend_package/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710746645516)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| package_id必填 | string | 应用包ID，获取方法见接口文档[【查询应用信息】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710742760448) |
| mode必填 | string | 更新分包模式，All全部更新（将所有子包更新至最新母包版本）List部分更新（将指定子包更新至最新母包版本） |
| channel_list | object[] | 渠道号信息，（mode=List时需指定），一次调用，list的size取值范围1~100 |
| channel_id | string | 渠道号，渠道号ID支持英文，数字，下划线和连字符-，不超过50个字符，超出部分会被截断，示例：oceanengine_ads_sample-12 |
| remark | string | 备注，渠道包备注信息，至多20个字符，超出部分会被截断处理 |
