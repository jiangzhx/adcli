# 获取穿山甲客户分流联合实验数据

获取穿山甲的客户分流的联合实验数据

# 请求地址

[https://api.oceanengine.com/open_api/2/report/rta_cus_exp/get/](https://api.oceanengine.com/open_api/2/report/rta_cus_exp/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1789418134779913)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

.method("GET", null)

}
