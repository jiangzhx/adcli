# 巨量引擎开发工具

| 适用场景 | 根据开放平台提供接口文档进行功能开发时，可在IDE中配置用于模型检索文档内容生成代码示例 |
| --- | --- |
| 支持协议类型 | SSE |

## 配置示例

```
xxxxxxxxxx
{
"mcpServers": {
"巨量引擎开发工具": {
"url": "https://open.oceanengine.com/sse?app_id={app_id}&developer_id={developer_id}"
}
}
}
```

url中的{app_id}与{developer_id}需替换为开发者自身创建账户以及应用信息

## 工具列表

| 工具名称 | 工具介绍 |
| --- | --- |
| open_api_doc_gen | 用于检索召回巨量引擎开放平台的文档内容，该工具的返回值包含了检索到的文档内容以及接口对应的path |
| open_api_schema_gen | 依据接口path查询对应巨量引擎开放平台的接口schema，返回格式为OpenApi规范的yaml格式 |
| open_sdk_example_code_tool | 依据接口path查询对应接口的sdk调用代码，该sdk为巨量引擎开放平台提供的sdk，本工具需要用户明确要求按照sdk风格生成，返回值为sdk调用代码片段 |
