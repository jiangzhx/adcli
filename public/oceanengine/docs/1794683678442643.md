# 获取蓝海流量包

本接口支持查询搜索蓝海项目可用的蓝海流量包ID

您需要先通过本接口查询blue_flow_package_id，再前往「创建项目」接口创建搜索蓝海项目

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/blue_flow_package/list/](https://api.oceanengine.com/open_api/v3.0/tools/blue_flow_package/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1794683678442643)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

filtering

object

过滤器

blue_flow_package_name

string

蓝海流量包名称，支持模糊搜索

blue_flow_package_id

number

蓝海流量包ID，名称与ID同时传入则ID忽略名称筛选

字段

类型

json返回值

object[]

蓝海流量包信息列表

number

string

蓝海流量包名称

string

流量覆盖比例，返回整数（百分比），范围是0% ～ 100%（周维度更新）

number[]

预计触达流量区间（周维度更新）

number[]

预估消耗区间（周维度更新）

number[]

建议出价区间（周维度更新）

string

number[]
