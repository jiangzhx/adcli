# 获取穿山甲流量包

获取穿山甲流量包。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/union/flow_package/get/](https://ad.oceanengine.com/open_api/2/tools/union/flow_package/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710720565248)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| filtering | object | 过滤字段 |
| flow_package_ids | number[] | 按照流量包ID过滤，最大数量限制：100 |
| flow_package_type | string | 按照流量包类型过滤，允许值： CUSTOMIZE：自定义、 FEATURED：运营推荐、 SYSTEM： 系统推荐 |
| page | number | 页数 默认值: 1 |
| page_size | number | 页面大小 默认值: 10，最大值100 |

##### 应答字段

| 字段 | 类型 | 描述 |
