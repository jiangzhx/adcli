# 数据上报管理

事件管理相关的操作手册请查阅：[事件管理操作手册](https://event-manager.oceanengine.com/docs/8650/event_manager_docs/)

相关链接：

通过上报转化数据，可有效衡量营销投放效果，并且让营销系统找到更多高价值用户。根据投放目标，可分为线索和APP应用两大类： 线索转化跟踪:例如希望用户提交表单、拨打电话、在线咨询等，可选择的追踪方式包括JS布码、Xpath追踪、API回传 应用转化跟踪:例如希望用户激活app、注册、购物、付费等，可选择的追踪方式包括API回传、SDK回传
