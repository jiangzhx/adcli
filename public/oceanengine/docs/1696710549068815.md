# 数据报表

## 报表数据

数据报表是对账户内所有投放后的数据进行全方位，客观严谨的分析，通过比如消耗 、展示数 、点击数、转化数 、深度转化数等指标，方便客户进行数据分析，精准作出投放策略，以提高投放效果； 数据报表：客户数据、项目数据、营销计划数据、营销创意数据、视频素材数据（消耗数据，互动流失数据）、图片素材数据、关键词数据、搜索词数据、试玩素材数据；

## 数据报表使用方式：报表、监控通知、自动化调整

报表

定时拉取数据，按照不同维度以及环比，生成自定报表发送给运营同学，以帮助运营同学了解投放情况；

监控通知

某些组、计划、创意消耗达到预期的消耗或者成本，及时发通知（集成沟通平台：比如微信等）给到运营进行调整；

自动化调整

某些组、计划、创意消耗达到程序设定预期的消耗、成本、转化，然后进行调整出价、预算、或者是自动复制创建计划/创意，以满足整体的投放策略；
