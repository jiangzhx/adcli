# 建站工具——查询已有智能电话

通过此接口，用户可以获取已有智能电话实例列表，可用于建站。

列表栏包括智能电话实例ID、实例名称、是否为真实可接通的电话、电话id、电话号码、创建时间等信息。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/clue/smart_phone/get/](https://ad.oceanengine.com/open_api/2/tools/clue/smart_phone/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710621120524)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，范围：1 <= advertiser_id <= MAX_INT64 |
| page | number | 页码，默认为1，范围：page >= 1 |
| page_size | number | 页面大小，默认10条每页，范围：page_size >= 1 |

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| adv_key | string | 表征创建者信息的字段，历史为加密的advId |
| page_info | json | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| list | object | 智能电话列表 |
| instance_id | number | 智能电话组件ID |
| name | string | 智能电话名称 |
| call_display | number | 是否为真实可接通的电话： 0：表示不向商家显示用户真实手机号，而是虚拟号 1：表示商家看到用户的真实号码 |
| phone_id | number | 电话id |
| phone_number | number | 电话号码，支持固话(如01012345678)、手机号；暂不支持400号码 |
| create_time | string | 创建时间，格式：Y-m-d H:M:S |
| request_id | string | 请求日志id |
