# 查询白名单能力

支持客户通过接口查询客户是否命中各项灰度/白名单功能

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/gray/get/](https://api.oceanengine.com/open_api/v3.0/tools/gray/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1763407285579840)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

gray_keys

string[]

白名单能力key，目前仅支持单次查询1个白名单能力key，允许值见[【附录-枚举值-白名单能力key】](https://open.oceanengine.com/labels/7/docs/1696710760171535)

version

string

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

list

object[]

命中白名单请求

gray_key

string

白名单能力的唯一key

in_whitelist

number

是否命中白名单。1代表命中，0代表未命中

request_id

string

请求日志id
