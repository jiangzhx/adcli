# 引流下单转化信息获取

引流下单转化信息获取

# 请求地址

[https://api.oceanengine.com/open_api/2/adv_convert/ole/convert/](https://api.oceanengine.com/open_api/2/adv_convert/ole/convert/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1774836168252548)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| date_start_time 必填 | string | 订单所在自然日范围 开始时间，格式 yyyy-MM-dd。假设获取8.16数据，则填写2023-08-16。获取8.16-8.17数据，则填写2023-08-16。 |
| date_end_time 必填 | string | 订单所在自然日范围 结束时间，格式 yyyy-MM-dd。假设获取8.16数据，则填写2023-08-16。获取8.16-8.17数据，则填写2023-08-17。 |
| order_start_time 必填 | string | 下单时间范围 开始时间，可精确到分钟。格式 yyyy-MM-dd HH:mm:ss，示例 2023-08-16 12:00:00 |
| order_end_time 必填 | string | 下单时间范围 结束时间，可精确到分钟。格式 yyyy-MM-dd HH:mm:ss，示例 2023-08-16 13:00:00 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| order_infos | object[] | 订单明细 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
