# 获取微信库微信号列表

### 请求地址

[https://api.oceanengine.com/open_api/2/clue/wechat_pool/list/](https://api.oceanengine.com/open_api/2/clue/wechat_pool/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1738854207455240)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| filter | object | 过滤条件 |
| wechat_number | string | 微信号，模糊匹配 |
| nick_name | string | 微信昵称，模糊匹配 |
| instance_id | number | 微信号码包ID |
| audit_status | string[] | 审核状态，允许值: AUDITING: 审核中 AUDIT_REJECTED: 审核拒绝 AUDIT_ACCEPTED: 审核通过 |
| page | number | 分页，页数>=1，默认值1 |
| page_size | number | 分页大小，1-100，默认值20 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| items | object[] | 微信号列表 |
| wechat_number | string | 微信号 |
| wechat_type | string | 微信号类型。 允许值: PERSONAL: 个人号 PUBLIC: 公众号 ENTERPRISE: 企业微信 |
| nick_name | string | 微信昵称 |
| suffix | string | 尾缀 |
| audit_status | string | 审核状态 允许值: AUDITING: 审核中 AUDIT_REJECTED: 审核拒绝 AUDIT_ACCEPTED: 审核通过 |
| reject_reason | string | 审核拒绝理由 |
| has_qr_code | string | 是否配置二维码 允许值: TRUE: 是 FALSE: 否 |
| app_id | string | 公众号的开发者ID |
| instance_count | number | 生效号码包数量 |
| clue_count | number | 复制次数 |
| create_time | string | 创建时间，格式：2022-07-21 15:36:09 |
| mod_time | string | 修改时间，格式：2022-07-21 15:36:09 |
| status | number | 启用状态 0启用 1禁用 |
| total_count | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
