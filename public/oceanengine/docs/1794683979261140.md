# 获取项目下可用蓝海关键词

创建蓝海单元时，您可先通过本接口获取同项目下可用的蓝海关键词blue_flow_keyword_name，再调用「创建单元」接口传入

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/blue_flow_keyword/list/](https://api.oceanengine.com/open_api/v3.0/tools/blue_flow_keyword/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1794683979261140)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法
