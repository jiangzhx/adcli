# Oauth2.0授权

# 申请授权并获取Token

操作业务接口能力前，需要开发者向业务账户持有人申请授权并获取对应AccessToken后，才可以获取到可操作账户并请求相关业务接口

详细申请授权流程可参考：【[如何申请授权](https://open.oceanengine.com/labels/41/docs/1812506430648330)】

# 获取已授权账号下管理账户

授权完成后，即可开始进行业务接口接入调试，需要先根据接口确认已授权可操作账户范围

在完成授权并得到Token后，可通过【[获取已授权账号](https://open.oceanengine.com/labels/37/docs/1696710506574848)】接口查询AccessToken下具体授权的账号信息

账号：登录时对应手机号/邮箱号为账号，可使用同账号在不同业务平台登录查看账号在不同业务线下的管理账户

账户：不同业务下开通的账户，例如巨量投放账户、工作台账户、方舟账户等
