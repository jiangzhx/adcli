# 基于站点创建模板

可通过此接口基于已有落地页创建落地页模版

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site_template/create/](https://ad.oceanengine.com/open_api/2/tools/site_template/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1722273160450059)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST Request-Body

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| site_id必填 | number | 站点ID，可通过[【橙子建站】](https://www.chengzijianzhan.com/)平台或[【获取橙子建站站点列表】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710620579852#item-link-%E8%AF%B7%E6%B1%82%E5%9C%B0%E5%9D%80)接口获取 |
| template_name | string | 模板名称，默认模版名称为原站点名词 |
