# 批量删除预算组

通过此接口，可以批量删除预算组。

# 注意

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/budget_group/delete/](https://api.oceanengine.com/open_api/v3.0/budget_group/delete/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1780643886635008)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

number

客户id

number[]
