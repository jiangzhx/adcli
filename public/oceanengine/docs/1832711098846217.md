# 代理商提交授信充值

充值接口分为三步, 包含校验能否充值接口、提交授信充值接口、查询充值结果接口，本接口为第二步；提交之前需要校验能否充值，提交之后，通过轮训查询充值结果

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/credit_charge/submit/](https://api.oceanengine.com/open_api/2/agent/credit_charge/submit/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1832711098846217)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
