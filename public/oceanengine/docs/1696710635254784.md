# 创建表单

本接口用于创建表单，主要包括以下信息的设置：

##### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/form/create/](https://ad.oceanengine.com/open_api/2/clue/form/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710635254784)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：application/json |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |  |
| --- | --- | --- | --- |
| advertiser_id 必填 | number | 客户id |  |
| name 必填 | string | 表单名称，不超过20个字符 |  |
| validate_type 必填 | string | 线索验证类型，主要指C端是否进行短信验证，表单中含有电话类型时生效 可选值：CLUE_PRIORITY（线索量优先）、VALIDITY_PRIORITY（有效性优先）、NONE_VERIFICATION（全不出）、AUTO_VERIFICATION（智能验证，历史逻辑，效果等同于全不出）、ALL_VERIFICATION（全出），详情见[【附录-青鸟表单线索验证类型】](https://openapi.bytedance.net/openapi/doc/index.html?id=528) |  |
| submit_text 必填 | string | 提交文案说明，如“提交表单”，文案长度不超过8个字符 |  |
| enable_layer | number | 是否开启分层，默认不开启 枚举值：0，1（0表示不启用，1表示启用） |  |
| layer_submit_text | string | 分层提交文案说明，分层信息在Elements中，当不开启分层时，所有分层信息无效。文案长度不超过8个字符 |  |
| elements 必填 | json | 表单元素 |  |
| label 必填 | string | 元素标签名称，如“姓名“等，文案长度不超过8个字符 |  |
| element_type 必填 | string | 元素类型 可选值：NAME（姓名）、TELEPHONE（电话）、EMAIL（邮箱）、NUMBER（数值）、SEX（性别）、DATE（日期）、CITY（城市）、TEXT（文本）、TEXTAREA（文本域） |  |
| sequence | number | 元素排序，即元素流式展示分布的先后顺序，默认按list顺序 |  |
| default_value | number | 元素默认值，目前已开放的元素类中，目前仅支持element_type=NUMBER，即数值类型时填写，表示默认值 |  |
| layer | number | 是否分层元素，和enable_layer、layer_submit_text结合用 可选值：0，1（0表示不是，1表示是） |  |
| allow_empty | number | 是否允许为空，默认允许 可选值：0，1（0表示不允许，1表示允许） |  |
| is_unique | number | 同一表单是否限制不可重复提交，默认可重复 可选值：0，1（0表示可重复，1表示唯一） |  |
| extend_info | json | 扩展属性 |  |
| count_config | json | 计数配置，包括递增/递减、计数开始值、计数文案模版 |  |
| count_type | string | 计数方式 可选值：COUNT_TYPE_INCREMENT（递增计数）、COUNT_TYPE_DECREMENT（递减计数） |  |
| prefix | string | 计数文案前缀，例如：“截止目前已有”，文案长度不超过100个字符 |  |
| start_num | number | 起始数值 |  |
| suffix | string | 计数文案后缀，例如：“名用户参加”，文案长度不超过100个字符 |  |
| sign_up_config | json | 报名信息配置 |  |
| sign_type | string | 表单报名配置展示形式 SIGN_TYPE_SCROLL_WALL（滚动墙展示）、SIGN_TYPE_SCROLL_BAR（滚动条展示） |  |
| success_tip | string | 提交成功后的文案提示，文案长度不超过128个字符 |  |
