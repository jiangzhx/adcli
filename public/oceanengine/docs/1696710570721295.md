# 人群包列表

通过此接口你可以查询客户下存在的人群包列表和信息。

信息包括人群包的id，可用状态，来源，覆盖人群等； 该接口可以查到客户id下所有人群包，无论是API创建的还是平台创建的。

### 请求地址

[https://ad.oceanengine.com/open_api/2/dmp/custom_audience/select/](https://ad.oceanengine.com/open_api/2/dmp/custom_audience/select/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710570721295)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| select_type 必填 | number | 查询类型,枚举值："0"：该客户创建的人群包和被推送给该客户的人群包，"1"：状态为可投放的人群包 |
| offset | number | 偏移,类似于SQL中offset(起始为0,翻页时new_offset=old_offset+limit），默认值：0，取值范围:≥ 0 |
| limit | number | 返回数据量，默认值：100，取值范围：1-100 |

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | 返回数据信息 |
| custom_audience_list | object[] | 人群包列表数据 |
| custom_audience_id | number | 人群包ID |
| isdel | number | 人群包是否删除，枚举值："1"：已删除，"0"：未删除 |
| data_source_id | string | 数据源ID |
| name | string | 人群包名称 |
| source | string | 人群包来源，详见[【附录-DMP相关-人群包来源】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710761459712#custom-anchor-%E4%BA%BA%E7%BE%A4%E5%8C%85%E6%9D%A5%E6%BA%90) |
| status | number | 人群包状态，详见[【附录-DMP相关-人群包状态】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710761459712#custom-anchor-%E4%BA%BA%E7%BE%A4%E5%8C%85%E7%8A%B6%E6%80%81) |
| delivery_status | string | 人群包可投放状态，只有当状态为CUSTOM_AUDIENCE_DELIVERY_STATUS_AVAILABLE时才可进行投放使用 可选值： CUSTOM_AUDIENCE_DELIVERY_STATUS_AVAILABLE：可投放，人群包发布完成且推送完成 CUSTOM_AUDIENCE_DELIVERY_STATUS_NEED_PUSH：不可投放，人群包发布完成，但未推送，需要进行推送后再使用 CUSTOM_AUDIENCE_DELIVERY_STATUS_NEED_PUBLISH：不可投放，群包未发布但已推送，需要进行发布后再使用 CUSTOM_AUDIENCE_DELIVERY_STATUS_UNAVAILABLE：不可投放，未发布完成且未推送 |
| cover_num | number | 人群包覆盖人群数目，基于"upload_num"：上传数据源包含的人群数目与uid对应后，再与头条系产品MAU交集后的数量（存在一个设备号/手机号对应多个uid的情况） 实际数量可能多于/少于"upload_num"：上传数据源包含的人群数目 |
| upload_num | number | 上传数据源包含的人群数目 |
| tag | string | 人群包标签，通过数据源创建的人群包，标签会默认为"API文件数据源" |
| offset | number | 偏移,类似于SQL中offset(起始为0,翻页时new_offset=old_offset+limit） |
| total_num | number | 总的人群包数量 |
| request_id | string | 请求日志id |
