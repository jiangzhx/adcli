# 获取商品库元信息

获取DPA元信息

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/meta/get/](https://ad.oceanengine.com/open_api/2/dpa/meta/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710577566735)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| platform_id 必填 | number | 商品库id |

public static JSONObject getDpaMeta() {
String access_token = "xxx";
final Long advertiser_id = 0L;

​
// 请求参数

}
};
​
// 构造请求
HttpEntityEnclosingRequestBase httpEntity = new HttpEntityEnclosingRequestBase() {
@Override

return "GET";
}
};
​
httpEntity.setHeader("Access-Token",access_token);
​
CloseableHttpResponse response = null;
CloseableHttpClient client = null;
​
try {
client = HttpClientBuilder.create().build();
httpEntity.setURI(URI.create(open_api_url_prefix + uri));
httpEntity.setEntity(new StringEntity(JSONObject.toJSONString(data), ContentType.APPLICATION_JSON));
​
response = client.execute(httpEntity);
if (response != null &amp;&amp; response.getStatusLine().getStatusCode() == 200) {
BufferedReader bufferedReader  = new BufferedReader(new InputStreamReader(response.getEntity().getContent()));
StringBuffer result = new StringBuffer();

}
bufferedReader.close();
return JSONObject.parseObject(result.toString());
}

} catch (ClientProtocolException e) {

##### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| list | list | json返回值 |
| status | int | 元信息状态 |
| name | string | 元信息名称 |
| title | string | 元信息头 |
| media_type | int | 元信息媒体类型 1-text 2-image 3-video 4-url |
| field_group | string | 分组 |
| type | string | 字段类型 |
