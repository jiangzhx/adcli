# 更新视频

通过此接口，用户可以批量更新素材视频的名称。

### 请求地址

[https://ad.oceanengine.com/open_api/2/file/video/update/](https://ad.oceanengine.com/open_api/2/file/video/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710605909004)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| videos 必填 | object[] | 限制最多 100 个 |
| filename 必填 | string | 视频素材名称，最多255个字，更新时必填 |
| video_id 必填 | string | 视频id，更新必填 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| videos | object[] | 视频列表 |
| video_id | string | 视频id |
| status_code | string | 更新状态 SUCCESS 成功 FAILURE 失败（原因：操作视频非当前客户所有或操作的视频不存在） |
| request_id | string | 请求的日志id，唯一标识一个请求 |
