# 获取模版预览链接

通过此接口，用户可以预览通过[【基于站点创建模板】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1722273160450059)接口创建的落地页模板

落地页模板的预览链接有效时间为20分钟

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site_template/preview/](https://ad.oceanengine.com/open_api/2/tools/site_template/preview/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1722273233146891)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET Query-String

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| template_id必填 | number | 模板ID，可通过[【获取站点模版列表】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1722273219292292)查询模板ID |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| preview_url | string | 预览URL，模板的预览地址，有效期：20分钟 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
