# 删除人群包

通过此接口可做人群包删除操作。已经在计划中使用的人群包不能被删除，只有该计划被删除后，人群包才可以删除。

### 请求地址

[https://ad.oceanengine.com/open_api/2/dmp/custom_audience/delete/](https://ad.oceanengine.com/open_api/2/dmp/custom_audience/delete/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710572836879)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：application/json |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| custom_audience_id 必填 | number | 人群包ID |
