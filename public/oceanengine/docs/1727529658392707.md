# 转化回传鉴权使用说明

## 转化回传鉴权是什么？

「转化回传鉴权」是一个安全性功能，建议有技术能力的客户接入使用，其本质是对数据回传方进行身份认证校验。完成技术对接后，客户可以生成特定密钥，使用密钥作为身份验证完成数据回传，可以有效避免恶意冒名回传请求风险，进一步保障客户的数据安全和数据回传质量。

## 如何完成转化回传鉴权？

面向客户：暂只面对已有接入Marketing API能力的客户，且有数据回传验证需求的客户。
