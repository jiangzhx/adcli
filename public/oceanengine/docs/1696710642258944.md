# 添加核销员

添加核销员接口

### 接口示例

接口示例

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/coupon/employee/create/](https://ad.oceanengine.com/open_api/2/clue/coupon/employee/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710642258944)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: "application/json" |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| list 必填 | object[] | 核销员列表 |
| user_id 必填 | number | 用户ID |
| user_type 必填 | string | 用户类型 TOUTIAO:头条用户 DOUYIN:抖音用户 |
| store_id | number | 门店ID，不填或0代表不限制门店 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| existed_list | object[] | 本次添加已存在的核销员列表 |
| user_id | number | 用户ID |
| user_type | string | 用户类型 TOUTIAO:头条用户 DOUYIN:抖音用户 |
| store_id | number | 门店ID，0代表不限制门店 |
| success_list | object[] | 本次添加成功的核销员列表 |
| user_id | number | 用户ID |
| user_type | string | 用户类型 TOUTIAO:头条用户 DOUYIN:抖音用户 |
| store_id | number | 门店ID，0代表不限制门店 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
