# 查询账号余额

获取客户或代理商账户余额信息。

### 请求地址

[https://ad.oceanengine.com/open_api/2/advertiser/fund/get/](https://ad.oceanengine.com/open_api/2/advertiser/fund/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710526192652)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID或代理商ID |
| grant_type_split | string | 是否拆分赠款类型，允许值： 开启 ON ，关闭 OFF（默认） |

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| advertiser_id | number | 客户ID或代理商ID |
| name | string | 账户名 |
| email | string | 联系邮箱 |
| balance | number | 账户总余额(单位元) |
| valid_balance | number | 账户可用总余额(单位元) |
| cash | number | 现金余额(单位元) |
| valid_cash | number | 现金可用余额(单位元) |
| grant | number | 赠款余额(单位元) |
| default_grant | float | 默认投放版位可用赠款余额(单位元) |
| common_grant | float | 通用投放版位可用赠款余额(单位元) |
| search_grant | float | 搜索投放版位可用赠款余额(单位元) |
| union_grant | float | 穿山甲投放版位可用赠款余额(单位元) |
| valid_grant | number | 赠款可用余额(单位元) |
| return_goods_abs | number | 返货余额(单位元)，仅支持部分客户 |
| valid_return_goods_abs | number | 返货可用余额(单位元)，仅支持部分客户 |
| return_goods_cost | number | 返货支出(单位元)，仅支持部分客户 |
| return_goods_grant | number | 返货赠款 |
| compensation_grant | number | 赔付赠款 |
| return_goods_valid_grant | number | 返货可用赠款 |
| compensation_valid_grant | number | 赔付可用赠款 |
| wallet_id | string | 钱包id（投放账户绑定的共享子钱包id） |
| wallet_name | string | 钱包名称（投放账户绑定的共享子钱包名称） |
| wallet_total_balance_valid | number | 账户绑定的子钱包的可用共享余额（单位元） |
| request_id | string | 请求日志id |
