# 私信线索转化回传接口

使用本途径回传的isv/客户需同时满足以下两个条件：

本接口允许服务商、代理商、crm等（即和客户主体不一致的主体）回传私信线索转化数据。

# 调用本接口前需要的准备步骤

需要申请的权限点 300000017

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/clue/private_message/callback/](https://api.oceanengine.com/open_api/2/tools/clue/private_message/callback/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1826462960148556)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求参数

字段

类型

描述

advertiser_id

number

event_timestamp

number

c_open_id

string

留资用户的抖音open id，对齐私信拉取接口的相关open_id。如果是飞鸽商家此字段可以传空字符串，填写下方飞鸽相关字段即可。

b_open_id

string

接待商家的抖音open id，对齐私信拉取接口里面的 open_id。如果是飞鸽商家此字段可以传空字符串，填写下方飞鸽相关字段即可。

client_key

string

抖开接口里面的开发者唯一标识，对齐私信拉取接口里面的 client_key。如果是飞鸽商家此字段可以传空字符串，填写下方飞鸽相关字段即可。

adinfo

object

adv_id

number

客户id，线索对应的客户id，传[抖开拉会话接口](https://developer.open-douyin.com/docs/resource/zh-CN/mini-app/develop/tutorial/open-capabilities/management/im-messaging-access)里面的adv_id即可

number

promotion_id

number

server_message_id

string

产生线索的消息 ID，和[抖开拉会话接口](https://developer.open-douyin.com/docs/resource/zh-CN/mini-app/develop/tutorial/open-capabilities/management/im-messaging-access)里面的content字段的相关信息对齐即可

## http接口返回200就说明我回传成功了吗？

不是，返回里面的code=0才说明回传成功。具体见本文档「应答示例」部分。
