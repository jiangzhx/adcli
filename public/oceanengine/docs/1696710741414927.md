# 应用管理

相关链接：

1、游戏预约介绍 游戏预约一般是精品游戏在正式发布前进行预热/冲榜进行的投放，由于圈定人群为高意愿游戏人群，整体在后端的留存、付费效果表现上都高于普通用户，因此是精品游戏投放中的重要一环。之前常见于应用商店、游戏社群当中，而预约营销投放属于新兴的一种投放方式。

2、API开放能力 本篇主要介绍应用管理的预约模块，通过预约管理会实现线上化预约全流程打通。 目前API只支持查询应用预约记录，用户可以通过该接口实现预约记录的查询。预计7月底可以支持更多管理接口功能。 此功能目前是白名单功能，如果需要开通可以咨询对接的销售运营；
