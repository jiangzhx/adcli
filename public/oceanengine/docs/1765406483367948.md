# 获取营销起量状态

批量获取营销当前的一键起量状态

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/promotion_raise_status_current_ids/get/](https://api.oceanengine.com/open_api/v3.0/tools/promotion_raise_status_current_ids/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1765406483367948)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

promotion_ids

number[]

营销ID，支持单次获取最多100条

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

message

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

data

object[]

promotion_id

number

营销ID

status

string

当前一键起量状态 可选值:

raise_info

object

当前的起量信息， 当status=RAISING时返回该字段

raise_end_time

string

预计结束时间，格式为yyyy-mm-dd HH:MM

raise_budget

double

起量预算

error_list

object[]

promotion_id

number

获取一键起量状态失败的营销ID

error_reason

string

获取一键起量状态失败的错误原因

request_id

string

请求日志id
