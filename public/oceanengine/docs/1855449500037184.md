# 升级版工作台更新视频

EBP场景下，更新组织节点中视频信息

# 请求地址

https://api.oceanengine.com/open_api/v3.0/tools/ebp/video/update/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1855449500037184)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

POST

number

视频对应素材ID
