# 获取字节小程序link详情

获取归属升级版巨量引擎工作台的字节小程序link详情

# 请求地址

https://api.oceanengine.com/open_api/v3.0/tools/ebp/micro_applet/link/list/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1847487745198092)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

GET

类型

描述

string

start_time

number

创建起始时间，格式：%Y-%m-%d

开始时间 >= 格林威治时间（GMT）的秒

end_time

number

创建结束时间，格式：%Y-%m-%d

结束时间 <= 格林威治时间（GMT）的秒
