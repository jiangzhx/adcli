# 获取橙子建站站点详细信息

通过此接口，用户可以获取站点的详细信息，包括新建或更新时传递的全量数据。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site/read/](https://ad.oceanengine.com/open_api/2/tools/site/read/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710620022799)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，范围：1 <= advertiser_id <= MAX_INT64 |
| site_id 必填 | string | 橙子建站站点id列表 ：1 <= 长度 <= 20 可通过[【获取橙子建站站点列表】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710620579852)获取 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| bricks | object | 具体见返回示例业务数据（新建或更新时传递的数据） |
| id | string | 站点ID |
| status | string | 站点状态，详见[【附录-枚举值-站点状态】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E7%AB%99%E7%82%B9%E7%8A%B6%E6%80%81) |
| site_type | string | 建站类型，详见[【附录-枚举值-建站类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%BB%BA%E7%AB%99%E7%B1%BB%E5%9E%8B) |
| thumbnail | string | 缩略图（只有发布的站点才会生成） |
| request_id | string | 请求日志id |
