# 查询短剧可投状态

通过短剧alblumID查询短剧可投状态

特别注意：请求本接口时需要使用APP Access Token，通过【[获取APP Access Token](https://ad.oceanengine.com/openapi/doc/index.html?id=1713655428885516)】接口获取

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/dpa/album_status/get/](https://api.oceanengine.com/open_api/v3.0/dpa/album_status/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1830811308177048)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

App-Access-Token

string

应用级token，区别于普通的Access Token，通过【[获取APP Access Token](https://ad.oceanengine.com/openapi/doc/index.html?id=1713655428885516)

】接口获取
