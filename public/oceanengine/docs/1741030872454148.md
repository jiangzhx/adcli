# 批量更新营销预算

批量更新 巨量引擎体验版 营销预算

不支持搜索周期稳投营销修改预算（项目层级delivery_type = DURATION）

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/budget/update/](https://api.oceanengine.com/open_api/v3.0/promotion/budget/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1741030872454148)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

POST Request-Body

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type | string | 请求消息类型，允许值：application/json |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

## 基础参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| data 必填 | object[] | 批量修改预算，list长度限制1～10 |
| promotion_id 必填 | number | 营销ID |
| budget_mode | string | 该营销所属的预算类型, （预算类型不支持修改，且需要上传）。允许值: BUDGET_MODE_DAY日预算（默认值） BUDGET_MODE_TOTAL总预算 |
| budget 必填 | float | 预算（总预算或日预算） 出价方式为OCPM时，不少于300元；24小时内修改预算操作，不能超过20次，24小时是指自然天的24小时；单次修改预算幅度不能低于100元（增加或者减少）；修改后预算金额，不能低于当前已消费金额的105%，以整百单位向上取整；取值范围: ≥ 0 |
