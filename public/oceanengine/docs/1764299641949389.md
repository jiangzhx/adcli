# 获取一键起量报告

巨量营销升级版-获取一键起量报告功能通过[「自定义报表」](https://open.oceanengine.com/labels/7/docs/1741387668314126)接口提供

您可以在该接口下输入参数data_topic=ONE_KEY_BOOST_DATA一键起量（巨量营销升级版），使用该功能

详细内容请点击查看[「自定义报表」](https://open.oceanengine.com/labels/7/docs/1741387668314126)接口文档
