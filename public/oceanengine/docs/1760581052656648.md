# 批量新增营销否定词

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/privative_word/promotion/add/](https://api.oceanengine.com/open_api/v3.0/tools/privative_word/promotion/add/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1760581052656648)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| promotion_list 必填 | object[] | 项目列表 |
| promotion_id 必填 | number | 项目id |
| phrase_words | string[] | 短语否定词列表，搜索直投场景下否定词列表不能超过500个否定词，单个否定词长度不能超过30个汉字（或60个英文字符），不能包含emoji，不能为空字符串。 |
| precise_words | string[] | 精确否定词列表，搜索直投场景下否定词列表不能超过500个否定词，单个否定词长度不能超过30个汉字（或60个英文字符），不能包含emoji，不能为空字符串。 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回信息 |
| error_list | object[] | 添加失败的否定词详情 |
| promotion_id | number | 添加失败的营销id |
| error_message | string | 营销维度的错误原因 |
| fail_phrase_words | object[] | 失败的短语否定词列表 |
| word | string | 短语否定词 |
| reason | string | 词维度的失败原因 |
| fail_precise_words | object[] | 失败的精确否定词列表 |
| word | string | 精确否定词 |
| reason | string | 词维度的失败原因 |
| promotion_list | object[] | 营销中添加成功的否定词列表 |
| promotion_id | number | 营销id |
| status | string | 添加结果，status=success时表示该项目的否定词全部添加成功，status= failure时表示否定词部分添加失败或全部失败，添加成功的否定词会返回在success_words |
| success_words | object[] | 添加成功的否定词 |
| phrase_words | string[] | 添加成功的短语否定词 |
| precise_words | string[] | 添加成功的精确否定词 |
| promotion_error_list | number[] | 营销维度原因导致添加失败的项目ID列表 |
