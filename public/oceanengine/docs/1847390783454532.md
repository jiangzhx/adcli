# 查询安卓应用母包详情

可通过当前接口查询归属升级版巨量引擎工作台的安卓母包详情。

https://api.oceanengine.com/open_api/v3.0/tools/ebp/app/detail/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1847390783454532)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
