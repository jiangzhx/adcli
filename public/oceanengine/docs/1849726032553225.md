# 查询游戏玩法列表

适用场景：支持通过账户id和资产id，获取资产的副线玩法列表，返回数据仅包含审核通过的玩法

资产范围：安卓游戏、抖音小游戏、微信小游戏

# 请求地址

https://api.oceanengine.com/open_api/v3.0/gameplay/list/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1849726032553225)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

GET

类型

描述

number

账户id，需传入有权限使用资产的账户id（创建资产的账户或bp下的账户，或申请过资产共享）

account_type

string

账户类型，可选值:

asset_id

string

资产id，可通过应用、小游戏列表获取接口获取资产id

安卓应用：应用列表接口出参package_id

抖音/微信小游戏：列表接口出参instance_id

asset_type

string

filter_play_names

string[]

玩法名称筛选，支持模糊搜索

page_info

object

分页信息

page

number

页码，默认值为1

page_size

number

页面大小，默认值为10，最大值100
