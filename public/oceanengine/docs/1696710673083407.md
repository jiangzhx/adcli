# 更新组件

该接口用户更新创意组件信息。

组件信息发生变更后，提交变更保存成功，会触发组件以及组件关联的所有计划的 重审逻辑、组件状态、以及组件关联的所有计划状态会变为审核中，请重点关注

# 请求地址

[https://ad.oceanengine.com/open_api/2/assets/creative_component/update/](https://ad.oceanengine.com/open_api/2/assets/creative_component/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710673083407)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| component_id必填 | number | 组件ID |

| component_type必填 | string | 【组件类型枚举】 可选值： CHOICE_MAGNET 选择磁贴 VOTE_MAGNET 投票磁贴 IMAGE_MAGNET 图片磁贴 COMMERCE_MAGNET 电商磁贴 COUPON_MAGNET 优惠券磁贴 GAME_PACK 游戏礼包码 PROMOTION_CARD 投放卡片 RESERVATION_BUTTON 游戏预约按钮 UNION_LIGHT_INTERACTIVE 穿山甲轻互动组件 ECOMMERCE_CARD卡片-展示价格力 HALF_INTERACTIVE半屏-点击互动类 |
| component_name必填 | string | 组件名称。长度小于等于20。 一个中文长度为2 |
| component_data必填 | json | 组件详细信息。不同的component_type的component_data不一样 |

### style_type和interactive_type对应关系

| style_type-互动前玩法 | interactive_type-搭配的互动后玩法 | 备注 |
| --- | --- | --- |
| 1-翻卡 | 1-全局优惠、2-礼包码、4-单品优惠、5-新人优惠 |  |
| 2-盲盒 | 1-全局优惠、2-礼包码、4-单品优惠、5-新人优惠 |  |
| 3-翻页 | 1-全局优惠、2-礼包码、4-单品优惠、5-新人优惠 |  |
| 4-擦除 | 3-配对 | 当style_type为4时，表示擦除玩法，仅限搭配interactive_type为3，表示配对玩法。其他可以任意搭配，否则会报【 "message": "互动前玩法类型与互动后玩法类型不匹配：配对仅支持擦除玩法"】。 |
| 5-刮刮卡 | 1-全局优惠、2-礼包码、4-单品优惠、5-新人优惠 |  |
