# 橙子建站落地页数据

获取橙子建站和程序化落地页的数据。不包含第三方落地页。

该接口披露橙子建站落地页用户行为关键指标分日数据，包括： 页面访问量:PV 用户访问量:UV 转化率:单元convert数/单元click事件 平均加载时长:页面平均加载渲染时长 平均阅读比例:用户平均浏览落地页的比例 平均访问时长:平均在落地页的停留时长

落地页数据以当日前两天的方式返回，如：当3号入参查询周期1天，返回1号数据

### 请求地址

[https://ad.oceanengine.com/open_api/2/report/site/page/](https://ad.oceanengine.com/open_api/2/report/site/page/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710565390348)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| site_id 必填 | string | 站点ID |
| time_duration 必填 | string | 查询时间周期。 DAY:1天。 THREE_DAY: 3天。 WEEK:7天。 MONTH:30天。 |
| inventory_type | string | 首选投放位置。不传默认查全部。 WATERMELON:西瓜。 HOTSOON:火山。 AWEME:抖音。 TOUTIAO: 头条。 UNION_SLOT:穿山甲。 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | 返回值 |
| site_id | string | 站点ID |
| site_record | json | 数据列表 |
| key | object[] | 指标名称。[【建站报表落地页指标名称】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%BB%BA%E7%AB%99%E6%8A%A5%E8%A1%A8%E8%90%BD%E5%9C%B0%E9%A1%B5%E6%8C%87%E6%A0%87%E5%90%8D%E7%A7%B0)。 |
| value | float | 指标值 |
| date_time | string | 时间。格式: "2021-03-14" |
