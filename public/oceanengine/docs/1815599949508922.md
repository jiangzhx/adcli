# 定向包查询关联项目信息

定向包查询关联项目信息

可通过【[获取定向包](https://open.oceanengine.com/labels/34/docs/1815573577937932)】接口获取定向包ID，根据定向包ID查询该定向包关联了哪些项目

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/audience_package_bindinfo/get/](https://api.oceanengine.com/open_api/v3.0/audience_package_bindinfo/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1815599949508922)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
