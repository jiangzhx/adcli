# 建站工具-建站复制

通过此接口，用户可以实现站点的复制功能，成功后生成一个新站点id，站点内容和原站点一致。 注意

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site/copy/](https://ad.oceanengine.com/open_api/2/tools/site/copy/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710623908864)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，数字范围：1 <= advertiser_id <= MAX_INT64 |
| site_ids必填 | string[] | 站点id列表, min_size=1, max_size=20 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| success_list | object | 复制成功列表，整体失败不返回该列表 |
| origin_site_id | string | 返回复制成功的原站点id |
| site_id | string | 返回复制成功后生成的新站点id |
| error_list | object[] | 复制失败列表，整体成功不返回该列表 |
| origin_site_id | string | 返回复制失败后的原站点id |
| error_reason | string | 当对origin_site_id的操作失败时，返回详细原因，成功无 |
| request_id | string | 请求日志id |
