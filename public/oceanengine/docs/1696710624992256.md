# 获取第三方落地页站点列表

通过此接口，客户可以获取客户下拥有的第三方落地页站点列表。

列表信息包含站点审核状态、站点创建时间、站点名称、站点ID、缩略图地址、站点地址。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/third_site/get/](https://ad.oceanengine.com/open_api/2/tools/third_site/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710624992256)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id |
| page | number | 页码,默认值: 1，范围：page >= 1 |
| page_size | number | 页面数据量，默认值: 20 ,取值范围：1～100 |
| filtering | object | 过滤字段 |
| site_id | string | 搜索字段，按照建站id进行模糊匹配 |
| site_name | string | 搜索字段，按照建站name进行模糊匹配 |
| start_time | string | 时间过滤条件：时间范围内创建的落地页，开始时间,默认七天前, 时间格式 %Y-%m-%d |
| end_time | string | 时间过滤条件：时间范围内创建的落地页，结束时间，默认到当天,时间格式 %Y-%m-%d |
