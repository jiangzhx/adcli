# 查询订单列表

查询DOU+订单属性

# 近期更新

2024/1/11，请求参数新增排序条件order_by_field、order_by_type参数，支持按照订单ID排序

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/douplus/order/list/](https://api.oceanengine.com/open_api/v3.0/douplus/order/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1771736421276680)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

aweme_sec_uid

string

抖音号ID，通过已授权账户接口可以获取

filter

object

筛选条件

scene_type

string

按营销目标过滤，允许值：LIVE 直播、VIDEO 短视频

order_id

number[]

订单ID，按id过滤数据

备注：请优先使用order_id查询

task_id

number[]

订单ID（PC端能看到的订单号），按id过滤数据

status

string[]

DOU+订单状态，允许值：

UNPAID 未支付

AUDITING 审核中

OFFLINE_AUDIT审核不通过

TIME_NO_REACH 待开播

DELIVERING 投放中

UNDELIVERIED 投放终止

DELIVERIED 投放完成/结束

AUDIT_PAUSE 审核暂停

WAIT_TO_START 等待开始

order_create_time

object

订单创建时间筛选

start_time

string

起始时间，格式为yyyy-MM-dd，开始时间不能大于结束时间

end_time

string

结束时间，格式为yyyy-MM-dd

注：返回的数据不包含end_time当天的数据，例：2024-01-01~2024-01-02，仅会返回2024-01-01号当天的数据。

page_size

number

页面大小，允许值：1- 100，默认值：10

page

number

页码，默认值：1

order_by_field

string

order_by_type

string
