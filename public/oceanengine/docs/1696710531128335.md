# 获取账户日预算

此接口可以获取客户账号设置的预算类型与预算，可以一次查询100个客户账号预算

当预算类型为不限，返回的预算为0.0元

### 请求地址

https://ad.oceanengine.com/open_api/2/advertiser/budget/get/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710531128335)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_ids 必填 | number[] | 客户id列表，长度限制：[1,100] |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] |  |
| advertiser_id | number | 客户ID |
| budget | float | 预算，单位：元； 精度：小数点后两位；举例：100.01。 当预算类型为不限，返回的预算为0.0 |
| budget_mode | string | 详见[【附录-预算类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E9%A2%84%E7%AE%97%E7%B1%BB%E5%9E%8B)，可选值: BUDGET_MODE_DAY（日预算）BUDGET_MODE_INFINITE（不限） |
| request_id | string | 请求的日志id，唯一标识一个请求 |
