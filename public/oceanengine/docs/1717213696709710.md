# 获取预估点击成本

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/estimated_price/get/](https://ad.oceanengine.com/open_api/2/tools/estimated_price/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1717213696709710)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【获取Access-Token】 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |

# 应答参数

| 字段 | 类型 | 描述 |  |
| --- | --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |  |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |  |
| data | json | json返回值 |  |
| upper_bound | float | 建议出价上界，单位为元 |  |
| lower_bound | float | 建议出价下界，单位为元 |  |
| cpc_price | float | 回填建议出价，单位为元 |  |
| request_id | string | 请求日志id |  |
