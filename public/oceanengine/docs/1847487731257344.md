# 新建微信小游戏

创建的微信小游戏资产将归属于升级版巨量引擎工作台，默认下级组织的账户可用

请您确保您已经上传了投放资质，且您上传投放资质的投放账户认证的公司主体，需与资产创建流程中选择的公司主体一致，以用于资质审核。若未提交投放资质，请通过【[投放资质提交接口](https://open.oceanengine.com/labels/7/docs/1761501324513352)】提交资质，否则资产审核将不通过。资质信息可以通过【[获取投放资质信息](https://open.oceanengine.com/labels/7/docs/1761500990156815?origin=left_nav)】接口查询。
