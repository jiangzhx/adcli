# 更新定向包

# 请求地址

[https://ad.oceanengine.com/open_api/2/audience_package/update/](https://ad.oceanengine.com/open_api/2/audience_package/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710732349455)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

| ac | string[] | 受众网络类型, 详见[【附录-受众网络类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535) |
| device_brand | string[] | 受众手机品牌, 详见[【附录-手机品牌】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535) |
| platform | string[] | 受众平台(当营销目的landing_type=APP时,不填,且为保证投放效果,平台类型定向PC与移动端互斥)，android_osv或ios_osv非空时，必填。详见[【附录-受众平台类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535) 新增：HARMONY鸿蒙，满意以下条件(同时满足)时支持设置 landing_type = APP_HARMONY marketing_goal = VIDEO_AND_IMAGE ad_type = ALL / SEARCH |
| auto_extend_targets | string[] | 智能放量定向，该字段不为空时表示开启智能放量，传空时表示不开启智能放量 允许值：REGION（地域）、GENDER（性别）、AGE（年龄）、CUSTOM_AUDIENCE（自定义人群） |
| launch_price | number[] | 手机价格定向,传入价格区间，最高传入11000（表示1w以上） 传值示例 "launch_price": [2000, 11000]，表示2000元以上 |

| interest_categories | number[] | 兴趣分类,如果传空数组 [] 表示不限，如果只传[0]表示系统推荐,如果按兴趣类型传表示自定义，通过“查询工具”-兴趣类目/兴趣关键词查询。当interest_action_mode传CUSTOM时有效 |
| interest_words | number[] | 兴趣关键词, 传入具体的词id，非兴趣词包id，可以通过词包相关接口或者兴趣关键词word2id接口获取词id，一个计划下最多创建1000个关键词。当interest_action_mode传CUSTOM时有效 |
| district | string | 地域类型 允许值: BUSINESS_DISTRICT 商圈、REGION 行政区域、OVERSEA 海外区域、NONE 不限 使用行政区域示例：{"district":"REGION": "city":[31], "region_version":"1.0.0"} 使用海外区域示例：{"district":"OVERSEA": "city":[3041566], "region_version":"1.0.0"} 当营销类型=SEARCH时不支持district=OVERSEA 海外区域 |
| region_version条件必填 | string | 行政区域版本号。通过[【获取全球国家信息】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1709606374050823)接口获取 district =REGION/OVERSEA时必填 |
| city条件必填 | number[] | 地域定向省市或者区县列表(当传递省份ID时,旗下市县ID可省略不传)，当district=REGION/OVERSEA时必填，通过[【获取行政信息】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1709606596424718)接口获取 |
| location_type | string | 受众位置类型,当city和district有值时，该字段必填，详见【[附录-受众位置类型](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)】 |
| superior_popularity_type | string | 媒体定向，详见[【附录-媒体定向】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E5%AA%92%E4%BD%93%E5%AE%9A%E5%90%91)。 对于选择自定义媒体定向流量包，该字段不传，传flow_package或exclude_flow_package字段即可 媒体定向仅支持穿山甲、穿山甲-精选游戏投放版位 |
| flow_package | number[] | 定向逻辑，可通过[【工具-穿山甲流量包-获取穿山甲流量包】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710720565248)定向逻辑和排除定向逻辑只能选其一 |
| exclude_flow_package | number[] | 排除定向逻辑，可通过[【工具-穿山甲流量包-获取穿山甲流量包】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710720565248)定向逻辑和排除定向逻辑只能选其一 |
| hide_if_converted | string | 过滤已转化用户。详见[【附录-过滤已转化用户类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E8%BF%87%E6%BB%A4%E5%B7%B2%E8%BD%AC%E5%8C%96%E7%94%A8%E6%88%B7%E7%B1%BB%E5%9E%8B)默认值：AD 营销计划 注意点：过滤已转化不支持投放目标为展示、点击计划，不支持受众预估 过滤已转化用户：选择过滤已转化用户之后，可以避免该营销再次投放给已转化过的用户，可以选择过滤5个单元层级的已转化用户（不限/营销计划/计划组/投放账户/公司账户/APP/全渠道应用） 仅当定向包投放类型为应用下载安卓、应用下载IOS、小程序时，可选择【全渠道应用】枚举 对于IOS14.5专属营销，只支持NO_EXCLUDE默认值：AD 营销计划 |
| converted_time_duration | string | 过滤时间范围，详见 [【附录-过滤时间范围】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535#custom-anchor-%E8%BF%87%E6%BB%A4%E6%97%B6%E9%97%B4%E8%8C%83%E5%9B%B4) 当hide_if_converted = APP 或 CUSTOMER 或 ORGANIZATION 或GLOBAL_APP时有效，默认为THREE_MONTH 仅投放穿山甲版位时不支持设置 |
| filter_event | string[] | 自定义过滤事件，hide_if_converted = NO_EXCLUDE时不支持传入 系统默认选择项目的优化目标作为要过滤的已发生转化的人群，支持选择其他要过滤的转化事件，允许值： 优化目标：AD_CONVERT_EXTERNAL_ACTION (新增） 激活：AD_CONVERT_TYPE_ACTIVE 注册：AD_CONVERT_TYPE_ACTIVE_REGISTER 付费：AD_CONVERT_TYPE_PAY |
| device_type | string[] | 设备类型。 允许值是："MOBILE", "PAD"。缺省表示不限设备类型。穿山甲已经全量，投放范围为默认时需要有白名单权限才可以 当营销类型=SEARCH时不支持该字段 |
| geolocation | object[] | 地图位置，district为BUSINESS_DISTRICT才有效 |
| name | string | 地点名字 |
| long | float | 经度 |
| lat | float | 纬度 |
| radius | number | 半径范围 |
| aweme_fans_numbers | number[] | （抖音投放特有）账号粉丝相似人群（添加抖音账号，会将营销投放给对应账号的相似人群粉丝） |
| filter_aweme_abnormal_active | number | 过滤高活跃用户 允许值：0:表示不过滤，1:表示过滤 设置场景： marketing_goal = LIVE landing_type = LIVE landing_type = EXTERNAL && marketing_goal = VIDEO_AND_IMAGE && delivery_range = DEFAULT |
| filter_aweme_fans_count | number | 过滤高关注数用户，例如"filter_aweme_fans_count": 1000表示过滤粉丝数在1000以上的用户 设置场景： marketing_goal = LIVE landing_type = LIVE landing_type = EXTERNAL && marketing_goal = VIDEO_AND_IMAGE && delivery_range = DEFAULT |
| filter_own_aweme_fans | number | 过滤自己的粉丝，允许值：0表示不过滤，1表示过滤 设置场景： marketing_goal = LIVE landing_type = LIVE landing_type = EXTERNAL && marketing_goal = VIDEO_AND_IMAGE && delivery_range = DEFAULT |
| aweme_fan_accounts | number[] | 抖音号id，与aweme_fan_behaviors同时设置才会生效（抖音达人定向） 设置场景： UBMAX项目不支持设置该字段 ad_type=SEARCH时不支持该字段 |
| aweme_fan_categories | number[] | 抖音类目id，与aweme_fan_behaviors同时设置才会生效（抖音达人定向） 设置场景： UBMAX项目不支持设置该字段 ad_type=SEARCH时不支持该字段 |
| aweme_fan_behaviors | string[] | 抖音用户行为类型, 详见[【附录-抖音用户行为类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)允许值:"FOLLOWED_USER","COMMENTED_USER","LIKED_USER","SHARED_USER"（抖音达人定向） 设置场景： UBMAX项目不支持设置该字段 ad_type=SEARCH时不支持该字段 |
| aweme_fan_time_scope | string | 抖音达人互动行为时间范围，允许值：FIFTEEN_DAYS 15天、THIRTY_DAYS 30天、SIXTY_DAYS 60天，默认时间范围为15天 设置场景： UBMAX项目不支持设置该字段 ad_type=SEARCH时不支持该字段 |
| hide_if_exists | number | 已安装用户，0表示不限，1表示过滤，2表示定向； 过滤表示投放时不给安装客户展示营销，支持应用； 定向表示投放时给安装客户展示营销； 投放时优先获取直达链接，无直达链接时使用应用包名进行投放； 如果无直达链接或应用包名，定向安装选项实际不生效；定向仅对Android链接生效。 默认值:0允许值: 0, 1,2 置场景： 当landing_type=LINK、MICRO_GAME时只支持传UNLIMITED 当delivery_mode = PROCEDURAL &&landing_type=APP 时仅支持「不限」和「过滤」 当landing_type =APP&&app_promotion_type = LAUNCH&& delivery_mode = PROCEDURAL&&marketing_goal = VIDEO_AND_IMAGE&&ad_type = ALL时支持「不限」、「过滤」、「定向」 当delivery_mode =PROCEDURAL&&landing_type=MICRO_GAME 时不支持传入 当landing_type = MICRO_GAME && micro_promotion_type = AWEME时，过滤已安装仅支持设置为UNLIMITED（不限） |
