# 营销素材预审提交接口（连山云视频点播版）

支持连山云视频点播的视频进行预审

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/open_material_audit/pro/submit/](https://api.oceanengine.com/open_api/v3.0/open_material_audit/pro/submit/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1832825155593417)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

file_id

string

连山云视频点播视频片源转码产物ID

当type=FILEID时有效且必传

business_type

string
