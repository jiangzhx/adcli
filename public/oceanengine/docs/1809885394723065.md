# 续费DOU+订单接口

用以追加DOU+订单预算与投放时长。

注意：不可以仅增加投放时长，可以仅增加投放预算

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/douplus/order/renew/](https://api.oceanengine.com/open_api/v3.0/douplus/order/renew/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1809885394723065)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

# 请求参数

字段

类型

描述

aweme_sec_uid

string

抖音号ID，通过已授权账户接口可以获取

task_id

number

renewal_budget

number

注意：数值需为10,000-500,000,000之间的10的倍数，

单位：人民币（分）

renewal_delivery_hour

double

延长的投放时长
