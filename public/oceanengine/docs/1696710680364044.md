# 获取绑定的抖音号

此接口可以帮助查询投放账户当前绑定的抖音号信息

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/ies_account_search/](https://ad.oceanengine.com/open_api/2/tools/ies_account_search/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710680364044)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| advertiser_id | number | 客户id |
| advertiser_name | string | 客户名 |
| aweme_id | string | 抖音id |
| aweme_name | string | 抖音昵称 |
| aweme_avatar | url | 抖音头像 |
| bind_type | string | 绑定类型，“1”：竞价，“2”：品牌 |
| status | string | 绑定状态，详见[【附录-枚举值-抖音号绑定状态】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535) |
| start_time | string | 绑定起始日期，格式：YYYY-MM-DD |
| commerce_user_level | string | 企业主页等级 |
| shared_advertisers | list[object] | 共享的客户列表 |
| shared_advertiser_id | number | 共享的客户id |
| shared_advertiser_name | string | 共享的客户名称 |
| request_id | string | 请求日志id |
