# 请求地址

[https://api.oceanengine.com/open_api/v3.0/shared_wallet/transaction_detail/get/](https://api.oceanengine.com/open_api/v3.0/shared_wallet/transaction_detail/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1798465768014922)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

GET

# Header

字段

类型

描述

string

# 请求参数

字段

类型

描述

account_id

number

账户ID，对于巨量AD、本地推是客户ID，对于代理商是代理商ID

account_type

string

账户类型 可选值:

shared_wallet_id

number

共享钱包ID

start_date

string

开始时间，格式YYYY-MM-DD

end_date

string

结束时间，格式YYYY-MM-DD

page

number

页码；默认为1

page_size

number

页面大小；默认为10; 注意：page*page_size不可大于10000
