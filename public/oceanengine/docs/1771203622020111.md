# 获取微信小程序列表

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/wechat_applet/list/](https://api.oceanengine.com/open_api/v3.0/tools/wechat_applet/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1771203622020111)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

filtering

object

过滤条件

name

string

小程序名称或备注的模糊匹配

audit_status

string

审核状态 允许值: AUDIT_ACCEPTED 审核通过、AUDITING 审核中、AUDIT_REJECTED 审核不通过。不传表示全部状态

search_type

string

搜索类型

允许值： CREATE_ONLY 只查询该账户创建的应用（默认值）、SHARE_ONLY 只查询被共享的应用

create_time

object

按创建时间查询的时间范围

start_time

string

创建起始时间，格式：%Y-%m-%d

end_time

string

创建结束时间，格式：%Y-%m-%d

update_time

object

start_time

string

更新开始时间，格式：%Y-%m-%d

end_time

string

更新结束时间，格式：%Y-%m-%d

user_name

string

page

number

页码，默认值为1

page_size

number

页面大小，默认值为10，最大值100

order_type

string

order_field

string
