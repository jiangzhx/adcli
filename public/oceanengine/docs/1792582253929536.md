# 【代理商】上传自产首发素材至方舟（搬运治理）

代理商可以通过此接口将「首发素材」上传视频素材至巨量方舟，上传后「首发素材」即可自动完成保护（前置需要先完成整体保护授权，参考详细文档介绍[「搬运治理-首发保护」说明手册（可对外）](https://bytedance.larkoffice.com/docx/Fpn4dn4ldoAHrmxrCEfcpWO1n8f)）。保护后系统将根据代理授权范围识别搬运素材生效打压，避免其他方抢夺代理的流量。

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/agent/](https://api.oceanengine.com/open_api/2/file/video/agent/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1792582253929536)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

字段

类型

描述

agent_id

number

代理商id

upload_type

string

视频上传方式，暂仅支持文件上传，可选值:

video_file

file

视频文件数据，upload_type为UPLOAD_BY_FILE必填

video_url

string

只支持连山tos地址，接入流程详见：[https://ad.oceanengine.com/cloud/doc/1756427622851659](https://ad.oceanengine.com/cloud/doc/1756427622851659)

video_signature

string

视频的md5值(用于服务端校验)

upload_type为UPLOAD_BY_FILE文件上传时必填

file_name

string

视频上传名称

is_aigc

bool

是否为AIGC生成，允许值：

is_need_auth

bool

标注是否允许生效代理商公司的首发保护授权范围

（允许授权的前提是素材是代理商自产且与客户达成了一致）
