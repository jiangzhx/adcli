# 添加屏蔽用户

添加屏蔽用户

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/aweme_banned/create/](https://api.oceanengine.com/open_api/v3.0/tools/aweme_banned/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1754804387645448)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户id

banned_type

string

屏蔽类型，允许值：

CUSTOM_TYPE 自定义规则，根据昵称关键词屏蔽

AWEME_TYPE 根据抖音ID屏蔽

aweme_id

string

需添加屏蔽用户的抖音号，当is_apply_to_adv不传或为false时，aweme_id生效

is_apply_to_adv

bool

是否应用于当前账户，当is_apply_to_adv不传或为false时，aweme_id生效

aweme_user_ids

string[]

抖音用户id列表，传入屏蔽用户的抖音号，当banned_type = AWEME_TYPE时必填

每次最多传50个抖音id，纯数字id不能以0开头

一个抖音号最多屏蔽5000个抖音ID

nickname_keywords

string[]

抖音昵称关键词列表，当banned_type = CUSTOM_TYPE时必填

关键词长度不大于20个字

每次最多传50个关键词
