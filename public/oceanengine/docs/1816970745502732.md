# 代理商创建前测任务

通过此接口，用户可以通过传入视频id和投放设置(setting)创建对应的前测任务。

使用限制：

# 请求地址

[https://api.oceanengine.com/open_api/2/diagnosis_task/agent/create/](https://api.oceanengine.com/open_api/2/diagnosis_task/agent/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1816970745502732)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

string

string

bool

target_cost

bool

nobid

bool

double

double

ref_ad_id

number

参考营销id，可复用该营销id的setting进行前测，非空情况下会忽略diagnose_config。 和ref_promotion_id二选一

ref_promotion_id

number
