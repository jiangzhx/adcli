# 批量更新项目状态

批量更新 巨量引擎体验版 营销项目的状态

不允许修改搜索周期稳投项目的操作状态（项目层级delivery_type = DURATION)

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/project/status/update/](https://api.oceanengine.com/open_api/v3.0/project/status/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1740941413906432)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

POST Request-Body

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type | string | 请求消息类型，允许值：application/json |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

## 基础参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| data | object[] | 批量更新项目状态，包含项目ID和目标操作，list长度限制1～10 |
| project_id | number | 项目ID |
| opt_status | string | 目标操作，允许值： ENABLE 启用项目、 DISABLE 暂停项目 对于删除的营销项目不可进行任何操作，否则会报错 |
