# 创建单元

本接口支持在巨量单元项目下创建单元，需要注意

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/create/](https://api.oceanengine.com/open_api/v3.0/promotion/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1740946299496459)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

POST Request-Body

# 请求Header。

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type | string | 请求消息类型，允许的值：application/json |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

## 基础参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| project_id 必填 | number | 项目ID |
| name 必填 | string | 单元名称，长度是1-50个字（两个英文字符占1个字）。名称不可重复，否则会报错 |
| operation | string | 单元状态， 允许值: ENABLE开启(默认值）、DISABLE关闭 |
