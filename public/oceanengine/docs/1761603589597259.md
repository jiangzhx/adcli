# 开启/更新一键起量

一键起量使用条件：oCPM计划、非nobid&自动化计划（旧版巨量引擎工作台，省心投），仅状态为“投放中”的营销支持“立即生效”

每个起量方案生效时间为6小时，冲突会报错

全量更新。传空则更新为空，此时已预约的方案将被删除，生效中的方案不受影响

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/promotion_raise/set/](https://api.oceanengine.com/open_api/v3.0/tools/promotion_raise/set/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1761603589597259)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| promotion_id 必填 | number | 营销ID |
| raise_info 必填 | object[] | 起量信息，如果需要全部删除请传[] |
| raise_budget 必填 | number | 起量预算，单位：元，允许小数点后两位起量预算需大于等于计划出价，小于等于计划预算 |
| is_effective_now 必填 | bool | 是否立即生效，仅支持营销状态为“投放中”的营销，仅支持1个方案设置“立即生效”，传入True时不支持填写appointed_time |
| appointed_time | object | 指定投放时间，当is_effective_now为FALSE时填写有效 |
| repeated_day | string[] | 重复周期，仅生效一次。不传则不重复，传入EVERY_DAY则每天重复 允许值： PER_MONDAY；PER_TUESDAY；PER_WEDNESDAY；PER_THURSDAY；PER_FRIDAY：PER_SATURDAY；PER_SUNDAY；EVERY_DAY EVERY_DAY和其他允许值不可同时传入 |
| raise_time | string | 起量时间 重复周期不传时，格式为yyyy-mm-dd HH:MM 传重复周期时，格式为HH:MM |
