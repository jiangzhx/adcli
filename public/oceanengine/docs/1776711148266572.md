# 获取营销产品资质

用于查询客户以营销产品形式提交的投放资质，可以获取到资质审核状态等信息

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_pkg/get/](https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_pkg/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1776711148266572)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

pkg_id

number

营销产品组id，是营销产品的组标识
