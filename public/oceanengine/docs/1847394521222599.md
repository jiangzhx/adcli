# AD业务工具

| 适用场景 | 作为个人业务人员/开发者，通过工具获取账户关系，业务报表数据做数据分析或二次加工处理等业务场景 |
| --- | --- |
| 支持协议类型 | Streamable HTTP |

## 配置示例

### 方式一：官方应用授权

可在【[MCP鉴权](https://open.oceanengine.com/mcp?app_id=1841425772187712)】页面登录对应营销业务账户并授权获取url

```
xxxxxxxxxx
{
"mcpServers": {
"巨量引擎工具": {
"url":·"https://open.oceanengine.com/ad/mcp?API_KEY={API_KEY}"
}
}
}
```

【警惕信息保护】url中的{API_KEY}为授权后生成的密码，密码如钥匙，绝不外借！请勿向任何人透露

### 方式二：开发者授权

已注册成为开发者，并且有申请过用户授权，可直接使用API接口的AccessToken

```
xxxxxxxxxx
{
"mcpServers": {
"巨量引擎工具": {
"url": "https://open.oceanengine.com/ad/mcp",
"headers": {
"Access-Token": "{Access-Token}",
"Content-Type": "application/json"
}
}
}
}
```
