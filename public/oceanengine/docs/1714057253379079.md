# 查询安卓应用母包

查询安卓应用母包信息

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/app_management/android_basic_package/get/](https://api.oceanengine.com/open_api/2/tools/app_management/android_basic_package/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1714057253379079)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

account_id

number

账户id，指可以接的账号体系如客户id、旧版巨量引擎工作台组织id等

account_type

string

账户类型，允许值：

AD 客户类型

BP 巨量引擎工作台账号类型

package_id

string

修改的安卓母包id
