# 上传券码

通过此接口，你可以上传优惠券的券码。调用此接口前，需要先创建卡券，获取优惠券id，优惠券的券码类型需要为商家上传券码类型。

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/coupon/code/upload/](https://ad.oceanengine.com/open_api/2/clue/coupon/code/upload/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710638086159)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: "multipart/form-data" |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| file 必填 | file | 券码文件 建议txt格式，券码数量不超过100万，否则可能解析失败。 每行一个券码，券码格式4-30位，字母+数字 |
| mark 必填 | string | 本次上传的标识 任意字符串，建议每次不一样，会在response中透传回来 |
| resource_id 必填 | number | 优惠券ID，调用卡券详情接口返回 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| count | number | 本次上传的有效券码总数 卡券未开始前，支持重复上传和补量 卡券开始后，不支持补量 |
| key | string | 生成文件的唯一标识（保留字段，暂时没有地方使用） |
| lines | number | 文件解析得到的行数 |
| mark | string | 本次上传的标识 |
| url | string | 本次上传有效的券码文件地址 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
