# 术语解释

本章节将帮助您快速了解API相关术语

## APPID

说明：以上账号维度不区分类型，例如代理商账号授权后记为绑定1个，但是可以操作代理商下的所有客户

## callback address

注意：需要确保回调地址能够成功接收回调信息。授权URL的形式如： [https://ad.oceanengine.com/marketing_api/auth?app_id=XXX&state=xxx&scope=xxx&redirect_uri=](https://ad.oceanengine.com/marketing_api/auth?app_id=XXX&state=xxx&scope=xxx&redirect_uri=) Callback address

## scope

注意：权限范围只能在自己的应用拥有的权限范围之内，否则会报错。格式例如：scope=[1,2,3,41]。scope权限范围取值关系，参见[Scope权限说明](https://open.oceanengine.com/labels/7/docs/1699352157034496)
