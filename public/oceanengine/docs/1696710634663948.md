# 获取表单列表

本接口用于获取青鸟线索通表单列表，可根据时间和表单实例id等条件进行过滤，本接口仅获取表单等基本信息，需要详细信息需调用【获取表单详情】接口。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/form/list/](https://ad.oceanengine.com/open_api/2/clue/form/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710634663948)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| instance_ids | number[] | 表单实例ID，可从【表单创建】【表单列表获取】接口获取 若指定instance_ids进行过滤，其他的过滤条件将不生效 |
| start_time | string | 起始时间，格式：%Y-%m-%d，默认7天前，即不指定起止时间获取最近7天数据 |
| end_time | string | 截止时间，格式：%Y-%m-%d，默认当天，即不指定起止时间获取最近7天数据 |
| is_del | number | 删除标记，默认获取未删除的数据 可选值：0，1（0表示未删除，1表示已删除） |
| page | number | 起始页，默认值为1 |
| page_size | number | 页大小，默认值为10，最大不超过50 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| page_info | json | 分页信息 |
| page | number | 页数 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| list | object[] | 表单列表 |
| name | string | 表单名称 |
| form_type | string | 表单类型，目前不支持设置，仅支持查询时返回 可选值：NORMAL_FORM（普通落地页）、ADVANCED_CREATIVE_FORM（附加创意）、NATIVE_FORM（原生表单） |
| validate_type | string | 线索验证类型，主要指C端是否进行短信验证，表单中含有电话类型时生效 可选值：CLUE_PRIORITY（线索量优先）、VALIDITY_PRIORITY（有效性优先）、NONE_VERIFICATION（全不出）、AUTO_VERIFICATION（智能验证，历史逻辑，效果等同于全不出）、ALL_VERIFICATION（全出），详情见[【附录-青鸟表单线索验证类型】](https://openapi.bytedance.net/openapi/doc/index.html?id=528) |
| instance_id | number | 表单实例ID |
| enable_layer | number | 是否开启分层 可选值：0，1（0表示不启用，1表示启用） |
| is_del | number | 删除标记 可选值：0，1（0表示未删除，1表示已删除） |
| create_time | string | 表单创建时间，格式：%Y-%m-%d %H:%M:%S |
| update_time | string | 表单上一次修改时间，格式：%Y-%m-%d %H:%M:%S |
| request_id | string | 请求日志id |
