# 上传图文内的音频素材

通过此接口，用户可以上传和营销相关的音频图片，例如图文中的音频。

# 请求地址

[https://api.oceanengine.com/open_api/2/file/audio/ad/](https://api.oceanengine.com/open_api/2/file/audio/ad/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1780005237921792)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: multipart/form-data

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

upload_type

string

音频上传方式 ，允许值：

audio_signature

string

音频的md5值(用于服务端校验)

upload_type为UPLOAD_BY_FILE时必填

audio_file

file

音频文件，upload_type为UPLOAD_BY_FILE时必填

允许格式：m4a、mp3

文件大小：≤20M

audio_url

string

音频url地址，如[http://xxx.xxx](http://xxx.xxx/)，upload_type为UPLOAD_BY_URL必填
