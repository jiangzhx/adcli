# 修改字节小游戏归因激活时间窗

使用场景：当前接口仅支持修改字节小游戏归因激活时间窗，表示已激活用户最后一次打开小程序后，时隔多久可被重新判定为未激活用户，该周期针对字节自归因的营销转化目标生效，默认为30天，可通过该接口编辑修改。

请注意：

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/micro_game/convert_window/update/](https://api.oceanengine.com/open_api/v3.0/tools/micro_game/convert_window/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1824379551374339)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| account_id必填 | number | 账户ID，account_type对应的账户ID |
| account_type必填 | string | 账户类型，允许值： AD投放账户 本次字节小游戏归因激活时间窗传AD |
| asset_id必填 | string | 资产id 字节小游戏传instance_id的值（[对应创建接口出参的值](https://open.oceanengine.com/labels/34/docs/1780614284853257?origin=left_nav)） 本次字节小游戏传instance_id的值 |
| asset_type必填 | number | 资产类型，允许值： BYTE_GAME：字节小游戏 目前仅支持字节小游戏，固定传BYTE_GAME |
| convert_window必填 | number | 时间窗口设置，允许值： ONE ：1天 THREE :3天 SEVEN ：7天 FOURTEEN ：14天 THIRTY ：30天 SIXTY ：60天 NINETY ：90天 ONE_HUNDRED_EIGHTY：180天 PERMANENT ：永久 |

1. 当前小游戏今日可修改次数达到上限；

2.“请联系运营配置白名单”表示当前入参adv需要联系运营/销售同学开放该试点功能；

json返回值，请求成功时，返回的data为空是正常现象，并非未更新
