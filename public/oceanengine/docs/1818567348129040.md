# 查询订阅 Adv

对于SPI订阅任务，支持「App维度」/「订阅任务维度」管理推送账户列表

对于RDS订阅任务，支持「订阅任务维度」管理推送账户列表，暂不支持「App维度」

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/subscribe/accounts/list/](https://api.oceanengine.com/open_api/v3.0/subscribe/accounts/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1818567348129040)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
