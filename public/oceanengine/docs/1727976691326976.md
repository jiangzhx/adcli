# 设置账户下RTA策略生效范围

设置投放账户下某个RTA策略的生效范围，设置一个新的RTA策略时，默认为启用状态，需要继续调用修改RTA策略状态

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/rta/set_scope/](https://ad.oceanengine.com/open_api/2/tools/rta/set_scope/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1727976691326976)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST Request-Body

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 投放账户id |
| rta_id必填 | number | 预期设置的rta策略id |
| target_type | string | 生效维度 默认值:ADV投放账户 允许值:ADV投放账户，PROJECT项目（体验版） |
| target_ids | number[] | 生效列表，当target_type = PROJECT 有效当target_type = PROJECT，传入项目id |
