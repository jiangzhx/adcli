# 批量查询账户余额

批量查询账号余额，支持AD、星图、本地推账户

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/account/fund/get/](https://api.oceanengine.com/open_api/v3.0/account/fund/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1799006604968971)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

account_ids

number[]

需要查询的账户ids，支持客户、代理商、星图客户、星图MCN、星图服务商、本地推

注意：

account_type

string

账户业务类型，可选值:

grant_type_split

string

是否拆分赠款类型，允许值：
