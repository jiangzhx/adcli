# 获取同主体下客户视频素材

通过此接口，用户可以查询同主体下的客户视频信息。

接口权限：查看视频的信息，获取视频md5、宽高、预览地址等内容，合理利用还可以搭建自己的素材库进行素材的管理。

### 请求地址

[https://ad.oceanengine.com/open_api/2/file/video/ad/get/](https://ad.oceanengine.com/open_api/2/file/video/ad/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710603509772)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| video_ids 必填 | string[] | 视频ids，长度限制 1~60 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| list | object[] | 视频列表 |
| id | string | 视频ID |
| size | number | 视频大小 |
| width | number | 视频宽度 |
| height | number | 视频高度 |
| url | string | 视频地址 |
| duration | number | 视频时长 |
| signature | string | 视频md5值 |
| poster_url | string | 视频首帧截图 |
| bit_rate | number | 码率，单位bps |
| format | string | 视频格式 |
| material_id | number | 素材id，一个素材唯一对应一个素材id |
| request_id | string | 请求日志id |
