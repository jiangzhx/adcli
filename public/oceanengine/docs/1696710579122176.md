# 获取投放条件列表（通用版）

该接口主要查询汽车行业投放条件信息，根据商品id查询投放条件列表

投放汽车行业商品，需要在计划中传入商品信息（商品库ID+商品ID）+投放条件信息（投放条件ID）

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/assets/list/](https://ad.oceanengine.com/open_api/2/dpa/assets/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710579122176)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| platform_id 必填 | number | 商品库ID |
| product_ids 必填 | number[] | 商品ID列表 |
| filtering | object | 过滤条件 |
| status | number | 投放条件状态， 0代表暂停， 1代表启用 |
| page | number | 页码， 默认值: 1 |
| page_size | number | 页面数据量， 默认值: 10，最大 100 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息 |
| data | object | 返回数据 |
| page_info | object | 分页信息 |
| page | number | 当前页数 |
| page_size | number | 页面大小 |
| total_number | number | 总数量 |
| total_page | number | 总分页数 |
| list | object[] | 商品库商品列表 |
| advertiser_id | number | 客户id |
| asset_id | number | 投放条件id |
| platform_id | number | 商品库id |
| asset_type | number | 投放条件类型，AUTO为汽车 |
| product_id | number | 商品id |
| status | string | 投放条件状态，DISABLE代表暂停， ENABLE代表启用 |
| audit_status | string | 投放条件状态，详见[【附录-物件状态】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| source | number | 来源，MANUAL为用户操作，AUTO为系统生成 |
| asset_create_time | string | 投放条件创建时间，格式: yyyy-MM-DD |
| asset_modify_time | string | 投放条件最近一次修改时间，格式: yyyy-MM-DD |
| request_id | string | 请求的日志id，唯一标识一个请求 |
