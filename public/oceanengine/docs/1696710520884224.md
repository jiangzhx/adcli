# 获取旧版巨量引擎工作台下账户列表

获取当前巨量引擎工作台下的资产账户列表（客户，企业号），如果巨量引擎工作台下还有子级组织账号，会把cc_account_id下所有可操作的资产账户全量返回

Token支持不同角色用户授权生成，某些无全量权限的角色（ 巨量引擎工作台(协作者) ）仅能看到当前巨量引擎工作台账户id下有权限操作的账户列表。操作权限受授权账号影响，所以可能会出现相同的工作台帐号在不同access_token请求下获取的列表不同的情况。

### 请求地址

[https://ad.oceanengine.com/open_api/2/customer_center/advertiser/list/](https://ad.oceanengine.com/open_api/2/customer_center/advertiser/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710520884224)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| cc_account_id 必填 | number | 巨量引擎工作台（原纵横组织）id，通过[【获取已授权账户】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |
| account_source | string | 账户类型，允许值： AD 巨量营销投放账号（默认） ENTERPRISE企业号 LOCAL 本地推 |
| filtering | object | 过滤条件 |
| account_name | string | 根据账户名称过滤 |
| page | number | 页码 默认值: 1 |
| page_size | number | 页面大小 默认值: 10，最大值：100 |

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| list | object[] | 账户列表 |
| advertiser_id | number | 客户id（一个对象只会返回企业号和投放账号其中一种） |
| advertiser_name | string | 客户名称（一个对象只会返回企业号和投放账号其中一种） |
| advertiser_type | string | 客户类型 枚举值：DOU+ DOU+类投放账号、NORMAL普通投放帐号、LOCAL：本地推头投放账号 DOU+类投放账号不支持任何调用接口操作 |
| e_douyin_id | string | 企业号id（一个对象只会返回企业号和投放账号其中一种） |
| e_douyin_name | string | 企业号名称（一个对象只会返回企业号和投放账号其中一种） |
| page_info | json | 分页信息 |
| page | number | 页数 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求日志id |
