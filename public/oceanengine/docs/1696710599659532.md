# 上传资质图片

通过此接口，用户可以按照一定方式上传符合格式的客户投放资质、主体资质相关图片，例如营业执照等，接口会返回"code_0"和"message_OK"，代表上传成功

图片格式：jpg、jpeg、png、bmp、gif，大小2M内

此接口上传的是资质图片，如需上传素材图片请调用【[上传营销图片](https://open.oceanengine.com/labels/7/docs/1696710600176640)】接口！

### 请求地址

[https://ad.oceanengine.com/open_api/2/file/image/advertiser/](https://ad.oceanengine.com/open_api/2/file/image/advertiser/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710599659532)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：multipart/form-data |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| upload_type 必填 | string | 图片上传方式 默认值：UPLOAD_BY_FILE允许值：UPLOAD_BY_FILE、UPLOAD_BY_URL |
| image_signature 条件必填 | string | 图片的md5值(用于服务端校验)，upload_type为UPLOAD_BY_FILE必填 |
| image_file 条件必填 | file | 图片文件,upload_type为UPLOAD_BY_FILE必填 格式: jpg、jpeg、png、bmp、gif, 大小2M内 |
| image_url 条件必填 | string | 图片url地址，如http://xxx.xxx，upload_type为UPLOAD_BY_URL必填 |
| upload_to 必填 | string | 调用上传资质 默认值：CG允许值：CG：客户资质接口、AD：投放资质接口 |
