# 获取推荐使用的视频素材

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/recommend/video/list/](https://api.oceanengine.com/open_api/v3.0/recommend/video/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1806800041632778)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

aweme_id

string[]

抖音号id，暂仅支持传入1个抖音号

project_id

number

项目id，建议您先调用【[创建项目](https://open.oceanengine.com/labels/34/docs/1740868093375503)】/【[获取项目列表](https://open.oceanengine.com/labels/34/docs/1740937147595776)】接口，获取新创建的项目
