# 获取组织认证及授权主体列表

使用场景：获取升级版巨量引擎工作台认证及关联认证主体相关信息（主体id、名称、主体及资质认证状态）

# 请求地址

https://api.oceanengine.com/open_api/v3.0/tools/ebp/subject/list/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1855089467521035)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
