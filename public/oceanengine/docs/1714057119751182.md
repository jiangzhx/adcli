# 更新安卓应用母包

更新安卓应用母包对应的信息

注意： 单APP_ID接口调用限制为 3QPS

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/app_management/android_basic_package/update/](https://api.oceanengine.com/open_api/2/tools/app_management/android_basic_package/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1714057119751182)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

account_id

number

账户id指可以接的账号体系如客户id、巨量引擎工作台组织id等

account_type

string

账号类型 可选值:

package_id

string

应用包id

app_name

string

应用名称

permissions_description

string

隐私权限说明

update_description

string

版本更新说明

recommend

string

推荐语

app_description

string

应用介绍

payment_type

string

付费类型 可选值:

app_developer_name

string

应用开发者名称

auto_publish

bool

是否自动发布

theme_tag_id

string

应用题材标签id

应用-游戏类目下该参数必填

填写题材标签的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

files

object[]

文件内容 必须包含一个apk；可选：五个图片、一个视频

upload_id

number

文件上传id

file_type

string

文件类型 可选值:

file_tag

string

文件标示 可选值:

file_option

string

文件选项 可选值:

category_id

number

应用类型

填写末级分类id，分类及id详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

art_style

string

画风分类

应用-游戏类目下该参数必填

填写题材标签的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

network_environment

string[]

网络环境

应用-游戏类目下该参数必填,最多支持传入3个

填写网络环境的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

feature_tags

string[]

特色标签

应用-游戏类目下该参数必填,最多支持传入3个

填写网络环境的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

game_box_name

string

盒子内游戏名称,

自定义填写,仅题材标签=游戏盒子时需填写

game_ip_type

string[]

IP类型

应用-游戏类目下该参数可填写

可选值:

game_ip_tag

string

细分IP，自定义填写，举例：如三国、西游等

spokesperson

string

代言人名字，自定义填写

revenue_model

string

变现模式

可选值:

has_online_earning

bool

是否包含网赚内容

min_payment_tier_range

string

最低付费档位区间（元）

应用-游戏类目下该参数必填

仅对变现模式=付费、混合支持传入

可选值:

mid_payment_tier_range

string

中间付费档位区间（元）

应用-游戏类目下该参数必填

仅对变现模式=付费、混合支持传入

可选值:

max_payment_tier_range

string

最高付费档位区间（元）

应用-游戏类目下该参数必填

仅对变现模式=付费、混合支持传入

可选值:

max_recharge_tier

string

最高充值档位（元）

应用-软件-工具&软件类目下参数必填

仅对变现模式=付费、混合 支持传入

可选值:

min_recharge_tier

string

最低充值档位（元）

应用-软件-工具&软件类目下参数必填

仅对变现模式=付费、混合 支持传入

可选值:

recommended_recharge_tier

string

推荐充值档位（元）

可选值:

schedule_cards

string[]

是否含周卡/月卡/季卡等

应用-游戏类目下参数必填

仅对变现模式=付费、混合支持传入

可选值:

has_discount

bool

是否有折扣

应用-游戏类目下参数必填

discount_rate

number

折扣比例

应用-游戏类目下参数必填

填写实际折扣力度，仅支持整数形式，如游戏盒子付费9折可填写 90

题材标签=游戏盒子，且变现模式=付费、混合，是否有折扣=是 支持传入

membership_type

string

会员类型

应用-软件-工具&社交类目下参数必填

可选值:
