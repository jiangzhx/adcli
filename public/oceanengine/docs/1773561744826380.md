# 批量删除图文

通过此接口，用户可以对图文素材进行批量删除。当素材删除失败时，会展示在carousel_id列表，不在此列表内的素材表示删除成功。

# 请求地址

[https://api.oceanengine.com/open_api/2/carousel/delete/](https://api.oceanengine.com/open_api/2/carousel/delete/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773561744826380)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| carousel_ids 必填 | number[] | 图文 id |
