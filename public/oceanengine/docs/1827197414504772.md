# 代理商获取视频素材

代理商获取视频素材

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/agent/get/](https://api.oceanengine.com/open_api/2/file/video/agent/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1827197414504772)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求参数

字段

类型

描述

agent_id

number

代理商ID

filtering

object

视频过滤条件

width

number

视频宽度

height

number

视频高度

ratio

float[]

视频宽高比，示例: [1.7, 2.5] 输入1.7则搜索满足宽高比介于1.65-1.75之间的视频，即精度上下浮动0.05

video_ids

string[]

视频ids，示例: ["86adb23eaa21229fc04ef932b5089bb8"] 数量限制：<=100 注意：video_ids、material_ids、signatures只能选择一个进行过滤

material_ids

number[]

素材id列表，可以根据material_ids（素材报表使用的id，一个素材唯一对应一个素材id）进行过滤 数量限制：<=100 注意：video_ids、material_ids、signatures只能选择一个进行过滤

signatures

string[]

md5值列表，可以根据素材的md5进行过滤 数量限制：<=100 注意：video_ids、material_ids、signatures只能选择一个进行过滤

start_time

string

根据视频上传时间进行过滤的起始时间，与end_time搭配使用，格式：yyyy-mm-dd

end_time

string

根据视频上传时间进行过滤的截止时间，与start_time搭配使用，格式：yyyy-mm-dd

labels

string[]

视频标签

source

string[]

素材来源枚举，可选值：

AD_SITE : ad后台本地上传

CREATIVE_CENTER : 创意中心（巨量创意）

OPEN_API : 开放平台

SUPPLIER : 即合视频

VIDEO_CAPTURE : 易拍视频

ACCOUNT_PUSH : 推送视频

STAR : 星图视频

CEWEBRITY_VIDEO : 达人视频（抖音主页视频）

BP : 巨量纵横

E_COMMERCE : 巨量千川

DPA_MERSCHANT_CENTER : 行业产品中心

QF_FUWU : 群峰服务市场

AIC : 即创

page

number

页码，默认值1

page_size

number

页面大小，默认值20
