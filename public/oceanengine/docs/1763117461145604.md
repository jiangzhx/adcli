# 获取营销预览二维码（升级版）

新建通过后支持生成获取预览二维码，项目/营销暂停不支持获取预览二维码

注意：通过本接口获取到的预览url，需要自行转换为二维码，使用巨量引擎app或抖音app扫码预览

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/ad_preview/qrcode_get/](https://api.oceanengine.com/open_api/v3.0/tools/ad_preview/qrcode_get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1763117461145604)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

id_type

string

查询条件，允许值

ID_TYPE_PROMOTION 按营销ID查询

ID_TYPE_MATERIAL按素材ID查询

promotion_id

number

营销ID

material_id

number

素材ID，查询素材预览时使用

id_type = ID_TYPE_MATERIAL时 必填
