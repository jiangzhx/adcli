# 查询核销员

查询核销员接口

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/coupon/employee/get/](https://ad.oceanengine.com/open_api/2/clue/coupon/employee/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710641718284)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值10，不超过50 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] | 核销员列表 |
| user_id | number | 用户ID |
| user_type | string | 用户类型 TOUTIAO:头条用户 DOUYIN:抖音用户 |
| store_id | number | 门店ID，0代表不限制门店 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
