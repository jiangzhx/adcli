# 获取商品投放条件列表（线索版）

该接口主要查询汽车行业投放条件信息，根据新线索商品id查询投放条件列表

# 请求地址

[https://api.oceanengine.com/open_api/2/dpa/asset_v2/list/](https://api.oceanengine.com/open_api/2/dpa/asset_v2/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1788119750912003)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST
