# 转账-发起转账（代理）

发起转账，支持1:N转账、不停投转账、虚客互转

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/cg_transfer/create_transfer/](https://api.oceanengine.com/open_api/v3.0/cg_transfer/create_transfer/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1789755060558916)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

biz_request_no

string

请求幂等id，同一biz_request_no发起转账代表同一转账申请，返回的转账单号相同，推荐uuid

agent_id

number

代理商账户id，用于鉴权

account_id

number

锚定账户id，1:N的1

target_account_detail_list

object[]

目标账户列表，1:N的N，需要列表内账户类型相同，最多支持100个

account_id

number

目标账户id

transfer_capital_detail_list

object[]

锚定账户与目标账户转账资金列表

capital_type

string

发起转账时申请的资金类型

转账资金类型 可选值:

transfer_amount

number

转账资金金额(单位：分)

capital_sub_type

string

transfer_direction

string

转账方向，以目标账户视角确定 可选值:

remark

string

备注
