# 获取微信小游戏列表

支持获取纵横账户下及投放账户下的微信小游戏资产

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/wechat_game/list/](https://api.oceanengine.com/open_api/v3.0/tools/wechat_game/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1771362185154564)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

account_id

number

账户ID，accout_type类型对应账户ID

account_type

string

账户类型

允许值：旧版巨量引擎工作台 旧版巨量引擎工作台账号、AD 客户账号

filtering

object

过滤条件

name

string

小游戏名称或备注的模糊匹配

asset_status

string

资产状态 允许值：升级版UPGRADED、 原版ORIGINAL（默认值）

audit_status

string

审核状态 允许值: AUDIT_ACCEPTED 审核通过、AUDITING 审核中、AUDIT_REJECTED 审核不通过。不传表示筛选全部状态

search_type

string

搜索类型

允许值： CREATE_ONLY 只查询该账户创建的应用（默认值）、SHARE_ONLY 只查询被共享的应用

create_time

object

按创建时间查询的时间范围

start_time

string

创建起始时间，格式：%Y-%m-%d

end_time

string

创建结束时间，格式：%Y-%m-%d

page

number

页码，默认值为1

page_size

number

页面大小，默认值为10，最大值100
