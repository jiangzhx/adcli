# 查询授权直播抖音达人列表

授权直播抖音达人列表

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/live_authorize/list/](https://ad.oceanengine.com/open_api/2/tools/live_authorize/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1703442176442381)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| status必填 | string[] | 直播达人授权状态列表 ，允许值：APPLYING申请中、APPLY_OVERDUE申请过期、AUTHORIZING授权中、AUTHORIZE_OVERDUE授权过期 |
| page | number | 当前页码，默认值：1 |
| page_size | number | 页面大小，默认值：10，最大值：50 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| page_info | json | 分页信息 |
| page | number | 页数 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| list | json | 达人信息列表 |
| ies_id | string | 抖音号id |
| ies_user_name | string | 抖音昵称 |
| advertiser_id | number | 客户id |
| create_time | string | 发起时间，格式yyyy-mm-dd hh:MM:ss |
| authorize_starttime | string | 直播授权开始时间，格式yyyy-mm-dd hh:MM:ss |
| authorize_endtime | string | 直播授权结束时间，格式yyyy-mm-dd hh:MM:ss |
| limited_promotion_types | object[] | 限制抖音号使用场景信息 |
| msg | string | 场景信息 |
| request_id | string | 请求日志id |
