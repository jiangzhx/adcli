# 查询投放账户头像

##### 请求地址

[https://ad.oceanengine.com/open_api/2/advertiser/avatar/get/](https://ad.oceanengine.com/open_api/2/advertiser/avatar/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710512904192)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| advertiser_id | string | 客户id |
| avatar_status | string | 头像审核状态 UNSET-未设置 IN_AUDIT-审核中 AUDIT_REJECT-审核被拒 AUDIT_PASS-审核通过 |
| avatar_reason | string | 头像被拒原因 |
| source_status | string | 品牌审核状态 UNSET-未设置 IN_AUDIT-审核中 AUDIT_REJECT-审核被拒 AUDIT_PASS-审核通过 |
| source_info | string | 品牌信息 |
| source_reason | string | 品牌信息被拒原因 |
| avatar_info | json | 头像信息 |
| web_uri | string | 当前头像的uri |
| audit_web_uri | string | 审核中头像的uri |
| height | number | 审核中头像的高度 |
| width | number | 审核中头像的宽度 |
| web_url | string | 当前头像的预览链接，有效期1h |
| audit_web_url | string | 审核中头像的预览链接，有效期1h |
| request_id | string | 请求日志id |
