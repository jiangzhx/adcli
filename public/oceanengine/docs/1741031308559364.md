# 批量更新营销启用状态

批量更新 巨量引擎体验版 营销启用状态

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/status/update/](https://api.oceanengine.com/open_api/v3.0/promotion/status/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1741031308559364)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

POST Request-Body

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type | string | 请求消息类型，允许值：application/json |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

## 基础参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| data 必填 | object[] | 批量修改营销状态，list长度限制1～10 |
| promotion_id 必填 | number | 营销ID |
| opt_status 必填 | string | 操作，ENABLE 启用营销、DISABLE 暂停营销 |
