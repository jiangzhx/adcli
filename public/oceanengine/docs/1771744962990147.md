# 创建微信小游戏

微信小游戏创建后不支持更新

请您确保您已经上传了投放资质，且您上传投放资质的投放账户认证的公司主体，需与资产创建流程中选择的公司主体一致，以用于资质审核。若未提交投放资质，请通过【[投放资质提交接口](https://open.oceanengine.com/labels/7/docs/1761501324513352)】提交资质，否则资产审核将不通过。资质信息可以通过【[获取投放资质信息](https://open.oceanengine.com/labels/7/docs/1696710513397775)】接口查询。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/wechat_game/create/](https://api.oceanengine.com/open_api/v3.0/tools/wechat_game/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1771744962990147)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求参数

字段

类型

描述

account_id

number

账户id，accout_type类型对应账户ID

account_type

string

账户类型 允许值：旧版巨量引擎工作台 旧版巨量引擎工作台、 AD 客户账号

company_id

number

公司主体id，通过【[获取旧版巨量引擎工作台下所有主体信息](https://open.oceanengine.com/labels/7/docs/1741478619724813)】接口获取，应用资质审核环节将以该主体作为拥有应用投放相关权利的主体进行审核，account_type=旧版巨量引擎工作台时必填

user_name

string

小游戏原始ID，获取方式：小程序后台>设置>基本设置>账号信息>原始ID，例如“gh_”

name

string

小游戏名称，最大长度不超过50

path

string

小游戏路径参数，如有跳转指定页面或自定义参数监测营销效果数据诉求，需填写。

anchor_list

object

锚点组件信息

icon_image_url

string

小游戏icon图片的url地址，图片尺寸60*60px,大小不超2MB，格式要求jpeg、png或gif。

header_image_url

string

顶部头图的url地址，大小不超过5MB的图片，推荐尺寸2:1，格式要求jpeg、png或gif。

labels

string[]

小游戏标签，每个标签长度不超过6，最多支持6个标签

guide_text

string

引导文案，最大长度不超过14

images_vertical_url

string[]

images_horizontal_url

string[]

introduction

string

小游戏简介，最大长度不超过45

anti_addiction_url

string

防沉迷提示URL，请上传游戏内包含限制游戏时间或建议合理安排游戏相关内容的截图。如防沉迷提示、防沉迷公共、健康系统相关内容。常见形式（包括不限于）：进入青少年模式，青少年防沉迷公告弹窗/提示，健康系统提示，强制休息提醒等

图片格式要求jpeg、png或gif

screen_record_url

string

游戏内容视频URL，请上传3分钟的游戏内实际画面操作录屏视频，要求MP4格式文件，最大不超过100M。横版16:9，分辨率不低于960540。竖版9:16，分辨率不低于540960

图片格式要求jpeg、png或gif，视频格式要求mp4

real_name_url

string

实名认证URL，请上传游戏内包含实名注册功能的截图链接

图片格式要求jpeg、png或gif

age_remind_url

string

适龄提醒URL，请上传包含适龄提示图标的游戏界面截图，图标样式需与中国音像数字出版协会官方网站下载保持一致

图片格式要求jpeg、png或gif

tag_info

string

玩法分类

填写玩法分类的末级分类ID category_id，分类及ID详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)
