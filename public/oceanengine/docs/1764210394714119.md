# 获取清理任务列表

返回已创建的低效/同质视频素材清理任务列表

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/material/clear_task/get/](https://api.oceanengine.com/open_api/2/file/video/material/clear_task/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1764210394714119)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| clear_ids | number[] | 清理任务id列表，最多支持10个 |
| page | number | 页码，默认：1 |
| page_size | number | 页面大小，默认10，范围：1-100 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] |  |
| clear_id | number | 清理任务id |
| clear_task_params | object | 任务的参数 |
| material_ids | number[] | 待清理的素材列表 |
| clear_material_types | string[] | 清理素材类型： INEFFICIENT_MATERIAL低效素材； SIMILAR_MATERIAL 同质化挤压严重素材； SIMILAR_QUEUE_MATERIAL 同质化排队素材 |
| create_time_upper | string | 清理此日期之前创建的素材，格式为"yyyy-mm-dd"，不包含该日期 |
| start_time | string | 账户下累积转化数和累积消耗数的数据统计开始时间，格式为"yyyy-mm-dd"，包含该日期 |
| end_time | string | 账户下累积转化数和累积消耗数的数据统计结束时间，格式为"yyyy-mm-dd"，包含该日期 |
| material_source | string | 清理的素材来源: AD平台 QIANCHUAN千川 |
| convert | number | 账户下累积转化数，在start_time和end_time期间发生的累积转化数，清理小于该累积转化数的素材 当start_time和end_time传入时有效 |
| cost | float | 账户下累积消耗，在start_time和end_time期间发生的累积消耗，清理小于该累积消耗的素材 当start_time和end_time传入时有效 |
| create_time | string | 任务创建时间，格式：%Y-%m-%d %H:%M:%S |
| task_status | string | 任务状态: DONE 已完成 RUNNING 运行中 TIMEOUT超时 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
