# 获取投放账户头像ID

本接口用户获取投放账户头像的image_id，您可使用该id调用「[更新投放账户头像](https://open.oceanengine.com/labels/34/docs/1696710512435215)」接口完成账户头像更新

【注意】本接口的功能仅用于获取image_id，上传成功 ≠ 更新头像，更新头像的接口是「[更新投放账户头像](https://open.oceanengine.com/labels/34/docs/1696710512435215)」接口

# 请求地址

[https://api.oceanengine.com/open_api/2/advertiser/avatar/upload/](https://api.oceanengine.com/open_api/2/advertiser/avatar/upload/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1794039016601604)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
