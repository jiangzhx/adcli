# 上传试玩/直玩素材

可通过本接口创建试玩或直玩素材

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/file/playable/create/](https://api.oceanengine.com/open_api/v3.0/file/playable/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1836623015076167)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

Content-Type

string

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

app_material_id

number

试玩/直玩素材Material_ID

可通过抖音开放平台 - 游戏管理中获取，参考：[小游戏试玩指导手册](https://bytedance.larkoffice.com/docx/FlFCdsoh1o8J9mxkU3Fc3XzEnbd)

app_id

string

小游戏资产的app_id

通过[【获取字节小游戏】](https://open.oceanengine.com/labels/7/docs/1778100726681600?origin=left_nav)接口获取，支持上传直玩素材需要设置关联的小游戏资产ID

material_type

string

素材类型，可选值:
