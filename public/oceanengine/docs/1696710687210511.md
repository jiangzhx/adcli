# 查询抖音帐号和类目信息

查询绑定的抖音号信息

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/aweme_info_search/](https://ad.oceanengine.com/open_api/2/tools/aweme_info_search/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710687210511)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| query_word必填 | string | 搜索词 |
| behaviors | string[] | 抖音用户行为类型, 详见[【附录-抖音用户行为类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)允许值: "FOLLOWED_USER","COMMENTED_USER","LIKED_USER","SHARED_USER" 注：该字段为选填，需表达特定行为类型才可返回有效值 |
