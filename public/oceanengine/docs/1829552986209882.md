# 下载获取升级版巨量引擎工作台账户任务结果

下载获取升级版巨量引擎工作台账户任务结果，返回结果为文件类型，结构同「[获取升级版巨量引擎工作台下账户列表](https://open.oceanengine.com/labels/7/docs/1829550825614739)」

# 请求地址

[https://api.oceanengine.com/open_api/2/ebp/advertiser/task/download/](https://api.oceanengine.com/open_api/2/ebp/advertiser/task/download/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1829552986209882)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
