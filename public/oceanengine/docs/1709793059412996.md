# 获取可创建事件列表

此接口用于查询资产下支持创建的事件列表

### 请求地址

[https://ad.oceanengine.com/open_api/2/event_manager/available_events/get/](https://ad.oceanengine.com/open_api/2/event_manager/available_events/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1709793059412996)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| asset_id必填 | number | 资产ID |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| event_configs | object[] | 可创建事件列表 |
| event_id | number | 事件ID |
| event_type | string | 事件类型 |
| event_cn_name | string | 事件中文名称 |
| description | string | 事件描述信息 |
| track_types | string[] | 事件回传方式列表，允许值:落地页支持:JSSDK JS埋码 、EXTERNAL_API API回传、XPATH XPath圈选 应用支持：APPLICATION_API 应用API、APPLICATION_SDK 应用SDK、快应用支持：QUICK_APP_API 快应用API、小程序支持：MINI_PROGRAME_SDK小程序SDK、MINI_PROGRAME_API 小程序API |
| properties | object[] | 事件的附加属性 |
| field | string | 附加属性英文名称 |
| field_name | string | 附加属性中文名称 |
| variable_type | string | 附加属性值类型 |
| enum_value | dict[string] | 附加属性枚举值 |
| unit | string | 附加属性单位 |
| description | string | 附加属性描述 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
