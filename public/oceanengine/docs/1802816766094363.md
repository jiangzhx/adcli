# 创建/编辑升级版商品

新增、编辑行业产品中心【产品管理-升级版】商品。截止至更新日期 2025/06/11 支持类目见下表：（具体创建商品时请参考商品类目枚举）

| 支持类目 | 一级类目id（category_id） | 支持方式 |
| --- | --- | --- |
| 电商店铺商品 | 140000000 | 部分加白客户可用，如有创建诉求需先找相应销售同学咨询 |
| 活动商品 | 250000000 | 部分加白客户可用，如有创建诉求需先找相应销售同学咨询 |
| 房产 | 130000000 | 房产行业客户默认可用，其他行业客户如发现无法创建可联系运营同学 |
| 汽车 | 60000000 | 功能灰度开放，如有对接诉求请联系相应销售/行运同学 |
| 游戏 | 260000000 | 游戏行业客户默认可用，其他行业客户如发现无法创建可联系运营同学 |
| 影视 | 290000000 | 部分加白影视类客户可用，如有创建诉求需先找相应销售同学咨询 |

若有查询升级版商品列表的需求，请参考：[https://open.oceanengine.com/labels/34/docs/1779430442685440](https://open.oceanengine.com/labels/34/docs/1779430442685440)

若有查询升级版商品详情的需求，请参考：[https://open.oceanengine.com/labels/34/docs/1779436000953415](https://open.oceanengine.com/labels/34/docs/1779436000953415)

# 请求参数

## 公共参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| product 必填 | object | 商品信息（不同行业类目商品的product传值参考下面对应行业说明） |
| brand_info条件必填新增字段 | object | 品牌信息 汽车品牌信息以下方汽车类目说明为准，不按照当前结构传值 游戏及影视类目不需要传品牌信息 |
| brand_id条件必填新增字段 | string | 品牌ID 当profession下brand_type_choose入参标准品牌时必填，需要通过[品牌查询列表接口](https://open.oceanengine.com/labels/7/docs/1855078665651657)获取标准品牌ID，入参ID不为品牌查询列表接口中的ID会被拒绝请求 当profession下brand_type_choose入参自定义品牌时非必填（即使入参了品牌ID也会忽略） |
| brand_name必填新增字段 | string | 品牌名称 当profession下brand_type_choose入参标准品牌时，需要通过[品牌查询列表接口](https://open.oceanengine.com/labels/7/docs/1855078665651657)获取标准品牌ID及对应的brand_full_name 当profession下brand_type_choose入参自定义品牌时，当前字段可自定义入参 |
| profession | object | 行业特定字段 |
| brand_type_choose条件必填新增字段 | string | 品牌类型选择，汽车、游戏及影视类目不需要传当前字段，可选枚举： 标准品牌：被品牌库收录的品牌，可通过品牌查询列表接口获取 自定义品牌：实际有品牌，但在「标准品牌」接口范围中无法搜索到，可入参此选项 无品牌：实际经营商品无品牌，入参当前枚举可不填写品牌名称（非实际情况不建议） |
