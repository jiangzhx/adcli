# 获取定向包

获取客户下定向包，不同类型定向包可应用于不同类型的计划

落地页：可用于营销目的为销售线索收集或营销目的为应用且下载方式为落地页的计划

应用（Android）：可用于营销目的为应用且下载方式为Android下载链接的计划

应用（iOS）：可用于营销目的为应用且下载方式为iOS下载链接的计划

其余类型：可应用于营销目的为该类型名称的计划
