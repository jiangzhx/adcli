# 获取项目列表

获取 巨量引擎体验版 项目列表

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/project/list/](https://api.oceanengine.com/open_api/v3.0/project/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1740937147595776)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET Request-Query

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 投放账户id |
| fields | string | 查询字段集合，如果指定则应答结果仅返回指定字段 可参考应答参数返回的指标字段 |
| filtering | object | 过滤条件，若此字段不传，或传空则视为无限制条件 |
| ids | number[] | 按项目ID过滤，范围为1-100 |
| delivery_mode | string | 投放模式，允许值：MANUAL手动投放、PROCEDURAL自动投放 |
| delivery_type | string | 按投放类型过滤，可选值： NORMAL 常规投放 DURATION周期稳投（搜索下为「极速智投」） UBX_INTELLIGENT智能投放 如需筛选搜索「极速智投」项目，必须同时传入ad_type = SEARCH筛选 如需筛选通投「周期稳投」项目，请同时传入landing_type筛选，否则筛选结果不准确。ad_type = ALL 通投场景下，仅2种营销目的支持周期稳投 landing_type = SHOP landing_type = LINK |
| landing_type | string | 营销目的，允许值： APP 应用 LINK 销售线索投放 MICRO_GAME小程序 SHOP电商 QUICK_APP快应用 NATIVE_ACTION 内容营销 DPA商品目录 |
| app_promotion_type | string | 子目标，允许值： DOWNLOAD 应用下载、LAUNCH 应用调用、RESERVE 预约下载 |
| marketing_goal | string | 营销场景，允许值：VIDEO_AND_IMAGE 短视频/图片、LIVE直播 |
| ad_type | string | 营销类型，允许值：ALL 所有营销，SEARCH 搜索 |
| name | string | 项目名称，长度是1-50个字 （两个英文字符占1个字，该字段采取模糊查询的方式。） |
| status | string | 项目状态过滤， 注意：此参数已不再维护，请您使用下方的status_first、status_second 允许值： PROJECT_STATUS_ENABLE 启用 PROJECT_STATUS_DISABLE 暂停 PROJECT_STATUS_DELETE 删除 PROJECT_STATUS_ALL 所有包含已删除 PROJECT_STATUS_NOT_DELETE 所有不包含已删除 PROJECT_STATUS_BUDGET_EXCEED 项目超出预算 PROJECT_STATUS_BUDGET_PRE_OFFLINE_BUDGET 项目接近预算 PROJECT_STATUS_NOT_START 未达投放时间 PROJECT_STATUS_DONE 已完成 PROJECT_STATUS_NO_SCHEDULE 不在投放时段 |
| status_first | string | 项目一级状态过滤，允许值： PROJECT_STATUS_DELETE已删除 PROJECT_STATUS_DONE已完成 PROJECT_STATUS_DISABLE未投放 PROJECT_STATUS_ENABLE启用中 不传默认返回不限（不含已删除） |
| status_second | string | 项目二级状态过滤，允许值： PROJECT_STATUS_STOP 已暂停 PROJECT_STATUS_BUDGET_EXCEED 项目超出预算 PROJECT_STATUS_NOT_START 未达投放时间 PROJECT_STATUS_NO_SCHEDULE 不在投放时段 PROJECT_STATUS_BUDGET_GROUP_OFFLINE_BUDGET 预算组超出预算（新增） 当status_first = PROJECT_STATUS_DISABLE时传入有效 PROJECT_STATUS_REACH_DAILY_DELIVERY 未投放-已达每日投放时长 |
| project_create_time | string | 项目创建时间，格式yyyy-mm-dd，表示过滤出当天创建的项目 |
| project_modify_time | string | 项目更新时间，格式yyyy-mm-dd，表示过滤出当天更新的项目 |
| project_modify_start_time | string | 项目更新开始时间，该筛选支持您细分到秒级查询一段时间内修改过的项目信息 格式yyyy-mm-dd hh:mm:ss 注意 必须与结束时间project_modify_end_time结合使用，注意时间跨度不能超过30天 开始时间<=结束时间 可以与project_modify_time同时使用，但不太有必要。直接使用project_modify_start_time、project_modify_end_time组合筛选即可 |
| project_modify_end_time | string | 项目更新结束时间，该筛选支持您细分到秒级查询一段时间内修改过的项目信息 格式yyyy-mm-dd hh:mm:ss 注意 必须与开始时间project_modify_start_time结合使用，注意时间跨度不能超过30天 开始时间<=结束时间 可以与project_modify_time同时使用，但不太有必要。直接使用project_modify_start_time、project_modify_end_time组合筛选即可 |
| pricing | string | 按计费方式过滤 |
| inventory_type | string | 按首选位置过滤，允许值 INVENTORY_FEED 今日头条 INVENTORY_VIDEO_FEED 西瓜视频 INVENTORY_AWEME_FEED 抖音短视频 INVENTORY_TOMATO_NOVEL 番茄小说 INVENTORY_UNION_SLOT 穿山甲 UNION_BOUTIQUE_GAME ohayoo精品游戏 INVENTORY_HOMED_AGGREGATE 住小帮 INVENTORY_SEARCH 搜索投放版位 |
| platform | string | 按平台过滤，允许值：IOS、ANDROID、HARMONY |
| budget_group_id | number | 按预算组ID过滤，仅允许传入1个，该功能白名单开放，如需使用请联系销售 您需要首先通过「[创建预算组](https://open.oceanengine.com/labels/34/docs/1780159359081540)」/「[获取预算组列表](https://open.oceanengine.com/labels/34/docs/1780644067911818)」接口拿到此id |
| blue_flow_package_setting | string | 是否开启蓝海流量投放过滤，允许值：ON、OFF |
| star_delivery_type | string | 是否为星广联投项目 STAR_DELIVERY 星广联投项目 NOT_STAR_DELIVERY 非星广联投项目 |
| star_auto_delivery_switch | string | 星广联投全自动化开关，枚举值： ON 开启 OFF 关闭 注意：筛选全自动化项目时，筛选条件不需要设置star_delivery_type参数 |
| page | number | 页数默认值：1，page范围为[1,99999] |
| page_size | number | 页面大小默认值：10，page_size范围为[1,100] |
