# 开启素材起量

开启素材起量

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/material_raise/create/](https://api.oceanengine.com/open_api/v3.0/tools/material_raise/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1841780308383884)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

POST

# Header

字段

类型

描述

# 请求参数

字段

类型

描述

advertiser_id

number

project_id

number

material_ids

number[]

素材ID，不传默认为项目下全量可用素材

raise_info

object

raise_effective_type

string

raise_budget

float

起量预算，单位：元，允许小数点后两位 起量预算上下限制校验[100,9999999.99]

appoint_time

object

指定投放时间

end_time

string

结束时间 格式：yyyy-mm-dd hh:mm
