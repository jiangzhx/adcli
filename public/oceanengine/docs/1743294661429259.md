# 上传主体资质（新版）

提交客户的主体资质信息为全量接口。更新的时候需要全量先获取所有的主体资质，然后更新相应主体资质。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/qualification/submit/](https://api.oceanengine.com/open_api/v3.0/advertiser/qualification/submit/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1743294661429259)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST
