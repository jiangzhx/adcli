# 设置商品库&商品共享

能力介绍：通用接口，支持通用版商品库、升级版商品及小程序小游戏资产共享。共享模式支持旧版巨量引擎工作台下所有账户、及单账户共享，商品库和商品目前仅支持共享给ad投放账户。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/](https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1834652218001420)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

object[]
