# 获取客户公开信息

获取投放账户基础信息，无需申请权限

# 请求地址

[https://ad.oceanengine.com/open_api/2/advertiser/public_info/](https://ad.oceanengine.com/open_api/2/advertiser/public_info/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710511950860)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_ids 必填 | number[] | 客户ID集合(如果包含没有访问权限的ID,将返回PERMISSION_ERROR).取值范围: 1-100 |

public static JSONObject getAdvertiserPublicInfo() {

​
CloseableHttpResponse response = null;

​

if (response != null && response.getStatusLine().getStatusCode() == 200) {

bufferedReader.close();

# 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/12/docs/1699633774328846) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/12/docs/1699633774328846) |
| data | object[] | 客户列表 |
| id | number | 账户ID |
| name | string | 账户名 |
| company | string | 公司名 |
| first_industry_name | string | 一级行业名 |
| second_industry_name | string | 二级行业名 |
| request_id | string | 请求日志ID |
