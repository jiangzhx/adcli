# 获取DPA私有模板

获取dpa私有模板（需要客户在巨量创意平台上传自行设计的PSD格式的模板文件，巨量创意平台生产对应的私有模板id，再关联到即将投放的客户id）

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/template/get/](https://ad.oceanengine.com/open_api/2/dpa/template/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710581781519)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| page | string | 页码。默认值: 1 |
| page_size | string | 页面数据量。默认值: 10 |

### 应答参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| list | list | json返回值 |
| template_id | int | 模板id |
| template_name | string | 模板名称 |
| template_mode | string | 模板类型，对应创意素材类型中的大图小图组图 |
| industry | int | 行业 |
| template_data_list | list | 模板数据列表 |
| web_url | string | 背景图 |
| template_image_url | string | 预览图 |
| page_info | object | 分页信息 |
| page | number | 当前页 |
| page_size | number | 页大小 |
| total_page | number | 总页数 |
| total_number | number | 总条数据 |
| request_id | string | 请求日志id |
