# 项目管理模块

## 投放位置

| 投放版位置 | 描述 |
| --- | --- |
| 首选场景 | 营销将主要投放在首选场景上，同时为了帮助营销计划更好的跑量和优化成本，系统有可能将营销智能拓展投放到其他流量场景 沉浸式竖版视频场景、信息流场景、视频后贴和尾帧场景 |
| 首选媒体 | 营销将主要投放在首选媒体上，同时为了帮助营销计划更好的跑量和优化成本，系统有可能将营销智能拓展投放到其他流量场景 |
| 通投智选 | 通投智选下，营销会投放到信息流和搜索场景 |
| 系统优选投放版位 | 系统将根据您设置的定向，选择预估效果最好的投放版位进行投放，并触达尽可能多的目标用户 |
