# 获取代理商信息

获取代理商信息

### 请求地址

[https://ad.oceanengine.com/open_api/2/agent/info/](https://ad.oceanengine.com/open_api/2/agent/info/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710518158351)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_ids 必填 | number[] | 代理商ids |
| fields | string[] | 查询字段集合, 如果指定, 则返回结果数组中, 每个元素是包含所查询字段的字典，默认全部指定.允许值: "agent_id", "agent_name", "customer_id", "customer_name","company_id", "company_name", "create_time", "role" |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/34/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/34/docs/1696710760866831) |
| data | json | json返回值 |
| agent_id | number | 代理商ID |
| agent_name | string | 代理商名称 |
| customer_id | number | 客户id |
| company_id | number | 公司id |
| company_name | string | 公司名称 |
| account_status | string | 账户状态 可前往[【枚举值】](https://open.oceanengine.com/labels/34/docs/1696710760171535)文档查询【账户状态】 |
| create_time | string | 注册时间 |
| role | string | 角色 可前往[【枚举值】](https://open.oceanengine.com/labels/34/docs/1696710760171535)文档查询【账户角色】 |
| request_id | string | 请求日志id |
