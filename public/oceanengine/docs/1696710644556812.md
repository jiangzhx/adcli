# 获取智能电话列表

本接口用户获取青鸟线索通智能电话列表，可根据智能电话id，智能电话创建时间，删除标记进行过滤。

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/smartphone/get/](https://ad.oceanengine.com/open_api/2/clue/smartphone/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710644556812)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| instance_ids | number[] | 智能电话ID列表 若指定instance_ids进行过滤，其他的过滤条件将不生效 |
| start_time | string | 起始时间，格式：%Y-%m-%d，默认7天前，即不指定起止时间获取最近7天数据 |
| end_time | string | 截止时间，格式：%Y-%m-%d，默认当天，即不指定起止时间获取最近7天数据 |
| is_del | string | 删除标记，默认获取未删除的数据 可选值：0，1（0表示未删除，1表示已删除） |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值为10，最大不超过50 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| key | string | advertiser_id的加密字段 |
| list | object[] | 智能电话列表 |
| call_display | number | 是否显示真实手机号 1：显示真实手机号 0：显示虚拟号（默认） |
| create_time | function | 表单创建时间，格式：%Y-%m-%d %H:%M:%S |
| instance_id | number | 生成的智能电话ID |
| name | string | 智能电话名称，长度不超过20 |
| no_disturb_times | object[] | 免打扰开启时间段(最多5组，不填或者空数组代表不开启免打扰) 所选时间内，用户通过营销页所拨电话将由语音机器人应答，商家将不会收到来电 非工作时段所有来电及录音可在飞鱼查看，并支持回拨 触发非工作模式同样记转化 |
| start_time | string | 免打扰开始时间，格式如"07:55:00" |
| end_time | string | 免打扰结束时间，格式如"07:55:00" |
| phone_number | string | 电话号码 支持固话(如01012345678)、手机号 暂不支持400号码绑定 |
| validate_type | string | 对用户手机号的验证类型，可选值: CLUE_PRIORITY：优先线索量。对系统判定异常电话进行语音验证，未通过验证的用户线索仍然获取 VALIDITY_PRIORITY： 优先有效性。对系统判定异常电话进行语音验证，只获取通过验证的用户线索 NONE_VERIFICATION：不进行验证。无论什么类型用户，均不进行语音验证 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
