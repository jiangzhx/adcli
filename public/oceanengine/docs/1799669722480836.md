# 资金共享-查询转账列表

通过筛选条件捞取转账记录

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/cg_transfer/wallet/transfer/list/](https://api.oceanengine.com/open_api/v3.0/cg_transfer/wallet/transfer/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1799669722480836)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

account_id

number

鉴权账户id

account_type

string

鉴权账户类型 可选值:

biz_request_no

string

请求id，推荐uuid，方便请求链路对齐

query_begin_time

string

查询开始时间(转账创建时间) yyyy-MM-dd HH:mm:ss

query_end_time

string

查询结束时间(转账创建时间) yyyy-MM-dd HH:mm:ss

query_wallet_id_list

number[]

需要查询的大小钱包id，加款方与减款方都需要包含在内

payee_id

number

加款方id，可选，需要包含在query_wallet_id_list

remitter_id

number

减款方id，可选，需要包含在query_wallet_id_list

page_info

object

分页信息

page_size

number

分页大小，范围：1-100

page_num

number

页码，从1开始，仅支持page_size * page_num < 1w的深度分页
