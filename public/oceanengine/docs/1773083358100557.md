# SDK使用介绍

## SDK简介

欢迎使用巨量引擎开放平台开发者套件（SDK），当前支持[Java](https://open.oceanengine.com/labels/7/docs/1773083420135496) 和[Go](https://open.oceanengine.com/labels/7/docs/1773083436761088)两个开发语言，后续会逐步扩展其他语言，有相关语言诉求也欢迎反馈给我们

## SDK开源

Java：[https://github.com/oceanengine/ad_open_sdk_java](https://github.com/oceanengine/ad_open_sdk_java)

Go：[https://github.com/oceanengine/ad_open_sdk_go](https://github.com/oceanengine/ad_open_sdk_go)
