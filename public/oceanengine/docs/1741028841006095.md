# 获取单元列表

获取巨量引擎体验版 单元列表

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/promotion/list/](https://api.oceanengine.com/open_api/v3.0/promotion/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1741028841006095)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方式

GET Request-Query

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token | string | 授权Access-Token，获取方法见接口文档【[获取Access-Token](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1696710505596940)】 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| filtering | object | 过滤条件 若此字段不传或传空，则视为无限制条件 |
| ids | number[] | 按单元ID过滤，范围为1-20 |
| name | string | 单元名称，长度是1-50个字（两个英文字符占1个字，该字段采取模糊查询的方式） |
| project_id | number | 按项目id过滤 |
| delivery_mode | string | 投放模式，允许值：MANUAL手动投放 ，PROCEDURAL自动投放 |
| status | string | 单元状态过滤 注意：此参数已不再维护，请您使用下方的status_first、status_second 允许值： NOT_DELETED 不限 、ALL 不限（包含已删除）、OK 投放中、DELETED 已删除、PROJECT_OFFLINE_BUDGET 项目超出预算、PROJECT_PREOFFLINE_BUDGET 项目接近预算、TIME_NO_REACH 未到达投放时间、TIME_DONE 已完成、NO_SCHEDULE 不在投放时段、AUDIT 新建审核中、REAUDIT 修改审核中、FROZEN 已终止、AUDIT_DENY 审核不通过、OFFLINE_BUDGET 单元超出预算、OFFLINE_BALANCE 账户余额不足、PREOFFLINE_BUDGET 单元接近预算、DISABLED 已暂停、PROJECT_DISABLED 已被项目暂停、LIVE_ROOM_OFF 关联直播间不可投、PRODUCT_OFFLINE 关联商品不可投，、AWEME_ACCOUNT_DISABLED 关联抖音账号不可投、AWEME_ANCHOR_DISABLED 锚点不可投、DISABLE_BY_QUOTA 已暂停（配额达限） |
| status_first | string | 单元一级状态过滤，不传默认返回不限（不包含已删除） 允许值： PROMOTION_STATUS_ALL 不限包含已删除 PROMOTION_STATUS_NOT_DELETE 不限（不包含已删除） PROMOTION_STATUS_ENABLE投放中 PROMOTION_STATUS_DISABLE未投放 PROMOTION_STATUS_FROZEN已终止 PROMOTION_STATUS_DONE已完成 PROMOTION_STATUS_DELETED已删除 |
| status_second | string | 单元二级状态过滤 当status_first = PROMOTION_STATUS_DISABLE时传入有效 允许值： AUDIT_DENY 审核不通过 AUDIT 新建审核中 REAUDIT 修改审核中 DISABLED 已暂停 DISABLE_BY_QUOTA 已暂停（配额达限） PROJECT_DISABLED 项目已被暂停 NO_SCHEDULE 不在投放时段 TIME_NO_REACH 未到达投放时间 OFFLINE_BALANCE 账户余额不足 BALANCE_OFFLINE_BUDGET 账户超出预算 PROJECT_OFFLINE_BUDGET 项目超出预算 PROMOTION_OFFLINE_BUDGET 单元超出预算 PRODUCT_OFFLINE 商品不可投 AWEME_ACCOUNT_DISABLED 关联抖音号不可投 LIVE_ROOM_OFF 关联直播间不可投 AWEME_ANCHOR_DISABLED 锚点不可投 BUDGET_GROUP_OFFLINE_BUDGET 预算组超出预算 STAR_TASK_NOT_STARTED星广联投任务未开始 PROJECT_REACH_DAILY_DELIVERY 项目已达每日投放时长 MATERIAL_STATUS_SYSTEM_DISABLE_QUOTA_LIMIT 素材排队中 |
| promotion_create_time | string | 单元创建时间，格式yyyy-mm-dd，表示过滤出当天创建的单元 |
| promotion_modify_time | string | 单元更新时间，格式yyyy-mm-dd，表示过滤出当天更新的单元 |
| promotion_modify_start_time | string | 单元更新开始时间，该筛选支持您细分到秒级查询一段时间内修改过的单元信息 格式yyyy-mm-dd hh:mm:ss 注意 必须与结束时间promotion_modify_end_time结合使用，注意时间跨度不能超过30天 开始时间<=结束时间 可以与promotion_modify_time同时使用，但不太有必要。直接使用promotion_modify_start_time、promotion_modify_end_time组合筛选即可 |
| promotion_modify_end_time | string | 单元更新结束时间，该筛选支持您细分到秒级查询一段时间内修改过的单元信息 格式yyyy-mm-dd hh:mm:ss 注意 必须与开始时间promotion_modify_start_time结合使用，注意时间跨度不能超过30天 开始时间<=结束时间 可以与promotion_modify_time同时使用，但不太有必要。直接使用promotion_modify_start_time、promotion_modify_end_time组合筛选即可 |
| reject_reason_type | string | 审核建议类型，允许值：NONE 无建议、REVIEW_REJECT 审核不通过、LOW_MATERAIL 低质素材、DISCOMFORT 引人不适、QUALITY_POOR 素材质量低、EXAGGERATION 夸大宣传、ELSE 其他 |
| learning_phase | string[] | 学习期状态 允许值：LEARNING（学习期中）、LEARNED（学习期结束）、LEARN_FAILED（学习期失败) 【注意】以下场景获取的学习期状态不具备参考价值： landing_type = APP， delivery_mode = PROCEDURAL 搜索周期稳投，ad_type = SEARCH，delivery_type = DURATION |
| has_carry_material | string | 按素材搬运打压状态过滤 TRUE：包含打压风险的单元 FALSE：不包含打压风险的单元 |
| blue_flow_package_setting | string | 是否开启蓝海流量投放过滤，允许值：ON、OFF |
| star_delivery_type | string | 是否为星广联投单元 STAR_DELIVERY 星广联投单元 NOT_STAR_DELIVERY 非星广联投单元 |
| star_auto_delivery_switch | string | 星广联投全自动化开关，枚举值： ON 开启 OFF 关闭 注意：筛选全自动化项目下的单元时，筛选条件不需要设置star_delivery_type参数 |
| fields | string[] | 查询字段集合，如果指定则应答结果仅返回指定字段 可参考应答参数返回的指标字段 |
| including_material_atrributes | string | 如需查询一个单元下是否包含搬运打压状态的视频素材，以及一个视频素材是否存在搬运风险，请传入此参数，并定义枚举值为RETURN_CARRY_DATA 可选值：RETURN_CARRY_DATA 返回视频素材的搬运属性 如果不传此参数，应答参数将不会返回搬运相关的字段：has_carry_material、is_carry_material |
| page | number | 页数默认值：1，page范围为[1,99999] page*page_size翻页方式仅支持≤10000的单元条数查询，若超过10000条单元数据，请使用cursor*count翻页方式 |
| page_size | number | 页面大小默认值：10，page_size范围为[1,20] page*page_size翻页方式仅支持≤10000的单元条数查询，若超过10000条单元数据，请使用cursor*count翻页方式 |
| cursor | number | 页码游标值：第一次拉取，传入0 cursor 翻页形式不支持与筛选参数ids同时传入 page与cursor同时传入时，cursor优先级大于page；同时不传入默认走page逻辑 page+page_size与cursor+count为两种分页方式，返回参数只返回与入参对应的分页参数 |
| count | number | 页面数据量，count范围为[1,20] page与cursor同时传入时，cursor优先级大于page；同时不传入默认走page逻辑 page+page_size与cursor+count为两种分页方式，返回参数只返回与入参对应的分页参数 |
