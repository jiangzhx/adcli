# 获取试玩/直玩素材列表

可通过本接口获取通过MAPI或巨量引擎工作台创建的试玩或直玩素材

接口仅支持白名单账号：白名单Key Bp_Asset_Game_WhiteList，如有加白诉求可联系对接销售/运营

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/file/playable/list/](https://api.oceanengine.com/open_api/v3.0/file/playable/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1836615534374475)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

字段

类型

描述

可选过滤参数

number[]

素材ID列表

app_material_ids

number[]

试玩/直玩包Material_id列表

keyword

string

试玩/直玩素材查询，可通过素材名称/素材ID进行查询

signatures

string[]

素材MD5值

start_time

string

根据素材上传时间进行过滤的起始时间，与end_time搭配使用

格式：yyyy-mm-dd hh:mm:ss

end_time

string

根据素材上传时间进行过滤的截止时间，与end_time搭配使用

格式：yyyy-mm-dd hh:mm:ss

status

string

素材状态，枚举值：

ONLINE 素材在线

OFFLINE 已下线

audit_status

string

审核状态，枚举值：

STATUS_CONFIRMING 审核中

STATUS_CONFIRMED已通过

STATUS_CONFIRM_FAIL 已拒绝

app_ids

string[]

直玩-小游戏app_id（仅直玩素材类型支持）

page_size

number

分页大小（默认为20）

page

number

页码（默认为1）
