# 素材推送

本接口支持推送投放账户下已有的视频素材

# 请求地址

[https://api.oceanengine.com/open_api/2/file/material/bind/](https://api.oceanengine.com/open_api/2/file/material/bind/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710604852236)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

素材归属客户

target_advertiser_ids

number[]

待推送的客户，数量限制：<=50

video_ids

string[]

视频ID，数量限制：

image_ids

string[]

图片ID，数量限制：

carousel_ids

number[]

图集ID，数量限制：

.method("POST", body)
.addHeader("Content-Type", "application/json")
.addHeader("Access-Token", ACCESS_TOKEN)
.build();
