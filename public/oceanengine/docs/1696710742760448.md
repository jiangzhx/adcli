# 查询安卓应用信息

查询安卓应用信息 可通过请求参数advertiser_id客户id 查询客户下应用列表及应用详细信息，具体返回信息见返回参数

与【[查询安卓应用信息（支持所有账户体系）](https://open.oceanengine.com/labels/34/docs/1710414664846349)】接口的区别仅仅是支持的账户体系不同

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/app/get/](https://ad.oceanengine.com/open_api/2/tools/app_management/app/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710742760448)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| search_key | string | 搜索关键字 appid或者应用名，可以为空，可以传中文 长度不超过50 |
| search_type | string | 搜索类型: ALL:查询全部，包括创建和被共享的应用（默认） CREATE_ONLY:只查询客户创建的应用 SHARED_ONLY:只查询被共享的应用 |
| status | string | 应用状态: ALL:所有状态 AUDIT_DOING:审核中 AUDIT_REJECTED:审核失败 AUDIT_ACCEPTED:审核成功，即待发布 ENABLE:已发布（默认） |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值为10，最大不超过200 |
| create_time | object | 按创建时间查询的时间范围 |
| start_time | string | 创建起始时间，默认为空，格式：%Y-%m-%d |
| end_time | string | 创建结束时间，默认为当天，格式：%Y-%m-%d |
| publish_time | object | 按发布时间查询的时间范围 |
| start_time | string | 发布起始时间，默认为空，格式：%Y-%m-%d |
| end_time | string | 发布结束时间，默认为当天，格式：%Y-%m-%d |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] | 应用列表 |
| package_id | string | 应用包ID |
| package_name | string | 包名 |
| app_cloud_id | number | app id |
| app_name | string | 应用名 |
| version | string | 版本号 |
| download_url | string | 下载地址 |
| icon_url | string | icon地址 |
| publish_time | string | 发布时间，格式：%Y-%m-%d %H:%M:%S |
| update_time | string | 更新时间，格式：%Y-%m-%d %H:%M:%S |
| create_time | string | 创建时间，格式：%Y-%m-%d %H:%M:%S |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
