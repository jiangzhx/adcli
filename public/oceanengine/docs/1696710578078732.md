# 获取商品列表

根据商品库查询商品库中的商品列表

通过商品库ID（product_platform_id）+过滤条件查询商品列表信息

【1.28 参数废弃】原出参中的audit_reason_text及audit_hit_field字段作废，审核拒绝原因以audit_reasons为准

请求参数中必须填写商品库ID

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/detail/get/](https://ad.oceanengine.com/open_api/2/dpa/detail/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710578078732)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET Query-String

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| product_platform_id 必填 | number | 商品库ID ，可通过[【获取商品库信息】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710577036288)获取 |
| filtering | object | 过滤条件 |
| dpa_categories | number[] | 商品行业分类，长度限制 50，使用的三级行业ID ，可从 [【获取行业列表】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710677879808) 接口获取 |
| is_recommend | number | 是否系统推荐，1表示只看系统推荐，0表示不限系统推荐，默认为0 |
| product_id | number | 商品ID，精准匹配，与product_name（商品名称）同时传入，以商品ID为准 |
| product_name | string | 商品名称，模糊匹配，长度限制50字，与product_id（商品ID）同时传入，以商品ID为准 |
| status | number | 商品投放状态，枚举值：0代表不可投放，1代表可投放 |
| audit_status | number | 商品审核状态，枚举值：1代表审核通过，2代表审核拒绝，-1代表审核中 |
| offline_modify_time | string | 查询最近一次修改时间小于入参时间的下架商品，传参格式yy-mm-dd，如2026-01-01 |
| page | number | 页码， 默认值: 1 |
| page_size | number | 页面数据量， 默认值: 20，最大 100 注：page*page_size需小于30000 |
