# 操作日志查询

可查询巨量营销、巨量本地推、巨量千川后台操作日志，默认查询最近7天的数据，最多查询跨度为一个月。

当前一次最多支持查询2000条日志数据，超过2000条返回的分页信息可能存在不准确的问题，建议您缩短查询时间（7天以内）。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/log_search/](https://ad.oceanengine.com/open_api/2/tools/log_search/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710682956815)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 业务账户ID |
| object_id | number[] | 操作对象ID，可以为campaign_id、ad_id、creative_id等业务下对应对象ID 长度限制：[1,20] |
| start_time | string | 日志查询开始时间 格式：yyyy-mm-dd HH:MM:SS |
| end_time | string | 日志查询结束时间 格式： yyyy-mm-dd HH:MM:SS |
| page | number | 页码，默认值: 1 |
| page_size | number | 页面大小，默认值: 10，最大值：20 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | json返回值 |
| page_info | json | 分页信息 |
| page | number | 页数 |
| page_size | number | 页面大小 |
| total_number | number | 总数。 当前一次最多支持查询2000条日志数据，超过2000条返回的分页信息可能存在不准确的问题，建议您缩短查询时间（7天以内）。 |
| total_page | number | 总页数 当前一次最多支持查询2000条日志数据，超过2000条返回的分页信息可能存在不准确的问题，建议您缩短查询时间（7天以内）。 |
| logs | object | 日志详情 |
| content_title | string | 操作内容 |
| object_type | string | 操作对象类型 |
| object_id | int | 操作对象ID |
| object_name | string | 操作对象名称 |
| create_time | string | 操作时间 |
| content_log | string[] | 操作前后内容 |
| operator | string | 操作人 |
| opt_ip | string | 操作IP |
| request_id | string | 请求日志id |
