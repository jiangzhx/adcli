# 微信加粉组件：

客户通过在橙子建站落地页中放置微信号/二维码的方式，引导用户跳转微信并导流至微信个人号/公众号/企业微信号中后进行转化。当前支持的加粉方式有微信小程序加粉、微信复制加粉。
