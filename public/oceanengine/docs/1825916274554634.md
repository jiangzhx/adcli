# 获取采纳素材修复建议任务结果

注意：本接口为异步接口，需与[创建采纳「拒审素材修复建议」任务API](https://open.oceanengine.com/labels/7/docs/1827080076923659)搭配使用

您可通过本接口查询采纳的最新状态

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/reject_material/ai_repair_accept_task/list/](https://api.oceanengine.com/open_api/v3.0/reject_material/ai_repair_accept_task/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1825916274554634)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 应答字段

字段

类型

描述

code

number

message

string

data

json

result_list

object[]

采纳结果

ai_repair_id

number

已创建采纳任务的修复建议id

ai_repair_status

string

采纳状态，枚举值如下

SUCCESS 采纳成功

FAILED 采纳失败

PROCESSING 采纳中

NO_TASK 采纳任务不存在，可能为未创建采纳任务/已过期/营销已删除/营销下素材已删除

ai_repair_error_message

string

采纳失败的原因，仅当ai_repair_status=FAILED 时返回

request_id

string
