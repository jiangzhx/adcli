# 查看rit数据

查看穿山甲投放版位数据，包括付费率、消耗等数据，默认查询历史30天数据。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/union/flow_package/report/](https://ad.oceanengine.com/open_api/2/tools/union/flow_package/report/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710723136512)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

GET

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| filter必填 | json | 筛选条件 |
| ad_ids | number[] | 营销 id 列表，最大数量限制：1000 |
| campaign_ids | number[] | 计划组列表，最大数量限制：1000 |
| rits | number[] | 投放版位列表，最大数量限制：1000 |
| start_time | string | 开始时间，格式为"yyyy-mm-dd" 限制范围100天内 默认7天前（不包括当天），即不指定起止时间获取最近7天数据 |
| end_time | string | 结束时间，格式为"yyyy-mm-dd" 默认昨天，即不指定起止时间获取最近7天数据 |
| order_field | string | 排序字段，所有的统计指标均可参与排序 默认按rit排序 |
| order_type | string | 排序方式 枚举值：ASC（升序），DESC（降序） 默认：DESC |
| page | number | 页数 默认值: 1 |
| page_size | number | 页面大小,限制范围[1, 100] 默认值: 10 |
