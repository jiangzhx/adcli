# 获取推送数据列表

该接口用于查询2天内SPI推送给开发者回调地址的数据

# 请求地址

[https://ad.oceanengine.com/open_api/2/spi_task/get/](https://ad.oceanengine.com/open_api/2/spi_task/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1725097846184964)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| APP-Access-Token 必填 | string | 授权app-access-token，获取方法见接口文档[【获取APP-Access-Token】](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1713655428885516) |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| app_id 必填 | number | 应用ID |
| subscribe_id 必填 | number | 订阅任务ID |
| start_date | string | 开始时间，默认值为当前时间前推一个小时，格式：yyyy-mm-dd hh:mm:ss |
| end_date | string | 结束时间，默认值为当前时间前推30分钟，格式：yyyy-mm-dd hh:mm:ss |
| status | string | 推送数据状态，所有数据ALL、失败数据FAIL、成功数据SUCCESS，默认值为ALL |
| cursor | number | 页码游标值，第一次拉取，传入0 |
| count | number | 页面数据量 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| list | object[] | 数据列表 |
| task_id | number | 数据id |
| data | json string | 推送数据，返回格式同推送数据 |
| status | string | 推送状态，待推送WAITING、失败FAIL、成功SUCCESS |
| cursor_info | object | 游标分页信息，当分页方式为cursor+count时返回 |
| has_more | bool | 是否有下一页 |
| count | number | 页面数据量 |
| cursor | number | 下一次分页拉取的游标值 |
| request_id | string | 请求日志id |
