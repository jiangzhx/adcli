# 新增 Adv 订阅

对于SPI订阅任务，支持「App维度」/「订阅任务维度」管理推送账户列表

对于RDS订阅任务，支持「App维度」/「订阅任务维度」管理推送账户列表

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/subscribe/accounts/add/](https://api.oceanengine.com/open_api/v3.0/subscribe/accounts/add/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1818567687958707)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

App-Access-Token

string

应用级token，区别于普通的Access Token，通过【[获取APP Access Token](https://ad.oceanengine.com/openapi/doc/index.html?id=1713655428885516)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

APP-Access-Token

string

# 请求参数

字段

类型

描述

app_id

number

开放平台调用MAP所使用的APPID

subscribe_task_id

number

订阅任务ID

注意：当subscribe_package_type=TASK时生效且必填

core_user_id

number

授权用户id

advertiser_ids

number[]

需要添加的账户id，一次请求最大支持500个账户

subscribe_package_type

string

自定义订阅账户包类型， 可选值:

account_type

string

添加的账户类型，可选值:
