# 查看应用共享范围

查询应用共享范围，查询旧版巨量引擎工作台下某个应用的共享范围

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/share_account/list/](https://ad.oceanengine.com/open_api/2/tools/app_management/share_account/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1710412460273664)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| organization_id必填 | number | 旧版巨量引擎工作台id |
| package_id必填 | string | 应用包ID |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值为10，最大值200 |
| search_type | string | 共享类型 允许值：ORGANIZATION_SHARE 共享的旧版巨量引擎工作台范围、OTHER 共享的其他账号 默认：ORGANIZATION_SHARE |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| list | object[] | 共享账号列表 |
| share_mode | string | 共享类型 如果共享类型为PART，读取account_info 如果共享类型为ALL，读取all_account，即表示该业务线下的全量账号 如果共享类型为COMPANY ，读取all_account_by_company，即表示旧版巨量引擎工作台下所属主体的全量账号。 枚举值：PART 指定账户共享、 ALL 旧版巨量引擎工作台内某类账户所有共享、COMPANY 公司主体内某类型所有账户共享 |
| account_info | object | 共享账号信息 |
| account_id | number | 账号id |
| account_type | string | 账号类型 枚举值：AD 客户、STAR 星图、BP 旧版巨量引擎工作台 |
| all_account | object | 共享账号类别 |
| account_type | string | 账号类型 枚举值：AD 客户、STAR 星图 |
| all_account_by_company | object | 主体信息 |
| company_id | number | 公司id |
| account_type | string | 账号类型 枚举值：AD 客户、STAR 星图 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
