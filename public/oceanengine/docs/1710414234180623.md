# 创建安卓应用分包 （支持所有账户体系）

创建应用分包，支持所有账户体系下创建应用分包

母包package创建成功后，可以通过当前接口创建分包，支持AD、巨量引擎工作台-旧版及巨量引擎工作台-升级版客户。

目前API开放平台不支持母包package的创建，创建母包需要前往 投放平台-资产-移动应用

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/extend_package/create_v2/](https://ad.oceanengine.com/open_api/2/tools/app_management/extend_package/create_v2/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1710414234180623)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| account_id | number | 账户id，account_type类型对应账户ID（近期升级到巨量引擎工作台-升级版的客户请传入应用包所属旧版巨量引擎工作台/子旧版巨量引擎工作台ID，账户类型传入旧版巨量引擎工作台） |
| account_type | string | 账户类型 允许值： 旧版巨量引擎工作台 巨量引擎工作台-升级版/旧版 AD 客户账号 STAR 星图 |
| package_id | string | 应用包ID，可通过「查询应用信息」接口查询获取 |
| mode | string | 分包模式 允许值： Auto 自动生成渠道号 Manual 自定义渠道号信息 |
| channel_count条件必填 | number | 自动创建分包数量，数量限制：1~100分包模式mode为Auto时可用且必填 |
| channel_list条件必填 | object[] | 自定义渠道号信息，数量限制：1~100分包模式mode为Manual时可用且必填 |
| channel_id必填 | string | 渠道号，渠道号ID支持英文，数字，下划线和连字符-，不超过50个字符，超出部分会被截断 示例：oceanengine_ads_sample-12 |
| remark | string | 备注，渠道包备注信息，至多20个字符，超出部分会被截断处理 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| package_id | string | 应用包id |
| request_id | string | 请求的日志id，唯一标识一个请求 |
