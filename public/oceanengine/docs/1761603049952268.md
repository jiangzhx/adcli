# 一键起量（巨量营销升级版）

相关链接：

一键起量旨在给客户提供一个花费可控的激进提价工具，并且通过短时详细数据披露，为客户提供更多数据价值。

一键起量使用条件：oCPM计划、非nobid&自动化计划（旧版巨量引擎工作台，省心投），营销状态为投放中
