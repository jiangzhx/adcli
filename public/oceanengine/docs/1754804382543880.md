# 获取屏蔽词

获取屏蔽词接口

注：当advid下所有的营销都为绑定抖音号投放时，需上传aweme_id，根据aweme_id获取屏蔽词

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/comment/terms_banned/get/](https://api.oceanengine.com/open_api/v3.0/tools/comment/terms_banned/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1754804382543880)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id |
| aweme_id | string | 抖音号，当is_apply_to_adv不传或为false时，aweme_id生效 |
| is_apply_to_adv | bool | 是否应用于当前账户，当is_apply_to_adv不传或为false时，aweme_id生效 |
| page | number | 页码 |
| page_size | number | 页面大小 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| terms | string[] | 屏蔽词列表 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
