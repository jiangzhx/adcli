# 创建定向包

不同类型定向包可应用于不同类型的计划；

该接口同时支持巨量营销和巨量营销升级版，「AD | 营销计划（若定向包被应用于巨量营销升级版，此处含义为营销 promotion）」 「CAMPAIGN」计划组（若定向包被应用于巨量营销升级版，此处含义为项目project）」

落地页：可用于营销目的为销售线索收集或营销目的为应用且下载方式为落地页的计划 应用（Android）：可用于营销目的为应用且下载方式为Android下载链接的计划 应用（iOS）：可用于营销目的为应用且下载方式为iOS下载链接的计划 其余类型：可应用于营销目的为该类型名称的计划

每一个投放账户下最多只能创建200个定向包

# 请求地址

[https://ad.oceanengine.com/open_api/2/audience_package/create/](https://ad.oceanengine.com/open_api/2/audience_package/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710735901711)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
