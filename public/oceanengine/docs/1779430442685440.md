# 获取升级版商品列表

获取行业产品中心「产品管理-升级版」商品列表

# 请求地址

[https://api.oceanengine.com/open_api/2/dpa/clue_product/list/](https://api.oceanengine.com/open_api/2/dpa/clue_product/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1779430442685440)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

filtering

object

过滤条件

product_ids

number[]

商品ID精确搜索

product_name

string

商品名称模糊搜索

audit_status

string[]

商品审核状态 可选值:

category_ids

number[]

类目id，会级联查询所有叶子类目

category_name

string

类目名称，支持模糊搜索

completion_status

string[]

商品必填字段完整性

product_id_or_name_search

string

商品ID或商品名称查询

statuses

string[]

可投状态过滤

rels

string[]

商品权限关系过滤，允许值：

REL_COP 授权

REL_OWN own

page

number

页码

page_size

number

页面大小，默认值为10，传参范围[1,100]
