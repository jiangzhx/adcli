# 创建异步文件上传任务

创建异步下载任务「支持所有账户体系」 可通过请求参数account_id 账户id和account_type 账户类型，执行异步下载&上传到TOS&apk解析。

# 请求地址

https://api.oceanengine.com/open_api/2/tools/app_management/upload_task/create/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1714118126561421)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

POST
