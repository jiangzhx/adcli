# 获取线索列表

该接口用于获取客户的飞鱼线索列表。

其中应答参数是按creative_time倒序返回的。如果您有数据处理的需求，可以参考此排序逻辑。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/clue/get/](https://ad.oceanengine.com/open_api/2/tools/clue/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710631271436)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_ids 必填 | number[] | 客户ids，上限：100 仅允许传入巨量引擎工作台（旧版巨量引擎工作台）账户授权后获取到的adv 当查询的投放账户主体与开发者主体不一致，授权时还需勾选「敏感物料授权」，方可调用本接口 |
| start_time 必填 | string | 查询起始时间，格式：yyyy-MM-dd 或者 yyyy-MM-dd hh:mm:ss 2024年12月17日起，飞鱼将对历史线索的保留时间实施限制，仅保留近2年内的线索数据。超过2年的线索将不再支持接口查询。 |
| end_time 必填 | string | 查询截止时间，格式：yyyy-MM-dd 或者 yyyy-MM-dd hh:mm:ss |
| page | number | 页数 默认值: 1 |
| page_size | number | 页面大小 默认值: 10 |
