# 查询项目信息

# 请求地址

[https://api.oceanengine.com/open_api/2/query/project/](https://api.oceanengine.com/open_api/2/query/project/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1784608638831619)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

agent_id

number

代理商ID

filtering

object

过滤器

customer_id

number

客户ID

platform_list

string[]

平台列表，可选值

AD:巨量营销

NATIVELIFE:本地推

QIANCHUAN:千川

serving_type_list

string[]

投放类型

PINPAI：品牌

PINPAIPMP：品牌PMP

XIAOGUO：效果

XIAOGUO_RTB：效果RTB

project_status_list

string[]

项目状态

APPROVED：审批通过

project_start_date_begin

string

项目开始时间-开始 例如 2023-07-06

project_start_date_end

string

项目开始时间-结束 例如 2023-07-06

project_end_date_begin

string

项目结束时间-开始 例如 2023-07-06

project_end_date_end

string

项目结束时间-结束 例如 2023-07-06

receipt_status_list

string[]

项目回款状态

COMPLETED：已回款

PART_COMPLETED：部分回款

UNCOMPLETED：未回款

deadline

string

应回款日期

advertiser_id

number

投放账号ID

count

number

页码游标值（单页限制，最大1000，必填）

cursor

number

页码游标值（根据cursor进行滚动查询，第一次传-1，后续传前一次返回列表中的最小值）
