# 获取穿山甲渠道RTA联合实验数据

## 功能

该接口用于查询穿山甲渠道的RTA联合实验数据

### 请求地址

[https://api.oceanengine.com/open_api/2/report/rta_exp/get/](https://api.oceanengine.com/open_api/2/report/rta_exp/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1738683217044484)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| rta_id必填 | number | RTA ID |
| start_date必填 | string | 开始时间，格式YYYYMMDD，示例：20220701 |
| end_date必填 | string | 结束时间，格式YYYYMMDD，示例：20220701 |
| strategy必填 | number | 联合实验策略。共10个实验分桶标记，每个分桶代表一种策略。允许值：0 代表基线策略 ，传入1、2、3、4、5、6、7、8、9 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| data_array | object[] | 返回数据结构体 |
| date | string | 数据统计日期，格式YYYY-M-D |
| strategy | string | 联合实验策略，请求入参 |
| show | number | 展现数据-展示数。营销展示给用户的次数。计算方式：经平台判定有效且被计费的展示次数 |
| click | number | 展现数据-点击数。当头条用户点击营销素材时，触发点击事件，该事件被认为是一次有效的营销点击 |
| convert | number | 转化数据-转化数。将转化数记录在转化事件发生的时间上。建议客户考核成本时参考“转化数据（计费时间）”例如您的营销在早上8点进行了展示和点击，用户晚上19点发生了激活行为，我们会把激活数披露在晚上19点 |
| cost | number | 展现数据-总花费。表示营销在投放期内的预估花费金额,当天数据可能会有波动，次日稳定 |
| win_ratio | float | 竞胜率。竞胜率=竞胜数/参竞数，代表客户参竞请求的胜出比例，范围 0~1 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
