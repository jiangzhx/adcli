# 获取已创建资产详情（新）

应答参数返回的资产范围如下：

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/event/all_assets/detail/](https://api.oceanengine.com/open_api/2/tools/event/all_assets/detail/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1800988620664954)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户 id

asset_ids

number[]

资产id列表，list长度最长50

当账户下不存在该资产id时不会返回详情信息；当资产共享失效时，不会返回详情信息。

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

message

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

asset_list

object[]

资产列表

asset_id

number

资产id，不会返回已删除资产

asset_name

string

资产名称，不会返回已删除资产

asset_type

string

资产类型，枚举值:

site_id

number

橙子落地页站点id，当asset_type = TETRIS_EXTERNAL 时返回，表示该资产是基于哪一个橙子落地页站点创建的，在巨量营销-事件资产页面，橙子落地页显示为该站点id

app_type

string

应用类型，当asset_type = APP 时返回，枚举值：

app_id

number

应用ID，当asset_type = APP 时返回

app_name

string

应用名称，当asset_type = APP 时返回

package_id

string

应用包id，当asset_type = APP 时返回，当调用API创建资产时，Android应用必填，建议获取保存

package_name

string

应用包名，当asset_type = APP 时返回

download_url

string

应用下载链接，当asset_type = APP 时返回

quick_app_id

number

快应用id，当asset_type = QUICK_APP 时返回

quick_app_package_name

string

快应用包名，当asset_type = QUICK_APP 时返回

micro_app_id

string

小程序appid，当asset_type = MINI_PROGRAME 时返回

micro_app_instance_id

number

小程序资产id，当asset_type = MINI_PROGRAME 时返回

request_id

string

请求日志id
