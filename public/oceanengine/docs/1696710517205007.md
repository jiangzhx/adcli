# 更新客户信息

修改客户信息，可更改内容包括账户名称、联系人、手机号码、固定电话、备注等，除此之外其他内容不允许修改。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/agent/advertiser/update/](https://ad.oceanengine.com/open_api/2/agent/advertiser/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710517205007)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| name | string | 账户名称 |
| contacter | string | 联系人 |
| phonenumber | string | 手机号码 |
| telephone | string | 固定电话 |
| note | string | 备注 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/34/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/34/docs/1696710760866831) |
| data | object | json返回值 |
| advertiser_id | number | 客户ID |
| need_audit | number | 此次修改是否触发进入待审状态(1表示进入待审状态,0表示不进入待审状态) |
| request_id | string | 请求日志id |
