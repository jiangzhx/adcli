# 客户匹配转化用户

1、概述

客户将自己统计到的达到转化目标的用户的信息与头条发给客户的营销展示点击信息进行匹配。匹配基于二者的用户设备信息是否一致。

2、匹配方式

头条会提供给客户营销展示点击用户的IDFA、ANDROIDID、OPENUDID、IP、UA等设备信息，客户将其与自己监测到的转化用户的相关信息进行匹配。如果成功匹配，则将该用户的信息通过回调地址回调给头条；如果无法成功匹配，则无需将其回传给头条。

3、注意事项

(1)、建议客户基于多个设备信息进行匹配，以避免某项设备信息缺失造成的无法匹配（高版本安卓和IOS系统获取idfa的成功率偏低）。

(2)、请匹配7天内的转化，例如，如果监测到的“激活”是在营销点击行为的7天以后，将不会作为有效的转化。
