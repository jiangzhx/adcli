# 批量获取锚点预览url

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/native_anchor/qrcode_preview/get/](https://api.oceanengine.com/open_api/v3.0/native_anchor/qrcode_preview/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1786414975773770)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

anchor_ids

string[]

锚点ID，请传入同一anchor_type类型的锚点id，否则报错

anchor_type

string

锚点类型，允许值：

游戏：APP_GAME

网服：APP_INTERNET_SERVICE

电商：APP_SHOP

高级在线预约：ONLINE_SUBSCRIBE

字节小程序：MICRO_APP

字节小游戏：MICRO_GAME

购物车、SHOPPING_CART

外跳：INSURANCE

咨询：PRIVATE_CHAT

string

锚点id

anchor_qrcode_url

string

锚点预览的二维码信息url，有效期一天（需要使用抖音App扫码），只有当锚点关联营销时，才可查询到预览url

error_list

object[]

查询失败的锚点及查询失败原因

anchor_id

string

锚点id

error_message

string

查询失败的原因

request_id

string

请求日志id
