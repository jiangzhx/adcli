# 建站管理

## 概念介绍

#### 建站管理

在这个功能模块下，用户可以实现对落地页的管理（主要是增、删、改、查），其中包括橙子建站落地页管理、程序化落地页管理、第三方落地页管理三个功能模块。

创建生成的橙子建站落地页、第三方落地页可以在[营销投放](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710530107404)流程中使用。

#### 建站

即创建站点，站点是指存放落地页的地方。

#### 落地页

即创建营销时的创意详情页，也曾命名为应用下载详情页，用户点击营销中诸如“立即下载”此类文案的按钮以外的区域时所到达的页面，是创建营销创意时必不可少的重要环节。

#### 落地页种类

橙子建站落地页：指客户通过橙子建站工具创建的落地页； 程序化落地页：又称落地页优选组，由多个单落地页打包形成的落地页组。投放时，根据流量分配设置在多个落地页之前进行营销流量分配； 第三方落地页：指客户自有的落地页。

## 相关问题Q&A

Q：橙子落地页、程序化落地页、第三方落地页有什么区别和联系？

A：就落地页本身而言，橙子建站落地页和第三方落地页都属于自定义落地页，区别在于是通过橙子建站创建还是客户自主上传的； 而程序化落地页是多个单张落地页打包的落地页组。就营销投放而言，橙子建站落地页、第三方落地页都是客户自主选择进行手动投放的，而程序化落地页是根据流量分配设置在多个落地页之间进行营销流量分配，再进行投放的。
