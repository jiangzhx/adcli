# 获取音频素材（用于图文新建）

本接口支持查询投放账户下自2024年4月26日起上传的音频素材，不支持查询版权音频相关信息

# 请求地址

[https://api.oceanengine.com/open_api/2/file/audio/get/](https://api.oceanengine.com/open_api/2/file/audio/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1797569167671364)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

filtering

object

过滤条件

audio_ids

string[]

按音频id过滤，list长度 1-100 注意audio_ids，material_ids，audio_signatures三者只能选一个过滤，同时传入会报错

material_ids

number[]

按音频素材id（全局唯一）过滤，list长度1-100 注意audio_ids，material_ids，audio_signatures三者只能选一个过滤，同时传入会报错

signatures

string[]

按音频md5过滤，list长度1-100 注意audio_ids，material_ids，audio_signatures三者只能选一个过滤，同时传入会报错

sources

string[]

音频素材来源 枚举值大小写敏感，请严格按照定义的名称传参

AD_SITEAD后台本地上传

OPEN_API 开放平台上传

TTS_TEXT_TO_AUDIO文本转音频

start_time

string

根据音频上传时间过滤，与end_time搭配使用，格式：yyyy-mm-dd hh:mm:ss

end_time

string

根据音频上传时间过滤，与start_time搭配使用，格式：yyyy-mm-dd hh:mm:ss

page

number

页码

page_size

number

页面大小
