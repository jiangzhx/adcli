# 更新关键词属性

根据keyword_id或word_id更新搜索词的出价、匹配类型等属性信息，增量更新。

不支持更换关键词，如需要，可通过删除接口与创建接口实现

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/keyword/update/](https://api.oceanengine.com/open_api/v3.0/keyword/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1768011243105411)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

promotion_id

number

待创建搜索关键词的营销ID，仅支持搜索常规投放在单元层级设置关键词

project_id

number

待更新搜索关键词属性的项目ID，仅支持搜索极速智投在项目层级设置关键词

keywords

object[]

关键词列表

keyword_id

number

待更新属性的关键词ID，keyword_id和word_id二者传一个即可，都传则报错。

word_id

number

待更新属性的词ID，keyword_id和word_id二者传一个即可，都传则报错。

match_type

string

匹配类型:

bid

float

出价，取值范围：0.2至999.0。

当pricing为PRICING_OCPC/PRICING_OCPM时，不支持出价

当pricing为PRICING_CPC时，有效且必填

is_pause

number

是否停用。

允许值：1（停用）、0（启用）

bid_type

string

string

string

匹配类型:

number

string

is_pause

number
