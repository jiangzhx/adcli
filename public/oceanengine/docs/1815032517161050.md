# 代理商竞价投放数据

代理商竞价投放数据，对应方舟平台「投放-数据概览-汇总数据-竞价数据」

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/adv/bidding/list/query/](https://api.oceanengine.com/open_api/2/agent/adv/bidding/list/query/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1815032517161050)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求参数

字段

类型

描述

agent_id

number

代理商账户id

start_date

string

开始时间。格式：YYYY-MM-DD。闭区间,可选日期范围是今天及以前（允许包括今天）。

end_date

string

结束时间。格式：YYYY-MM-DD。闭区间,可选日期范围是今天及以前（允许包括今天）。开始时间与结束时间的跨度不能超过90天。

filtering

object

advertiser_ids

number[]

客户 id 列表，若选填该字段，则最少应上传1个客户id，最多支持同时查询100个客户。

company_name

string

active

string

first_industry

string

second_industry

string

account_status

string

cursor

number

游标。第一次查询不填或传0。后续查询使用出参的cursor字段

cursor_size

number
