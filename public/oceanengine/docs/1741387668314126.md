# 自定义报表

自定义报表是基于巨量引擎升级版对于数据报表提供新的接口能力，您可以自由选择和组合指标和维度，以此定义数据报表的字段。

相关指标和维度可通过[获取自定义报表可用维度和指标](https://open.oceanengine.com/labels/7/docs/1755261744248832)接口获取不同数据主题下的可用维度和指标。例如，想要获取由起量带来的增量数据，需要在筛选项中上传 field=is_boost &&values=1

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/report/custom/get/](https://api.oceanengine.com/open_api/v3.0/report/custom/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1741387668314126)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| data_topic | string | 数据主题，默认值：BASIC_DATA营销基础数据 枚举值： BASIC_DATA营销基础数据 QUERY_DATA搜索词数据 BIDWORD_DATA关键词数据 MATERIAL_DATA素材数据 PRODUCT_DATA产品数据 ONE_KEY_BOOST_DATA一键起量（巨量营销升级版） DMP_DATA人群包数据 VIDEO_DUARATION_DATA视频分秒数据 MATERIAL_BOOST_DATA 素材一键起量 |
| dimensions 必填 | string[] | 维度列表。 获取方式有2种： 调用API获取：可通过[【获取自定义报表可用维度和指标】](https://open.oceanengine.com/labels/7/docs/1755261744248832)接口获取不同数据主题下的可用维度和指标 通过营销平台直接生成参数：进入巨量营销升级版平台—>报表—>新建/编辑自定义报表—>API参数生成。获取到API参数生成后需在请求参数中手动添加对应的数据主题，否则将无法正确获取数据 |
| metrics 必填 | string[] | 指标列表 。 获取方式有2种： 调用API获取：可通过[【获取自定义报表可用维度和指标】](https://open.oceanengine.com/labels/7/docs/1755261744248832)接口获取不同数据主题下的可用维度和指标 通过营销平台直接生成参数：进入巨量营销升级版平台—>报表—>新建/编辑自定义报表—>API参数生成。该字段从前端自定义报表中获取，建议不要修改。获取到API参数生成后需在请求参数中手动添加对应的数据主题，否则将无法正确获取数据 |
| filters | object[] | 过滤条件。 获取方式有2种： 调用API获取：可通过[【获取自定义报表可用维度和指标】](https://open.oceanengine.com/labels/7/docs/1755261744248832)接口获取不同数据主题下的可用维度和指标 通过营销平台直接生成参数：进入巨量营销升级版平台—>报表—>新建/编辑自定义报表—>API参数生成。该字段从前端自定义报表中获取，建议不要修改。获取到API参数生成后需在请求参数中手动添加对应的数据主题，否则将无法正确获取数据 |
| field 条件必填 | string | 过滤的消耗指标字段 |
| type 条件必填 | number | 字段类型。允许值： 1 -固定枚举值 2 - 固定输入值 3 -数值类型 |
| operator 条件必填 | number | 处理方式。 允许值： 1 -等于 2 -小于 3 -小于等于 4 -大于 5 -大于等于 6 -不等于 7-包含 8 -不包含 9 -范围查询 10 -多个值匹配包含 11 -多个值匹配都要包含 |
| values 条件必填 | string[] | 过滤字段具体值 |
| start_time 必填 | string | 开始时间。格式为：yyyy-MM-dd hh:mm:ss。例如：2025-07-07 00:00:00 |
| end_time 必填 | string | 结束时间。格式为：yyyy-MM-dd hh:mm:ss。例如：2025-07-07 00:00:00 |
| order_by 必填 | object[] | 排序 |
| field | string | 排序字段。字段必须在选中的metrics或dimensions中。其中metrics所有字段支持排序。dimensions是否排序请参考维度、指标说明。 |
| type | string | 排序类型 可选值: ASC: 升序 DESC: 降序 |
| page | number | 页码。默认为1 |
| page_size | number | 页面大小。默认为10，最大100 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| rows | object[] |  |
| metrics | dict[string] | 指标数据 |
| dimensions | dict[string] | 维度数据 |
| total_metrics | dict[string] | 指标汇总数据 |
| page_info | object | 分页信息 |
| page | number | 页码 |
| page_size | number | 页面大小，即每页展示的数据量 |
| total_number | number | 总数 |
| total_page | number | 总页数 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
