# 开票-新建开票申请单（代理商版）

创建结算单开票接口，支持差额开票

# 请求地址

[https://api.oceanengine.com/open_api/2/create/statement_invoice/](https://api.oceanengine.com/open_api/2/create/statement_invoice/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1806785904427155)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json
