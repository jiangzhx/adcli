# 获取主体资质（新版）

获取客户的主体资质信息为全量接口，会返回客户所有主体资质，注意如果客户没有任何主体资质，这个接口的data将会是空。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/qualification/get/](https://api.oceanengine.com/open_api/v3.0/advertiser/qualification/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1743294338713611)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
