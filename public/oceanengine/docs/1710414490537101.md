# 查询安卓应用分包列表 （支持所有账户体系）

查询分包列表（支持所有账户体系），可查询该应用包相关信息和包含的分包信息。

不受分包新建方式（系统自动、手动上传渠道号分包）影响，都可以获取到

本接口能力覆盖旧接口[查询应用分包列表](https://open.oceanengine.com/labels/7/docs/1696710745401356)，且支持多账户体系（巨量引擎工作台-旧版及巨量引擎工作台-升级版、AD、星图），因此可迁移至本接口获取应用分包列表

# 请求地址

[https://api.oceanengine.com/open_api/2/tools/app_management/extend_package/list_v2/](https://api.oceanengine.com/open_api/2/tools/app_management/extend_package/list_v2/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1710414490537101)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

# 请求参数

字段

类型

描述

account_id

number

账户ID，account_type对应的账户ID（近期升级到巨量引擎工作台-升级版的客户请传入应用包所属旧版巨量引擎工作台/子旧版巨量引擎工作台ID，账户类型传入旧版巨量引擎工作台）

account_type

string

账户类型，可选值:

package_id

string

应用包ID，可通过「[查询应用信息](https://open.oceanengine.com/labels/8/docs/1696710742760448)」接口查询获取

page

number

页码，默认值为1

page_size

number

页面大小，默认值为10，最大不超过100

filtering

object

过滤条件

status

string

分包状态，可选值：

channel_id

string[]

渠道号，可通过渠道号筛选应用分包，单次支持传入渠道号个数<=50

update_time

object

更新时间，可通过最近更新时间筛选应用分包，传入开始时间和结束时间查询该时间范围内更新过的应用分包

start_time

string

开始时间，示例:"2024-07-01 00:00:00"

end_time

string

结束时间，示例:"2024-07-01 23:59:59"
