# 创建订单接口

用于创建DOU+订单(注：此接口仅支持在DOU+完成企业入驻的客户调用且白名单管控，白名单请联系巨量引擎销售开通）

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/douplus/order/create/](https://api.oceanengine.com/open_api/v3.0/douplus/order/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1794677902211227)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# 请求参数

字段

类型

描述

aweme_sec_uid

string

抖音号ID，通过已授权账户接口可以获取

object_infos

object[]

选择投放物料

可批量填多条object，上限最多50条

marketing_goal

string

营销目标(素材类型)

可选值:

aweme_id

string

抖音号

在手机端上看到的抖音号

素材类型为为直播(LIVE_PROM_GOODS)时，必填

item_id

number

抖音视频ID

当素材类型为视频加热和视频加热直播间：传递视频ID；

external_action

string

投放目的

当素材类型为短视频(VIDEO_PROM_GOODS)时，允许值为：

当素材类型为直播(LIVE_PROM_GOODS)时，允许值：

delivery_time

double

投放时长

当marketing_goal= VIDEO_PROM_GOODS时，支持2、6、12、24或12的倍数，最大720小时

当marketing_goal= LIVE_PROM_GOODS &liveroom_heat_mode= BY_VIDEO 时，支持0.5、1、1.5、2、2.5、3、3.5、4、4.5、5、5.5、6、12、24

注：视频投放最短2小时

bid_type

string

出价类型

可选值:

默认值:AUTO_BID

budget

number

投放预算

注意：数值需为10,000-500,000,000之间的10的倍数， 单位：人民币（分）

bid_price

number

手动出价金额

10～1,000,000之间的10的倍数，不高于投放金额，单位：人民币（分） 注意：仅出价类型为MANUAL_BID时，必填

liveroom_heat_mode

string[]

直播间加热方式

可选值:

BY_ROOM: 直接加热直播间

BY_VIDEO: 选择视频加热直播间

注意：仅素材类型为LIVE_PROM_GOODS，必填。可传多值

若传了BY_VIDEO，则必须有素材类型为VIDEO_PROM_GOODS 的object，同理，若传了BY_ROOM，则必须有素材类型为LIVE_PROM_GOODS 的object

targeting

object

定向设置

audience_mode

string

想吸引的观众类型(自25年7月31日起不支持人群包投放！不支持DMP_PKG 这个枚举值了！！)

可选值:

默认值：AUTO

gender

string

性别

允许值：

注意：仅想吸引的观众类型为CUSTOM时，有效

age

string[]

年龄

允许值：若不传值，则默认为不限

AGE_BETWEEN_18_23：18-23岁

AGE_BETWEEN_24_30：24-30岁

AGE_BETWEEN_31_40：31-40岁

AGE_BETWEEN_41_49：41-49岁

注意：仅想吸引的观众类型为CUSTOM时，有效

dmp_pkg_ids

number[]

人群包ID（需从DOU+网页版查询获取）(自25年7月31日起不支持人群包投放！！！)

注意：仅想吸引的观众类型为DMP_PKG时，必填

audience_pkg_id

number

定向包id（需从DOU+网页版查询获取，支持：性别/年龄/地域/兴趣标签/达人相似粉丝/操作系统ios或安卓）

注意：想吸引的观众类型为AUDIENCE_PKG时，必填

rta_id

number
