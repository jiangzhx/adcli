# 获取通用版商品列表

适用场景：

# 请求地址

https://api.oceanengine.com/open_api/v3.0/dpa/ebp/product/list/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1846200485909513)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

# 请求参数

字段

类型

描述

account_id

number

升级版组织id

account_type

string

product_platform_id

number

商品库id，可通过商品库列表接口获得

filtering

object

过滤条件

dpa_categories

number[]

商品行业分类，长度限制 50，使用的三级行业ID ，可从 [【获取行业列表】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710677879808) 接口获取

is_recommend

number

product_id

number

商品ID，精准匹配，与product_name（商品名称）同时传入，以商品ID为准

product_name

string

商品名称，模糊匹配，长度限制50字，与product_id（商品ID）同时传入，以商品ID为准

status

number

商品投放状态，枚举值：0代表不可投放，1代表可投放

audit_status

number

spu_id

string

外部商品id

page

number

页码， 默认值: 1

page_size

number

页面数据量， 默认值: 20，最大 100

注：page*page_size需小于30000
