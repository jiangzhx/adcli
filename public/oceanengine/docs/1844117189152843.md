# 直播间分析报表【新】

直播间分析报表接口用于获取直播间的基础分析数据、互动分析数据

数据范围&口径：

# 请求地址

https://api.oceanengine.com/open_api/v3.0/report/live_room/analysis/get/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1844117189152843)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

# 请求参数

字段

类型

描述

advertiser_id

number

start_time

string

报表开始时间，格式为%Y-%m-%d %H:%M:%S

默认从7天前开始，仅支持天级别的数据查询，请从0点开始查询，否则返回值为空

end_time

string

报表结束时间，格式为%Y-%m-%d %H:%M:%S

默认为今天，最大时间范围为62天

fields

string[]

指标字段

filtering

object

过滤器

room_ids

number[]

直播间ID，最多筛选100个直播间ID

返回数据范围为筛选的直播间下营销和自然流量数据，包含直播间的所有账户数据

若不筛选则按照消耗指标倒序返回消耗top10的抖音号

aweme_ids

number[]

抖音号ID

返回数据范围为筛选的抖音号下全量直播间数据，包括营销和自然流量，包含直播间的所有账户数据，最多支持筛选 10 个抖音号ID

若不筛选则按照消耗指标倒序返回消耗top10的抖音号

直播间ID和抖音号ID须至少筛选其中一个

first_flow_category

string

直播间一级流量来源，可选值:

order_field

string

排序字段

order_type

string

page

number

页码，默认值为1

page_size

number

页面大小，默认值为20，最大值为100
