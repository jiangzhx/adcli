# 创建关键词

在指定的promotion_id下创建搜索关键词，支持在原有关键词基础上进行新增

目前仅支持搜索直投营销

创建关键词时会自动将优词添加为关键词，请您知悉

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/keyword/create/](https://api.oceanengine.com/open_api/v3.0/keyword/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1768010379579407)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

promotion_id

number

待创建搜索关键词的营销ID，仅支持搜索常规投放在单元层级设置关键词

project_id

number

待创建搜索关键词的项目ID，仅支持搜索极速智投在项目层级设置关键词

keywords

object[]

关键词列表，一个营销最多可添加1000个

word

string

关键词。关键词长度不超过30，一个汉字长度计为1，一个英文字符长度计为0.5，不能包含emoji

match_type

string

匹配类型:

bid

float

出价，取值范围：0.2至999.0

当pricing为PRICING_OCPC/PRICING_OCPM时，不支持出价

当pricing为PRICING_CPC时，有效且必填

bid_type

string
