# 获取 DPA 商品库视频模板

根据 商品库 和商品库商品 查询可用的 商品库视频模板

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/video/get/](https://ad.oceanengine.com/open_api/2/dpa/video/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710582303744)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 广告主id |
| filtering 必填 | object | 过滤条件 |
| product_platform_ids 必填 | number[] | 商品库ids，长度限制 50个 |
| product_ids 必填 | number[] | 商品库商品ids，长度限制 50个 |
| start_time | string | 视频创建的筛选起始时间 |
| end_time | string | 视频创建的筛选结束时间 |
| image_mode | string | 素材类型，详见[【附录-枚举值-素材类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)可选值: CREATIVE_IMAGE_MODE_VIDEO：横版视频,CREATIVE_IMAGE_MODE_VIDEO_VERTICAL：竖版视频 |
| package_id | string | 根据视频模板id搜索，精确匹配，长度限制 50字 |
| package_name | string | 根据视频模板名称搜索，精确匹配 长度限制 50字 |
| page | number | 页码， 默认值: 1 |
| page_size | number | 页面数据量 默认值: 20，最大 100 |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| page_info | object | 分页信息 |
| page | number | 当前页数 |
| page_size | number | 页面大小 |
| total_number | number | 总数量 |
| total_page | number | 总分页数 |
| list | object[] | 商品视频模板列表 |
| product_platform_id | number | 商品库id |
| product_id | number | 商品库商品id |
| package_id | string | 商品库视频模板id |
| name | string | 商品库视频模板名称 |
| create_time | string | 创建时间，格式 %Y-%m-%d %H:%M:%S |
| image_info | object | 视频模板封面图信息 |
| height | number | 图片高度 |
| image_id | string | 图片id |
| width | number | 图片宽度 |
| video_info | object | 商品库视频模板视频信息 |
| height | number | 视频高度 |
| width | number | 视频宽度 |
| image_mode | string | 视频类型，详见[【附录-枚举值-素材类型】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710760171535)可选值: CREATIVE_IMAGE_MODE_VIDEO：横版视频,CREATIVE_IMAGE_MODE_VIDEO_VERTICAL：竖版视频 |
| status | string | 视频状态，可选值: AVAILABLE：可用,UNAVAILABLE：不可用 |
| video_id | string | 视频id |
| request_id | string | 请求的日志id，唯一标识一个请求 |
