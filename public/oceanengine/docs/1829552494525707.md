# 创建获取升级版巨量引擎工作台账户任务

支持获取升级版巨量引擎工作台下全量账户列表

数据内容T+1更新，可在次日8点后创建获取更新全量账户关系

# 请求地址

[https://api.oceanengine.com/open_api/2/ebp/advertiser/task/create/](https://api.oceanengine.com/open_api/2/ebp/advertiser/task/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1829552494525707)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
