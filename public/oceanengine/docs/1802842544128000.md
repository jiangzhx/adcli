# 批量获取项目成本保障状态

本接口支持批量查询项目的成本保障状态及相关赔付信息（接口能力仅对齐巨量营销升级版），对应平台能力如下图所示

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/project/cost_protect_status/get/](https://api.oceanengine.com/open_api/v3.0/project/cost_protect_status/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1802842544128000)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

巨量营销平台客户id

project_ids

number[]

项目id列表，每次最多传入50个

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

message

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

compensate_status_info_list

object[]

项目成本保障信息列表

project_id

number

项目id

compensate_status

string

项目成本保障状态 可选值:

其他异常情况返回：

compensate_invalid_reasons

string[]

成本保障失效原因，仅当compensate_status = INVALID 时&&仅在以下情况会返回。原因可能有多个，因此可能会返回多个枚举

compensate_end_reasons

string[]

成本保障结束原因，仅当compensate_status = ENDED 时&&仅在以下情况会返回

原因可能有多个，因此可能会返回多个枚举

compensate_amount

double

赔付金额（元），最多2位小数

compensate_url

string

赔付规则链接，主要为飞书文档地址，如：[https://bytedance.larkoffice.com/docx/UWifd88wfou3bmxHAq7ciOgPn9e](https://bytedance.larkoffice.com/docx/UWifd88wfou3bmxHAq7ciOgPn9e)

request_id

string

请求日志id
