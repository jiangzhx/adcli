# 接入流程

## 步骤1：创建数据库

巨量开发者登录开放平台后点击[https://ad.oceanengine.com/cloud/volc/mysql](https://ad.oceanengine.com/cloud/volc/mysql)跳转连山控制台按页面提示创建MySQL

注意：数据库和私有网络务必选择北京机房，否则数据无法同步

磁盘空间预估：1W条有消耗创意，每周约占用1.2G左右的空间，每天平约落库0.17G的报表数据

## 步骤2：订阅RDS任务

点击[https://open.oceanengine.com/developer/admin/rds/#/rds](https://open.oceanengine.com/developer/admin/rds/#/rds) 按照页面提示完成RDS订阅

## 步骤3：访问数据

方式1：开启数据库公网访问

点击[https://ad.oceanengine.com/cloud/volc/mysql](https://ad.oceanengine.com/cloud/volc/mysql) 点击选中实例

方式2：数据库迁移（DTS）

点击[https://ad.oceanengine.com/cloud/volc/dts](https://ad.oceanengine.com/cloud/volc/dts) 申请公测后配置即可，支持迁移公网MySQL与ES

## 步骤4：数据查询

查询指定计划8月29日累计展示量SQL示例

```
xxxxxxxxxx
SELECT advertiser_id, ad_id, sum(`show`), count(`show`)
FROM  lianshan_realtime_202208XX
WHERE advertiser_id=17424770XXXXXXXX and  stat_time >= '2022-08-29 00:00:00' AND stat_time < '2022-08-30 00:00:00'
GROUP BY advertiser_id, ad_id
```
