# 批量更新项目预算

批量更新 巨量引擎体验版 项目预算

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/project/budget/update/](https://api.oceanengine.com/open_api/v3.0/project/budget/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1755353873798155)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST Request-Body

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| data 必填 | object[] | 批量修改预算，list长度限制1～10 |
| project_id 必填 | number | 项目ID |
| budget_mode 必填 | string | 允许值: BUDGET_MODE_DAY日预算 BUDGET_MODE_INFINITE不限 |
| budget | float | 项目预算，当budget_mode = BUDGET_MODE_DAY时有效且必填 填写限制： 当external_action=AD_CONVERT_TYPE_CLICK_NUM 点击量 或 AD_CONVERT_TYPE_SHOW_OFF_NUM 展示量时，取值范围: ≥ 100 当delivery_type =DURATION搜索周期投放时，取值范围为[2000, 9999999.99]，单位为元 当 bid_type = NO_BID最大转化时，取值范围为[100,999999.99] 当budget_optimize_switch = ON 开启预算择优分配时，取值范围为[300,999999.99] |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| project_ids | number[] | 更新成功的营销项目ID集合 |
| errors | object[] | 更新失败的营销项目列表 |
| project_id | number | 项目ID |
| error_message | string | 错误信息 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
