# 获取原生锚点列表

获取账户下的原生锚点列表，不同锚点的详情信息请前往「[获取原生锚点详情](https://open.oceanengine.com/labels/7/docs/1786407342576640)」查询

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/native_anchor/get/](https://api.oceanengine.com/open_api/v3.0/native_anchor/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1757715831323652)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户ID

page

number

页码，默认值: 1

page_size

number

页面数据量，默认值: 10，page_size范围为[1,100]

filtering

object

过滤器

anchor_id

string

按原生锚点id 做过滤

anchor_name

string

按原生锚点名称做过滤

status

string[]

锚点审核状态，允许值：

锚点新建：CREATE

审核通过：AUDIT_SUCCESS

审核不通过：AUDIT_FAILD

landing_type

string

营销目的，允许值：

LINK销售线索收集

APP应用

AWEME原生互动

external_action

string

优化目标，可通过[【获取优化目标】](https://open.oceanengine.com/labels/7/docs/1740944984250381)接口获取

source

string[]

锚点来源，允许值：

自动生成：AUTO

手动创建：MANUAL、CUSTOM

组织锚点：GROUP

anchor_type

string[]

锚点类型，允许值：

游戏：APP_GAME

网服：APP_INTERNET_SERVICE

电商：APP_SHOP

高级在线预约：ONLINE_SUBSCRIBE

字节小程序：MICRO_APP

字节小游戏：MICRO_GAME

购物车、SHOPPING_CART

外跳：INSURANCE

咨询：PRIVATE_CHAT

anchor_start_time

string

锚点创建开始日期，格式：yyyy-MM-dd

anchor_end_time

string

锚点创建结束日期，格式：yyyy-MM-dd

android_package_name

string

安卓应用包名

ios_package_name

string

ios应用包名
