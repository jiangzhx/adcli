# 查询在投计划配额

该接口用于查询投放账户的在投计划配额和使用进度。 详细说明，见[在投计划配额介绍文档](https://bytedance.feishu.cn/docx/doxcnIgDvxYHbTcGCU1xu2KD6LO)，也可在文档中通过评论提问。

### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/quota/get/](https://ad.oceanengine.com/open_api/2/tools/quota/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1731070251873293)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 投放账户id |
| campaign_type必填 | string | 计划组类型，FEED：信息流 SEARCH:搜索 |
| delivery_range | string | 可选值: DEFAULT: 默认 UNION: 穿山甲 当campaign_type = feed && delivery_range= default可获取所有营销配额; 当campaign_type = feed && delivery_range= union时可获取穿山甲营销配额; 当campaign_type = search && delivery_range= default时可获取搜索配额; |

### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| used_quota | number | 在投计划数 |
| sum_quota | number | 在投计划配额 |
| in_learning | bool | 是否在学习期，学习期即初始期，详细规则请进入巨量营销投放平台-工具-在投计划配额界面查看 |
| max_cost | number | 最高月消耗 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
