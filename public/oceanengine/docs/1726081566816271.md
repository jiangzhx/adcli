# 创建DPA商品（已有商品id）/修改DPA商品

【重要】

此接口适用于

若希望满足以下场景：

请求参数中必须填写商品库ID和商品ID

### 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/product/update/](https://ad.oceanengine.com/open_api/2/dpa/product/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1726081566816271)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：application/json |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://ad.oceanengine.com/openapi/doc/index.html?id=16967105812336766710505596940) |

### 请求参数

#### 公共参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| platform_id 必填 | number | 商品库ID |
| product_id 必填 | number | 商品ID |
| product_info 必填 | object | 商品详情，包括商品库基础字段、落地页、品牌、商户、价格以及其他字段信息，以下所有字段均包含在该结构体下，且不同类型商品库填入字段存在差异，具体格式见下方字段详情 |
