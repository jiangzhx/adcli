# 创意组件

## 概念介绍

创意组件：单元在客户端端展示时的附加内容，有更为创新的形态和表达方式。

## 流程简述

创建创意组件

通过调用 [【创建创意组件】](https://open.oceanengine.com/labels/7/docs/1696710672391183) 接口，可以创建创意组件，在创编流程中使用

查询创意组件

对于创建完成的创意组件，可以通过[【查询创意组件】](https://open.oceanengine.com/labels/7/docs/1696710673645580)进行信息查询，查询到的组件信息

更新创意组件

如果对于创建的创意组件进行信息变更，可以通过[【更新创意组件】](https://open.oceanengine.com/labels/7/docs/1696710673083407)接口进行修改

创建单元和创意

组件创建完成之后才可以在创建计划，创建创意的时候使用
