# 查询快应用信息

查询当前客户的快应用

# 请求地址

[https://api.oceanengine.com/open_api/2/tool/quick_app_management/quick_app/get/](https://api.oceanengine.com/open_api/2/tool/quick_app_management/quick_app/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1758407389420547)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

status

string[]

应用状态，允许值：

AUDIT_DOING 审核中、AUDIT_SEND_REJECTED 送审失败 、AUDIT_REJECTED 审核失败、AUDIT_ACCEPTED 审核成功、REMOVED 已下架

page

number

页码，默认值为1

page_size

number

页面大小，默认值为10，最大不超过100

update_time

object

修改时间

start_time

number

时间戳（秒级）

end_time

number

时间戳（秒级）

search_key

string

搜索关键字 快应用名称或者包名的模糊搜索，可以为空，可以传中文

quick_app_ids

number[]

快应用ids
