# 请求地址

[https://api.oceanengine.com/open_api/2/carousel/ad/get/](https://api.oceanengine.com/open_api/2/carousel/ad/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773560989475840)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
