# 推送人群包

每个人群包生成一个人群包id后，都需要经过推送，才可以在被推送的客户下使用。同时，推送人群包可以将人群包共享给同主体的客户。

您可以通过[【人群包详细信息】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710571259916)查询人群包的状态，其中返回参数delivery_status表征当前人群包可投放状态（包含是否推送完成）；

### 请求地址

[https://ad.oceanengine.com/open_api/2/dmp/custom_audience/push_v2/](https://ad.oceanengine.com/open_api/2/dmp/custom_audience/push_v2/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710572311552)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：application/json |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 人群包所属客户ID |
| custom_audience_id 必填 | number | 人群包ID |
| target_advertiser_ids 必填 | number[] | 推送客户ID列表，最多推送100个客户 |
