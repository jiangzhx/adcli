# 获取已授权账户

此接口用于获取AccessToken下已经授权的账户列表，获取到的账户可根据账户类型结合其他接口文档，确认其他接口支持账户类型范围。

# 请求地址

[https://api.oceanengine.com/open_api/oauth2/advertiser/get/](https://api.oceanengine.com/open_api/oauth2/advertiser/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710506574848)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# 请求参数

字段

类型

描述

access_token

string

根据授权auth_code获取生成的AccessToken

授权页面使用相同账号授权对应同一个AccessToken，如使用多个不同的账号授权，则需要区分维护多个不同的AccessToken
