# 核销券码

核销券码接口

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/coupon/code/consume/](https://ad.oceanengine.com/open_api/2/clue/coupon/code/consume/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710641144847)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: "application/json" |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| code_id | string | 券码ID，c端获取 |
| code | string | 券码code，c端获取code_id和code至少设置一个，两个都设置code_id优先 当设置code时，必须设置activity_id、coupon_id和resource_id中的一个 |
| activity_id | number | 线索通活动ID，可从查询券码记录接口获取 |
| coupon_id | number | 卡券ID，可从查询券码记录接口获取 |
| resource_id | number | 优惠券ID，可从查询券码记录接口获取 |
| employee 必填 | object | 核销员信息 |
| user_id 必填 | number | 用户ID |
| user_type 必填 | string | 用户类型 TOUTIAO:头条用户 DOUYIN:抖音用户 |
| store_id | number | 门店ID，不填或0代表不限制门店 |
| extra | object | 核销补充信息，即自定义信息 |
