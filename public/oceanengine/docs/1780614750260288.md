# 字节&微信小程序/小游戏共享管理

巨量营销Marketing API支持通过API完成字节、微信小程序/小游戏资产共享，支持的能力包括：

| 能力简述 | 说明 | 接口文档 |
| --- | --- | --- |
| 发起小游戏/小程序共享 | 支持将字节、微信小游戏/小程序资产从巨量引擎工作台组织共享到AD、其他组织账户下 | [https://open.oceanengine.com/labels/34/docs/1773089427219584](https://open.oceanengine.com/labels/34/docs/1773089427219584) |
| 取消小游戏/小程序共享 | 支持从巨量引擎工作台账户取消资产共享 | [https://open.oceanengine.com/labels/34/docs/1773102677218368](https://open.oceanengine.com/labels/34/docs/1773102677218368) |
| 查询共享关系 | 支持指定资产类型，查询巨量引擎工作台下字节、微信小程序/小游戏资产的共享范围 | [https://open.oceanengine.com/labels/34/docs/1773105971421187](https://open.oceanengine.com/labels/34/docs/1773105971421187) |
