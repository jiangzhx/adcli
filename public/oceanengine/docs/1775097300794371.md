# 投放账户复制

投放账户复制接口，支持复制原账户资质信息

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/advertiser/copy/](https://api.oceanengine.com/open_api/2/agent/advertiser/copy/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1775097300794371)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

被复制投放账户ID

agent_id

number

代理商账户ID

item

object[]

复制账户信息，一次最多复制出10个新账户

name

string

投放账户名称。

注意：1. 名称不可重复 2. 名称不能含有换行符或特殊字符

copy_tag

bool

是否复制原投放账户的账户标签

report_type

string

自运营报备类型 可选值:

默认值: EMPTY

copy_delivery

bool

true：复制

false：不复制
