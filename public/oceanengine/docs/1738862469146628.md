# 事件管理资产取消共享

事件管理资产取消共享

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/event_manager/share/cancel/](https://api.oceanengine.com/open_api/v3.0/event_manager/share/cancel/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1738862469146628)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| organization_id必填 | number | 纵横组织id |
| asset_id必填 | number | 事件资产id |
| share_mode必填 | string | 取消共享模式，可选值: ALL: 组织下某业务线账户PART: 指定账户共享 |
| account_infos条件必填 | object[] | 取消共享账户信息，一次最多操作10个 当share_mode="PART"时必填 |
| account_id | number | 账户id |
| account_type | string | 账户类型，可选值: AD |
| all_account_type条件必填 | string[] | 取消共享账户业务线 当share_mode="ALL"时必填，可选值：: AD |
