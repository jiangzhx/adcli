# 上传/更新营销产品资质

用于提交以营销产品形式整组提交的投放资质，该接口可以同时用于新增和编辑（针对审核不通过的营销产品资质支持编辑提交）

# 逻辑说明

业务逻辑总体介绍：根据投放内容所在的不同行业需要一组完整的、相关联的资质才可以形成证明效力。调用API之前，您需要首先查询【[营销产品资质规则配置查询](https://open.oceanengine.com/labels/34/docs/1776710970860551)】接口获取所选行业投放产品资质的配置规则（拿到config_id)

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_pkg/submit/](https://api.oceanengine.com/open_api/v3.0/advertiser/delivery_pkg/submit/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1776711210685572)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)
