# 查询代理商转账记录

代理商转账记录查询，相关功能与代理商平台的「商务-转账记录-账户转账记录」模块对齐。

# 请求地址

[https://api.oceanengine.com/open_api/2/agent/transfer/transaction_record/](https://api.oceanengine.com/open_api/2/agent/transfer/transaction_record/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1795124749017235)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取
