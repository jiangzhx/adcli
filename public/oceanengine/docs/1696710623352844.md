# 建站工具-建站样式复制（跨账户）

通过此接口，用户可以实现站点的转赠功能，将某一客户的站点复制给其他特定的客户，转赠成功后，被转增的投放账户下会新增一个站点id（内容同原站点），但会清空原站点下的资产信息。 不限制主体，同客户不能进行转赠操作

用户在传入请求参数site_ids站点id列表时，请确保id的正确性，存在错误将导致转赠全部失败！

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/site/handsel/](https://ad.oceanengine.com/open_api/2/tools/site/handsel/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710623352844)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户id，范围：1 <= advertiser_id <= MAX_INT64 |
| site_ids必填 | string[] | 站点id列表, min_size=1, max_size=20 |
| target_advertiser_ids必填 | string[] | 目标客户id列表, min_size=1, max_size=20 |

##### 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | json | json返回值 |
| success_list | object | 转赠成功列表，整体失败不返回该列表 |
| origin_site_id | string | 返回转赠成功的原site_ids |
| target_advertiser_id | string | 返回转赠成功的目标客户id |
| site_id | string | 转赠成功后的生成的新站点id，失败的数据无此参数返回 |
| error_list | object | 转赠失败列表，整体成功不返回该列表 |
| origin_site_id | string | 返回转赠失败的原site_ids |
| target_advertiser_id | string | 返回转赠失败的目标客户id |
| error_reason | string | 失败原因，当对origin_site_id的操作失败时，返回error_reason字段，成功无 |
| request_id | string | 请求日志id |
