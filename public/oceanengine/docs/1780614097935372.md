# 更新字节小程序

请注意：两种方式仅生效一种。当以上三个参数均有值时，将按照方式a.生效逻辑。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/micro_app/update/](https://api.oceanengine.com/open_api/v3.0/tools/micro_app/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1780614097935372)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

POST

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户id

instance_id

number

资产id

remark

string

小程序备注名称支持更新，20字以内

app_page

object[]

小程序链接，创建链接或更新链接时，小程序调起链接存在两种录入方式：

id

number

链接id，通过【[小程序详情接口](https://open.oceanengine.com/labels/8/docs/1778265753535620)】接口获取。当操作类型为 MODIFY和DELETE时，链接id必填

link

string

小程序链接，以 sslocal://microapp 开头 。生效优先级大于 启动页面 和 启动参数 ，详细内容见接口说明

remark

string

链接备注

operate_type

string

操作类型，可选值:

start_page

string

启动页面，形如：page/home/xxx。不得以反斜线开头，不得以反斜线结尾，请避免参数中出现特殊字符

start_param

string

启动参数，形如：name=zhangsan&age=25。不得以问号、连接符等特殊字符开头，请确保各参数间仅使用一个连接符

tag_info

string

细分类型或道具类目

填写类型的末级分类ID，详见[【应用管理标签及类目】](https://open.oceanengine.com/labels/7/docs/1812331394774028?origin=left_nav)

payment_form

string

可选值:

membership_type

string

可选值:

revenue_model

string

可选值:

has_online_earning

bool

max_recharge_tier

string

可选值:

min_recharge_tier

string

recommended_recharge_tier

string

prop_name

string

道具名称

min_payment_tier_range

string

最低付费档位区间（元）

mid_payment_tier_range

string

中间付费档位区间（元）

max_payment_tier_range

string

最高付费档位区间（元）

has_discount

bool

discount_rate

number
