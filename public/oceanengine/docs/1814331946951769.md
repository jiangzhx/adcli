# 【新版】创建通用版商品（无商品id）

此接口适用于

若希望满足以下场景：

需要根据不同的商品库类型入参对应的商品详情，包括商品库基础字段、落地页、品牌、商户、价格以及其他字段信息，以下所有字段均包含在【product_info】结构体下，需要注意不同类型商品库的入参差异

# 请求地址

[https://ad.oceanengine.com/open_api/2/dpa/product/create/](https://ad.oceanengine.com/open_api/2/dpa/product/create/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1814331946951769)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type 必填 | string | 请求消息类型，允许值：application/json |
| Access-Token 必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://ad.oceanengine.com/openapi/doc/index.html?id=16967105812336766710505596940) |

### 请求参数

#### 公共参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| platform_id 必填 | number | 商品库ID |
| product_info 必填 | object | 商品详情，包括商品库基础字段、落地页、品牌、商户、价格以及其他字段信息，以下所有字段均包含在该结构体下，且不同类型商品库填入字段存在差异，具体格式见下方字段详情 |
