# 查看2.0rit数据

查看穿山甲2.0投放版位数据

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/union/flow_package/promotion/report/](https://api.oceanengine.com/open_api/v3.0/tools/union/flow_package/promotion/report/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1775179912274944)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

filter

object

筛选条件

promotion_ids

number[]

营销 id 列表，最大数量限制：1000

rits

number[]

投放版位列表，最大数量限制：1000

landing_type

string

营销目的，允许值：APP 应用、LINK 销售线索投放、MICRO_GAME 小程序、SHOP 电商、QUICK_APP快应用、DPA商品目录 可选值:

high_cost

string

消耗金额上限，单位元

low_cost

string

消耗金额下限，单位元

start_time

string

开始时间，格式为"yyyy-mm-dd" 限制范围100天内 默认7天前（不包括当天），即不指定起止时间获取最近7天数据

end_time

string

结束时间，格式为"yyyy-mm-dd" 默认昨天，即不指定起止时间获取最近7天数据

order_field

string

排序字段，所有的统计指标均可参与排序 默认按rit排序

order_type

string

排序方式 枚举值：ASC（升序），DESC（降序） 默认：DESC 可选值:

page

number

页数 默认值: 1

page_size

number

页面大小,限制范围[1, 100] 默认值: 10
