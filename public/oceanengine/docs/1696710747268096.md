# RTA策略管理

相关链接：

本篇主要介绍RTA策略，该功能为白名单功能，如需开通请联系对应销售同学。目前支持获取RTA策略数据、获取可用的RTA策略、批量启停账户下RTA策略、设置账户下RTA策略生效范围、获取穿山甲渠道RTA联合实验数据、获取站内媒体RTA联合实验数据（分时t+5）、获取站内媒体RTA联合实验数据（分天t+1）七个接口。
