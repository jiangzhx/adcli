# 取消商品库&商品共享

使用场景：支持针对通用版商品库维度或升级版商品维度已有共享记录操作取消，请仔细核对需要取消的共享关系，取消成功后对应账户将不可使用对应的商品创建增量的计划及关联组件。

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/cancel/](https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/cancel/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1834829364474954)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

organization_id

number

普通版巨量引擎旧版巨量引擎工作台账户id

asset_type

string

泛商品资产类型，可选值:

instance_id

number

泛商品资产id，通用版商品库请传入商品库id，升级版商品请传入商品id。

share_mode

string

取消共享类型，为当前泛商品资产已有的共享类型，可选值:

account_infos

object[]

当取消共享类型为指定id共享，需要传入期望被取消的账户，一次最多取消共享100个账号

account_id

number

需要取消共享的投放账户

account_type

string

需要取消共享的投放账户业务线，可选值:

all_accounts_by_bp

string[]

取消旧版巨量引擎工作台下所有账户及新增账户的业务线，可选值:

all_accounts_by_company

object[]

account_type

string

可选值:

company_id

number
