# 删除升级版商品

【使用场景】

【支持的删除方式】

| 删除模式 | 支持类目 | 类目id（category_id） |
| --- | --- | --- |
| 支持按照店铺ID+外部商品ID删除 | 电商店铺商品 | 140000000 |
| 按照product_id指定删除 | 除电商店铺商品外的商品 截止12.9为活动、房产、招商加盟、教育培训、医疗服务、人资代招、岗位招聘 | 250000000 130000000 240000000 40000000 50900000 110100000 120000000 |

# 请求地址

https://api.oceanengine.com/open_api/v3.0/dpa/ebp/clue_product/delete/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1846200682152971)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)

string

string
