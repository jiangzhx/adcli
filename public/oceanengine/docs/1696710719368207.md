# 删除动态创意词包

使用此接口可以删除已创建的动态创意词包。

##### 请求地址

[https://ad.oceanengine.com/open_api/2/tools/creative_word/delete/](https://ad.oceanengine.com/open_api/2/tools/creative_word/delete/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710719368207)

##### 请求方法

POST

##### 请求Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Content-Type必填 | string | 请求消息类型，允许值：application/json |
| Access-Token必填 | string | 授权access-token，获取方法见接口文档[【获取Access-Token】](https://open.oceanengine.com/labels/7/docs/1696710505596940) |

##### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id必填 | number | 客户ID |
| creative_word_id必填 | number | 创意词包id |
