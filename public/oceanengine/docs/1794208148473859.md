# 批量修改项目ROI系数

本接口支持批量修改项目ROI系数，当前仅以下项目支持ROI系数修改：

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/project/roigoal/update/](https://api.oceanengine.com/open_api/v3.0/project/roigoal/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1794208148473859)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户id

data

object[]

批量修改项目ROI系数，list长度限制10

project_id

number

项目ID

roi_goal

float

深度转化ROI系数，填写要求如下：

shop_multi_roi_goals

object[]

电商平台多ROI系数设置，引流电商多平台投放ROI系数及平台信息，可按照电商平台分别确定ROI系数，分平台调控出价，此能力白名单开放。list长度最长为4

填写条件：仅当满足以下5个条件时可传入，此时无需传入roi_goal参数：

填写要求：

roi_goal

double

ROI系数值，范围[0.01,100]，精度：最多保留小数点后四位

shop_platform

string
