# 更新项目

[https://api.oceanengine.com/open_api/v3.0/project/update/](https://api.oceanengine.com/open_api/v3.0/project/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1740936504522831)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求参数

## 基础参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 投放账户id |
| project_id 必填 | number | 项目id |
| name | string | 项目名称，长度是1-50个字（两个英文字符占1个字） |
| aigc_dynamic_creative_switch | string | 是否开启AIGC动态创意，系统将根据创意生态要求生成素材探索，不传表示不开启，枚举值： ON 开启 OFF 关闭 注意： 该功能仅支持行业白名单客户使用，如需使用可咨询对接销售/运营 必须关联产品投放，才可设置该开关（即必须传入related_product结构体） 如投放短剧商品，需要开启AIGC动态创意，请确保： 关联短剧商品内包含字节小程序投放链接 短剧剧目已获取原片使用权，可通过「[查询短剧商品原片授权申请状态](https://open.oceanengine.com/labels/7/docs/1805903150040128)」接口查询授权申请状态 |
| star_auto_delivery_switch | string | 星广联投全自动化开关，支持编辑 暂仅支持短剧版，枚举值： ON 开启 OFF 关闭 注意： 开启星广联投全自动化，需同时满足以下条件 账户属于白名单（Key: star_v3_auto_task，调用[白名单查询](https://open.oceanengine.com/labels/7/docs/1763407285579840)接口时version请传入OLD_VERSION），加白请联系对接销售 landing_type = MICRO_GAME 小程序 delivery_mode = PROCEDURAL 自动投放 开启全自动化支持的营销产品+投放载体包含以下2种情况 情况1：营销产品=字节小游戏/微信小游戏，即micro_promotion_type = BYTE_GAME/WECHAT_GAME 情况2：营销产品=商品、投放载体=字节小程序/微信小程序，即必须添加商品related_product、micro_promotion_type =BYTE_APP / WECHAT_APP 开启全自动化开关时，必须注意其他传参要求： 不支持指定星广任务id（即传入star_task_id会报错)。启用全自动化代表系统将为您在项目下自动关联营销产品对应的任务，无需手动选择 不支持启用自动素材开关（即传入star_auto_delivery_switch会报错）。启用全自动化已经意味着系统将自动为您在单元下添加任务下的素材，您无需再设置自动素材 ⚠️更新项目为增量更新接口，不传star_auto_delivery_switch参数 表示不修改此项设置 存量的星广联投短剧版项目（指项目中已存在star_task_id），不支持打开全自动化开关，传入会报错 |
| star_auto_material_addition_switch | string | 星广联投自动优化素材开关，支持编辑。枚举值： ON 开启 OFF 关闭 注意：开启自动优选素材开关的项目下只有投放「全部抖音号且没有排除抖音号」的单元才生效自动优化。 |

## 搜索快投

可通过【[获取关键词推荐](https://open.oceanengine.com/labels/7/docs/1755356193295372)】获取系统推荐的搜索快投关键词

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| search_bid_ratio 条件必填 | float | 出价系数，默认系数为1，出价系数可通过【获取快投推荐出价系数】查询，小数点后最多两位,取值范围 [1.00,2.00] 当符合以下所有条件时填写有效 1. bid_type != NO_BID&&PRICING = OCPM 2. deep_bid_type = DEEP_BID_DEFAULT 无深度优化方式 /BID_PER_ACTION 每次付费/DEEP_BID_PACING 自动优化/ROI_COEFFICIENTROI系数/FORM_BID优选表单 |
| audience_extend | string | 定向拓展，允许值：ON:开启，OFF:关闭 |

## 搜索关键词

可通过【[获取关键词推荐](https://open.oceanengine.com/labels/7/docs/1755356193295372)】获取系统推荐的搜索关键词

仅通投链路下使用搜索快投关键词及搜索极速智投链路下自定义关键词时支持传入使用keywords参数设置

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| keywords条件必填 | object[] | 待添加搜索关键词列表 当ad_type = ALL时，支持传入关键词为搜索快投关键词 最多可添加1000个 当ad_type = SEARCH & delivery_type = DURATION 极速智投场景下时，支持传入关键词为自定义搜索关键词 客户拥有search_merge_auto_blue白名单时支持 最多可添加3000个 注：不传则表示不更新，传[]则表示删除全部关键词 |
| word 条件必填 | string | 关键词,关键词长度不超过30，一个汉字长度计为1，一个英文字符长度计为0.5，不能包含emoji |
| bid_type 条件必填 | string | 出价类型，允许值： 搜索快投场景 FEED_TO_SEARCH 搜索快投（默认值） 极速智投场景 CUSTOM 自定义出价，仅pricing出价方式设置为非O类场景时支持且默认 WITH_PROMOTION 随项目/单元出价（仅pricing出价方式设置为O类场景时支持且默认） |
| match_type 条件必填 | string | 匹配类型。 允许值： PHRASE 短语匹配 EXTENSIVE 广泛匹配 PRECISION 精准匹配 |
| bid 条件必填 | float | 出价，取值范围：0.2至999.0 当keywords.bid_type = CUSTOM 时有效且必填 |
| auto_extend_traffic | string | 智能拓流 ，允许值： 当ad_type = SEARCH 且 delivery_type = DURATION 时有效且必填 ON 开启 OFF 关闭 当仅使用自定义关键词（未设置蓝海关键词），且关键词不为空时，支持设置为OFF 当external_action=AD_CONVERT_TYPE_CLICK_NUM 点击量时，仅支持设置为OFF |

| products | object[] | 产品ID列表，当传入related_product结构体时，需设置products 若为UBP多品项目，传入个数限制为[2,30] 若为UBL项目，传入个数限制为[1,10] 【注意】 产品ID列表的更新应为全量更新，即更新接口传入的products覆盖项目下原本传入的所有产品ID 列表传空视为不做更新，维持原项目关联产品列表 |

| open_url | string | Deeplink直达链接 注意 【即将迁移至单元层级表达】landing_type = APP 且子目标app_promotion_type为 LAUNCH 应用调起时必填，其中当launch_type=DIRECT_OPEN 直接调起时，open_url和ulink_url二选一填写 注意2024年12月11日起，应用-应用调起场景下，直达链接、备用直达链接将必须在单元层级设置，项目层级传入会报错 2024年11月12日后，应用-应用调起场景下，如果项目在创建时没有添加直达链接、备用直达链接，不允许在编辑项目时新增链接 |
| ulink_url_type | string | 设置条件：同参数ulink_url 枚举值：UNIVERSAL_LINK（默认值）、APPLINK ulink直达链接备用链接类型 |
| ulink_url | string | ulink直达链接，Universal link为iOS系统建议使用的跨APP跳转方式链接，可以为用户提供更流畅的跳转体验，仅支持非穿山甲投放版位，请确保链接可跳转 仅支持穿山甲投放版位 针对鸿蒙应用投放，ulink_url支持设置Universal link 或 Applink 链接两种链接类型 【即将迁移至单元层级表达】landing_type = APP 且子目标app_promotion_type为 LAUNCH 应用调起时必填，其中当launch_type=DIRECT_OPEN 直接调起时，open_url和ulink_url二选一填写 注意2024年12月11日起，应用-应用调起场景下，直达链接、备用直达链接将必须在单元层级设置，项目层级传入会报错 2024年11月12日后，应用-应用调起场景下，如果项目在创建时没有添加直达链接、备用直达链接，不允许在编辑项目时新增链接 |

## 用户定向

### DPA商品定向（当landing_type=DPA时投放目标参数）

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| audience | object | 定向设置（商品定向字段与人群定向字段目前均为audience字段下的二级字段，设置商品定向时请同时传入人群定向相关字段） |
| dpa_city | string | 地域匹配-商品所在城市开启时，仅将商品投放给位于该商品设置的可投城市的用户默认值：OFF允许值：OFF，ON（OFF表示不启用，ON表示启用） DPA营销目的下有效 当产品库内未上传定向地域信息时，传入无效 |
| dpa_lbs | string | 地域匹配-适地性服务开启时，根据用户的地理位置信息，给用户投放位于其附近的产品 允许值：OFF，ON（OFF表示不启用，ON表示启用） 默认值：OFF DPA营销目的下有效 当产品库内未上传定向地域信息时，传入无效 |
| dpa_categories | number[] | 商品投放范围，分类列表，由[【DPA商品营销-获取DPA分类】](https://open.oceanengine.com/labels/7/docs/1696710580168716) 得到 个数限制 [0, 1000] 不传和传空数组即为不限商品投放范围 DPA营销目的下有效 ad_type =SEARCH搜索，landing_type = DPA 商品不生效 |
| dpa_product_target | object[] | 自定义筛选条件（商品投放条件）。用于圈定商品投放范围，结合商品库字段搭配判断条件，圈定商品投放范围。[获取商品库元信息-商品营销-商业开放平台](https://open.oceanengine.com/labels/7/docs/1696710577566735) 数组长度限制：最大5条 DPA营销目的下有效 仅支持品牌名称、现价、标签和星级 ad_type =SEARCH搜索，landing_type = DPA 商品不生效 |
| title | string | 筛选字段 |
| rule | string | 定向规则，允许值：type 为int、 double、 long其中一种时支持=、 !=、 >、 <； type = string时支持=、 !=、 notEmpty |
| type | string | 字段类型，允许值：int、 double、 long、 string |
| value | string | 规则值 |
