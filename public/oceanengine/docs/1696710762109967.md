# 过滤条件

#### 过滤条件

#### 示例

```
xxxxxxxxxx
[
{
"field": "id",
"operator": "EQUAL",
"value": 123
},
{
"field": "name",
"operator": "CONTAIN",
"value": "test"
}
]
```

#### 操作符列表

| 值 | 描述 |
| --- | --- |
| EQUAL | 等于（=） |
| CONTAIN | 包含（like） |
| LESS_EQUAL | 小于等于（<=） |
| LESS | 小于（<） |
| GREATER_EQUAL | 大于等于（>=） |
| GREATER | 大于（>） |
