# 事件管理

# 概念介绍

## 事件与优化目标：

事件：关键事件是业务中实际发生的事件，一般选取用户行为漏斗中的关键事件。 转化目标：以某个关键事件为转化目标。（未来会以两个事件为双目标，或者目前多转化是以多个事件为目标）

## 事件价值、出价、成本：

事件价值：为事件定义货币价值。事件价值=经营总收入/事件总额，可以更好的衡量您的营销效果。 出价：为获取一个转化进行的出价，可能比价值高，也可能比价值低。 成本：过往为获得转化产生的总消耗/总转化量，或除以总事件量。成本一般要控制低于出价，成本低于事件价值会为客户带来利润。

# 流程简述

## 创建资产

通过调用【创建资产】接口，创建投放账户下资产。目前开放平台仅支持创建自研落地页资产，其他资产（快应用、小程序）需要在营销投放平台手动创建。

## 创建事件

STEP1：获取资产

STEP2：查询资产下支持创建的事件

STEP3：创建事件

## 获取资产下已创建事件

获取资产信息之后，通过【获取已创建事件】接口，查询某一资产下已经创建的事件及其相关配置。

## 获取投放内容

通过调用 【获取投放内容】 接口，获取投放账户下的所有资产信息。

## 获取优化目标

获取资产信息之后，通过【获取优化目标】接口，通过资产信息查询资产绑定的优化目标
