# 设置应用共享

设置应用共享，可通过该接口将应用共享给相关巨量引擎工作台组织或指定账户

该接口为增量共享

# 请求地址

[https://ad.oceanengine.com/open_api/2/tools/app_management/bp_share/](https://ad.oceanengine.com/open_api/2/tools/app_management/bp_share/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1710413925222415)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type必填 | string | 请求消息类型 允许值: application/json |

# 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| organization_id | number | 巨量引擎工作台id |
| package_id | string | 应用包ID， 可通过[「查询安卓应用信息」](https://open.oceanengine.com/labels/7/docs/1696710742760448)接口查询获取package_id |
| share_mode | string | 共享类型 允许值： PART 指定账户共享 ALL 巨量引擎工作台内某类型所有账户共享 COMPANY 公司主体内某类型所有账户共享 |
| all_accounts条件必填 | object[] | 共享账号类别，一次最多操作10个 当share_mode 为ALL时可用且必填 |
| account_type必填 | string | 业务线 枚举值：AD 营销、STAR 星图 |
| account_infos条件必填 | object[] | 共享账号信息，一次最多操作10个 当share_mode为 PART时可用且必填 |
| account_type必填 | string | 业务线 枚举值：AD 营销、STAR 星图、BP 巨量引擎工作台 |
| account_id必填 | number | 账号id |
| all_account_by_company条件必填 | object[] | 公司主体信息，一次最多操作1个 当share_mode为 COMPANY时可用且必填 |
| account_type必填 | string | 业务线 枚举值：AD 营销、STAR 星图 |
| company_id必填 | number[] | 公司主体id, 一次最多操作10个 |

# 应答字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| success_list | object[] | 成功共享列表 |
| share_mode | string | 共享类型 允许值：PART 指定账户共享、ALL 巨量引擎工作台内某类账户所有共享 共享类型为PART，读取account_info 共享类型为ALL，读取all_account，即表示该业务线下的全量账号 |
| all_account | object | 共享账号类别 |
| account_type | string | 业务线 枚举值：AD 营销、STAR 星图 |
| account_info | object | 共享账号信息 |
| account_type | string | 业务线 枚举值：AD 营销、STAR 星图、BP 巨量引擎工作台 |
| account_id | number | 账号id |
| all_account_by_company | object | 共享账号信息 |
| account_type | string | 业务线 枚举值：AD 营销、STAR 星图 |
| company_id | number | 账号id |
| error_list | object[] | 共享失败列表 |
| share_mode | string | 共享类型 允许值：PART 指定账户共享、ALL 巨量引擎工作台内某类账户所有共享 共享类型为PART，读取account_info 共享类型为ALL，读取all_account，即表示该业务线下的全量账号 |
| all_account | object | 共享账号类别 |
| account_type | string | 业务线 枚举值：AD 营销、STAR 星图 |
| account_info | object | 共享账号信息 |
| account_type | string | 业务线 枚举值：AD 营销、STAR 星图、BP 巨量引擎工作台 |
| account_id | number | 账号id |
| all_account_by_company | object | 共享账号信息 |
| account_type | string | 业务线 枚举值：AD 营销、STAR 星图 |
| company_id | number | 账号id |
| error_message | string | 共享失败原因 |
| request_id | string | 请求的日志id，唯一标识一个请求 |
