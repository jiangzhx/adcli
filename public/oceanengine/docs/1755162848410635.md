# 通过优化目标获取橙子落地页站点信息

本接口支持获取优化目标关联橙子建站站点列表，方便您通过API创编营销

注意：本接口暂不支持区分橙子落地页是账户下自建的还是被共享的，您可通过以下方式进行区分：

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/orange_site/get/](https://api.oceanengine.com/open_api/v3.0/tools/orange_site/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1755162848410635)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户id

page

number

页码，范围：page >= 1

page_size

number

页面数据量，范围：1-50

status

string

站点状态，可选值:

filtering

object

过滤条件

search

string

搜索字段，按照建站id和建站name进行模糊匹配，范围：1 <= 长度 <= 50

micro_app_instance_id

number

小程序/小游戏资产id，传入可筛选出账户下绑定了该资产id的橙子落地页，用于营销创编。资产id可通过以下接口获取

【[获取微信小程序列表](https://open.oceanengine.com/labels/34/docs/1771203622020111)】

【[获取微信小游戏列表](https://open.oceanengine.com/labels/34/docs/1771362185154564)】

【[获取字节小程序列表](https://open.oceanengine.com/labels/34/docs/1778249831680135)】

【[获取字节小游戏列表](https://open.oceanengine.com/labels/34/docs/1778100726681600)】

object

优化目标信息

external_action

string

优化目标，枚举详见附录[【附录-枚举值-转化类型】](https://open.oceanengine.com/labels/7/docs/1696710760171535)
