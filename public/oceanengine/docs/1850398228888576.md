# 创建事件资产

此接口用于创建投放账户下资产

# 请求地址

https://api.oceanengine.com/open_api/2/event_manager/assets/create/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1850398228888576)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
