# 取消小游戏/小程序/商品库/商品共享关系

取消资产共享，支持取消字节、微信小程序/小游戏/升级版商品/通用版商品库共享关系

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/cancel/](https://api.oceanengine.com/open_api/v3.0/tools/bp_asset_management/share/cancel/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773102677218368)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

organization_id

number

旧版巨量引擎工作台组织ID

asset_type

string

可选值:

instance_id

number

资产ID

share_mode

string

共享类型 允许值：PART 指定账户共享、BP_ALL_ACCOUNTS 旧版巨量引擎工作台内所有账户共享、COMPANY_ALL_ACCOUNTS公司主体内所有账户共享

account_infos

object[]

取消指定共享账号信息，一次最多取消共享到100个账号 当 share_mode 为PART时可用且必填

account_id

number

账号id

account_type

string

业务线 枚举值：AD 营销、BP 旧版巨量引擎工作台

all_accounts_by_bp

string[]

取消共享账号类别，一次最多操作1个，枚举值：AD 营销 当 share_mode 为BP_ALL_ACCOUNTS时可用且必填

all_accounts_by_company

object[]

取消主体共享账号信息，一次最多操作1个 当share_mode为 COMPANY_ALL_ACCOUNTS时可用且必填

account_type

string

可选值:

company_id

number

账号id， 一次最多操作1个

# 应答字段

字段

类型

描述

code

number

返回码,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

message

string

返回信息,详见 [【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831)

data

json

json返回值

error_list

object[]

取消共享失败的账号列表，该字段为空则代表全部取消成功

share_mode

string

共享类型 允许值：PART 指定账户共享、BP_ALL_ACCOUNTS 旧版巨量引擎工作台内某类账户所有共享、COMPANY_ALL_ACCOUNTS公司主体内部分账户共享

共享类型为PART，读取account_info；

共享类型为BP_ALL_ACCOUNTS，读取all_account_by_bp，即表示该业务线下的全量账号；

共享类型为COMPANY_ALL_ACCOUNTS，读取all_account_by_company

error_message

string

取消共享失败的原因

all_accounts_by_bp

string

可选值:

account_info

string

可选值:

account_id

账号id

主体共享账号信息

account_type

string

可选值:

company_id
