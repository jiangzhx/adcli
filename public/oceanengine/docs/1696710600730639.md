# 上传视频

通过此接口，用户可以上传和营销投放相关的素材视频

建议：对于连山url链接上传视频，请及时切换至[异步上传接口](https://open.oceanengine.com/labels/7/docs/1801533703648324)

# 请求地址

[https://api.oceanengine.com/open_api/2/file/video/ad/](https://api.oceanengine.com/open_api/2/file/video/ad/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710600730639)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: multipart/form-data

# 请求参数

字段

类型

描述

advertiser_id

number

客户ID

upload_type

string

视频/音频上传方式，允许值：

默认值: UPLOAD_BY_FILE

video_signature

string

视频的md5值(用于服务端校验)

video_file

file

视频文件，允许格式：mp4、mpeg、3gp、avi、m4v，文件大小≤500M

filename

string

upload_type = UPLOAD_BY_URL有效且必填

素材的文件名，可自定义素材名，不传择默认取文件名，最长255个字符 注：若同一素材已进行上传，重新上传不会改名。

labels

string[]

标签

video_url

string

视频 url地址，如[http://xxx.xxx](http://xxx.xxx/)，upload_type为UPLOAD_BY_URL必填

is_aigc

bool

视频素材是否为AIGC生成

is_guide_video

bool

上传视频场景是否是引导视频（此功能仅游戏行业白名单客户可用，如需使用请联系运营）
