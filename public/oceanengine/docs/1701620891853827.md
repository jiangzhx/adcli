# 互动营销管理

本章节主要介绍如何获取[Rubeex平台](https://rubeex.oceanengine.com/)制作生成的互动作品详细信息：

概念解释：

互动作品：[Rubeex平台](https://rubeex.oceanengine.com/)制作生成的互动作品，系统标识为project id。

试玩素材：互动作品推送到AD营销投放平台，生成试玩素材，系统标识为playable id，project id和playable id是1对n 的关系。

核心接口：

[获取作品信息](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1701620909023236)[获取作品场景信息](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1701620936632388)

[获取作品版本信息](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1701620989640707)

[获取互动作品关联的试玩素材列表](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1719937690024973)

[获取试玩素材关联的营销计划列表](https://open.oceanengine.com/doc/index.html?key=ad&type=api&id=1719937796782094)
