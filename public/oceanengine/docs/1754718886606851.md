# 归因方式上报说明

## 介绍

客户在收到clickid/callback后进行转化归因时，由于存在多种归因逻辑，故期望客户能够把最终归上转化对应的归因方式上报到巨量引擎（例如本次转化是通过idfa归因，则回传该信息），以便一同监控归因设备ID的变化情况，异常时可以双方第一时间发现，减少归因gap损失。

## 归因方式对应的枚举值

| 归因方式 | 枚举值 | 说明 |
| --- | --- | --- |
| 调起链接上的clickid | uri_clickid | 一般在落地页、小程序、快应用的调起链接上会拼接clickid，客户可以根据用户转化情况将对应调起链接上的clickid上报给巨量，完成转化上报 |
| imei | imei | android端，通过imei进行归因上的转化 |
| mac | mac | 通过mac地址进行归因上的转化 |
| oaid/oaid_md5 | oaid | android端通过oaid(包括md5后的值)进行归因上的转化 |
| idfa/idfa_md5 | idfa | ios通过idfa进行归因上的转化 |
| caid/caid1/caid2/caid1_md5/caid2_md5 | caid | ios通过caid(包括caid1、caid2或者md5后的结果)进行归因上的转化 |
| ip/ipv4 | ipv4 | 只通过ipv4进行归因上的转化 |
| ipv6 | ipv6 | 只通过ipv6进行归因上的转化 |
| ip/ipv4+ua | ipv4+ua | 通过ipv4和ua(包括直接使用ua和解析ua中的关键信息)进行归因上的转化 |
| ipv6+ua | ipv6+ua | 通过ipv6和ua(包括直接使用ua和解析ua中的关键信息)进行归因上的转化 |
| time_fingerprint/paid | time_fingerprint | 通过time_fingerprint(宏参数: __TIME_FINGERPRINT__)或PAID(宏参数: __PAID__)归因上的转化 |
| android_id | android_id | 通过android id归因上的转化 |
| 渠道包 | channel_package | 通过渠道包(包含单独的包名或者在包里打入特定标识等方式)方式归因上的转化 |
| 用户账户id | customer_user_id | 通过客户内部账户体系归因到的转化 |
| 手机号归因(PA) | phone_num | 通过手机号归因到的转化 |

## 回传方式

以下两个方案，客户可以根据目前的回传方式进行适配

方案一：使用接口二回传

对接说明：https://open.oceanengine.com/labels/7/docs/1696710656359439

上报地址：https://[ad.oceanengine.com](https://cloud.bytedance.net/tlb/archive/search/domain?keyword=ad.oceanengine.com)/track/activate

请求方式：GET

回传字段：attribute_key

示例：

通过idfa归上了一个激活事件上报转化

callback/clickid: EJiw267wvfQCGKf2g74ZIPD89-vIATAMOAFCIjIwMTkxMTI3MTQxMTEzMDEwMDI2MDc3MjE1MTUwNTczNTBIAQ==

请求地址：https://ad.oceanengine.com/track/activate?callback=EJiw267wvfQCGKf2g74ZIPD89-vIATAMOAFCIjIwMTkxMTI3MTQxMTEzMDEwMDI2MDc3MjE1MTUwNTczNTBIAQ%3D%3D&conv_time=1574835097&event_type=0&os=1&idfa=FCD369C3-F622-44B8-AFDE-12065659F34B&source=mybestcustom&attribute_key=idfa

方案二：使用事件管理api回传

对接说明：https://event-manager.oceanengine.com/docs/8650/app_api_docs/

上报地址：https://analytics.oceanengine.com/api/v2/conversion

请求方式：POST

content-type：application/json

回传字段：attribute_key

示例：

通过idfa归上了一个激活事件

callback/clickid：EJiw267wvfQCGKf2g74ZIPD89-vIATAMOAFCIjIwMTkxMTI3MTQxMTEzMDEwMDI2MDc3MjE1MTUwNTczNTBIAQ==
