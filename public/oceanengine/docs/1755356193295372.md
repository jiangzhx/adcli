# 获取推荐关键词

获取推荐关键词（支持搜索直投和搜索快投）

### 请求地址

[https://api.oceanengine.com/open_api/v3.0/sugg_words/](https://api.oceanengine.com/open_api/v3.0/sugg_words/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1755356193295372)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户id |
| project_id | number | 项目id |
| query_list | string[] | 以词推词，最多只传入10个，长度要求30字内 ,可传入行业名称获取行业推词 |
| promotion_materials | object | 营销素材 |
| abstracts | string[] | 营销摘要 |
| quality_words | string[] | 优词 |
| app_name | string | 应用名 |
| source | string | 来源 |
