# 获取微信小程序列表

可以查询组织及下级组织可管理和被共享的小程序信息。

# 请求地址

https://api.oceanengine.com/open_api/v3.0/tools/ebp/wechat_applet/list/[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1847487716271104)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)
