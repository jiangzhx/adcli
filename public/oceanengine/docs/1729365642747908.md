# 推送dmp人群包到云图账户

通过此接口，将DMP人群包复制一个，并推送至对应的云图账号内，可在人群模块自定义分析查看和应用（该人群不支持再次推送）

### 请求地址

[https://ad.oceanengine.com/open_api/2/dmp/custom_audience/copy/](https://ad.oceanengine.com/open_api/2/dmp/custom_audience/copy/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1729365642747908)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

POST

### Header

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://open.oceanengine.com/labels/7/docs/1696710505596940)接口获取 |
| Content-Type 必填 | string | 请求消息类型 允许值: application/json |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 人群包所属客户ID |
| to_advertiser_id 必填 | number | 推送客户ID（云图虚拟客户ID，即virtual_adv_id），可以通过[【获取投放账户关联云图账户信息】](https://open.oceanengine.com/labels/7/docs/1729365568399363)接口获取 |
| custom_audience_id 必填 | number | 人群包ID |

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| message | string | 返回信息,详见[【附录-返回码】](https://open.oceanengine.com/labels/7/docs/1696710760866831) |
| data | object | 返回数据 |
| custom_audience_id | number | 推送到云图平台的 advid |
| request_id | string | 请求的日志id，唯一标识一个请求 |
