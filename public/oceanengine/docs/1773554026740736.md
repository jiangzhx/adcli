# 获取图文素材

通过此接口，用户可以获取经过一定条件过滤后的投放账户下创意素材库下图文及图文信息。

注意：page*page_size＞10000时会报错，请注意调增请求量级

# 请求地址

[https://api.oceanengine.com/open_api/2/carousel/list/](https://api.oceanengine.com/open_api/2/carousel/list/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1773554026740736)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

GET

根据图文上传时间进行过滤的起始时间，与end_time搭配使用。

例如："2025-12-01 00:00:00"

end_time

根据图文上传时间进行过滤的起始时间，与end_time搭配使用。

格式：yyyy-MM-dd HH:mm:ss

例如： "2025-01-01 00:00:00",

string[]

页码，默认值1，

注意：page*page_size＞10000时会报错，请注意调增请求量级

number

注意：page*page_size＞10000时会报错，请注意调增请求量级
