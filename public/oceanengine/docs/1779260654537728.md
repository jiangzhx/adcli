# 批量更新项目投放时段

批量更新「巨量营销升级版」项目投放时段

# 请求地址

[https://api.oceanengine.com/open_api/v3.0/project/week_schedule/update/](https://api.oceanengine.com/open_api/v3.0/project/week_schedule/update/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1779260654537728)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

# 请求方法

POST

# Header

字段

类型

描述

Access-Token

string

授权access_token，可以通过【[获取Access token](https://open.oceanengine.com/labels/7/docs/1696710505596940)】接口获取

Content-Type

string

请求消息类型 允许值: application/json

# 请求参数

字段

类型

描述

advertiser_id

number

投放账户id

data

object[]

批量修改投放时段，限制最多批量修改10个项目

project_id

number

项目ID

schedule_time

string

更新后的投放时段，格式是48*7位字符串，且都是0或1。也就是以半个小时为最小粒度，周一至周日每天分为48个区段，0为不投放，1为投放，不传、全传0、全传1均代表全时段投放。

例如：000000000000000000000001111000000000000000000000000000000000000000000001111000000000000000000000000000000000000000000001111000000000000000000000000000000000000000000001111000000000000000000000000000000000000000000001111000000000000000000000000000000000000000000001111000000000000000000000000000000000000000000001111000000000000000000000

则投放时段为周一到周日的11:30~13:30

schedule_scene

string

生效方式，允许值：

默认值: REALTIME
