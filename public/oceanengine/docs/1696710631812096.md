# 回传有效线索

# 接口说明

### 为什么要回传线索深转状态 ？

深转（深度转化），是指在从营销投放到与客户完成交易的过程中，更接近交易事件的客户行为。

巨量引擎需要追踪用户行为来优化营销效果，因此只能优化例如“表单提交”、“多转化”等相对浅度的转化行为。而对您来说往往客户的深度转化行为更有价值，例如“确认意向”、“到店”甚至“成交”等目标。

当巨量引擎能够追踪客户深转行为时，就能够优化营销投放，花更少的钱找到更倾向发生深转行为的客户（如表所示，投成交能带来更低的成交成本）。

因此，就需要您回传已收集线索的这些事件，协助实现巨量引擎追踪客户深度转化行为。（下表数字仅为示例，不代表真实效果）

| 投放 | 营销费 | 获取线索 | 成交 | 成本 |
| --- | --- | --- | --- | --- |
| 表单提交 | 10000 | 1000 | 20 | 500 |
| 成交 | 5000 | 400 | 20 | 250 |

# 支持回传事件类型

当您使用橙子建站落地页开始投放时，巨量引擎会把收集的客户线索存放到飞鱼 CRM 中。您可以使用此接口为这些线索回传线索深度转化跟进状态（clue_convert_state，目前支持：194：回访-信息确认、195：回访-加为好友、196：回访-高潜成交、527：顾客到店或销售人员上门、285：正价品支付、10000：无效线索），使巨量引擎完成对线索的追踪，并优化营销投放效果。

选择线索状态后，您可以使用reason_code字段将线索状态所属具体场景进行打标签。全行业的客户可使用通用标签进行标记，家居建材及教育行业的客户还可以使用更为详细的专属标签进行标记。为方便帮您优化线索有效性和数据分析。选择无效标签是建议您选择具体的无效标签，不建议您选择“其它”。

| 线索状态-行业 | reason_code | 标签类型 |
| --- | --- | --- |
| 无效标签-通用 | CALL_THREE_TIMES_NO_RESPONSE | 拨打3次以上未接通 |
| INVALID_NUMBER | 空号/停机/关机 |  |
| NO_AD_RESPONSE | 反馈未在营销留资 |  |
| CONNECTED_NO_INTENTION | 接通但无意向 |  |
| OFFENSIVE_LANGUAGE | 黑脏词 |  |
| OTHER | 其它 |  |
| 无效标签-家居建材 | HOME_MISMATCH_LOW_BUDGET | 需求不匹配-消费力差（指代预算少/面积小），原MISMATCH_LOW_BUDGET枚举逐步下线中 |
| HOME_MISMATCH_DIFFERENT_LOCATIONS | 需求不匹配-需求与供给异地，原MISMATCH_DIFFERENT_LOCATIONS枚举逐步下线中 |  |
| HOME_COLLABORATION_WITH_PEERS | 同行合作 |  |
| HOME_NO_SHOW_FOR_MEASUREMENT_VISIT | 爽约上门量房 |  |
| HOME_RENOVATION_TYPE_NOT_ACCEPTABLE | 装修类型不可承接 |  |
| 无效标签-教育行业 | EDU_MISMATCH_CLASS_TIME | 需求不匹配-上课时间不匹配 |
| EDU_MISMATCH_COURSE | 需求不匹配-课程不匹配 |  |
| EDU_MISMATCH_DIFFERENT_LOCATIONS | 需求不匹配-需求与供给异地 |  |
| EDU_UNABLE_TO_ATTEND_IN_PERSON | 无法线下参加 |  |
| 顾客到店/销售人员上门-家居建材 | HOME_MEASUREMENT_COMPLETED | 已量房 |
| HOME_ORDER_DISPATCHED | 已派单 |  |
| 顾客表达有意向-家居建材 | HOME_APPOINTMENT_SCHEDULED | 已约时间 |
| HOME_CAN_ADD_WECHAT | 可加微 |  |
| HOME_CONNECTED_WITH_INTERESTED_PARTY | 接通有意向 |  |
| 顾客表达有意向-教育 | EDU_FIRST_CLASS_ATTENDED | 首次到课 |
| EDU_INTERESTED | 有意向 |  |
| 正价品支付-家居建材 | HOME_CONTRACT_SIGNED | 已签约 |
| HOME_SERVICE_DELIVERY_COMPLETED | 已完成服务交付 |  |

### 注意事项

通过本接口回传的深度转化事件将在飞鱼CRM-营销-深度转化-上报明细中披露

# 请求参数

字段

类型

描述

advertiser_ids

string[]

客户id列表

clue_convert_state

number

线索当前事件状态，可选值:

clue_id

string

线索ID

source

number

营销来源，新版深度转化 source为1 允许值：0,1

click_id

string

clickid是一个加密字符串，代表的是字节的关键营销信息。需要从三方落地页url 中的 clickid 参数获取值。

event_data

object

无效事件标签；若clue_convert_state = INVALID_EVENT，则该字段必填

reason_code

string

对线索当前事件状态clue_convert_state选择扩展标签。

reason_message

string

自定义原因信息，选填，当reason_code = OTHER时建议填写
