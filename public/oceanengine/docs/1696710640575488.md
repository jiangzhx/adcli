# 查询券码记录

查询券码记录接口。

### 请求地址

[https://ad.oceanengine.com/open_api/2/clue/coupon/code/get/](https://ad.oceanengine.com/open_api/2/clue/coupon/code/get/)[可视化调试](https://open.oceanengine.com/tools/visual_debug.html?docId=1696710640575488)[SDK下载](https://open.oceanengine.com/labels/7/docs/1773083358100557)[MCP调用](https://open.oceanengine.com/labels/7/docs/1847394521222599)

### 请求方法

GET

### Header

| 字段 | 类型 | 描述 |  |
| --- | --- | --- | --- |
| Access-Token 必填 | string | 授权access_token，可以通过[【获取Access token】](https://ad.oceanengine.com/openapi/doc/index.html?id=1696710506574848)接口获取 |  |

### 请求参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| advertiser_id 必填 | number | 客户ID |
| activity_id | number | 活动ID，可从获取卡券详情接口获取 |
| coupon_id | number | 卡券活动ID，可从获取卡券详情接口获取 |
| resource_id | number | 优惠券ID，可从获取卡券详情接口获取 |
| start_time | string | 起始时间，格式：%Y-%m-%d，默认7天前，即不指定起止时间获取最近7天数据 |
| end_time | string | 截止时间，格式：%Y-%m-%d，默认当天，即不指定起止时间获取最近7天数据 |
| status | string | 券码状态 VALID:有效中 EXPIRED:已过期 USED:已使用 ABANDONED:已废弃 |
| page | number | 页码，默认值为1 |
| page_size | number | 页面大小，默认值10，不超过50 |
