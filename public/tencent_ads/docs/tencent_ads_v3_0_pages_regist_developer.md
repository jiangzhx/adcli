# 如何注册成为开发者？

当您决定开始使用 Marketing API 进行接口调用，您需要首先注册成为开发者：

Step 1：进入注册页面。点击开发者官网右上角[注册](https://developers.e.qq.com/reg)按钮即可进入；

Step 2：关联 QQ 号作为开发者登录凭证。进入注册页面后，如果当前电脑已经登录 QQ 客户端，则默认显示以当前登录 QQ 号作为快捷登录，您也可以点击“帐号密码登录”换用其他 QQ 号；

Step 3：完善开发者资料。将资料填写完整并通过手机验证码验证通过后，即完成开发者注册。
